Use this matrix to pick compatible releases across the ImFusion product line. Only combinations listed on the same row are officially supported.

## Supported combinations

| C\+\+ SDK | Python SDK | Suite | Labels |
| --- | --- | --- | --- |
| 4.3 | 0.10 | 12.0 | 1.0 |

## Notes

- Mixing versions across rows is not supported and may produce runtime or ABI errors.
- Always upgrade all four products together when moving to a new supported row.
- For version-specific documentation, use the version dropdown at the top of each product's sidebar.
