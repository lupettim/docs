## What can you build with ImFusion?

ImFusion is a modular, plugin-based SDK designed to take you from prototype to production without rewriting your code. It supports Windows, Linux, and macOS, and integrates with NumPy, PyTorch, and standard medical formats like DICOM and NIfTI.

## Get started in three steps

<Steps>
  <Step title="Install the Python SDK">
    Install `imfusion-sdk` from the ImFusion package index and activate your license.

    ```bash
    pip install imfusion-sdk --extra-index-url https://pypi.imfusion.com/simple
    ```
  </Step>
  <Step title="Load and process an image">
    Use `imfusion.load()` to open any supported format and run built-in algorithms.

    ```python
    import imfusion
    
    images = imfusion.load("scan.nii")
    result = imfusion.execute_algorithm("Base.ImageResampling", images, {"targetSpacing": [1.0, 1.0, 1.0]})
    imfusion.save(result, "resampled.nii")
    ```
  </Step>
  <Step title="Explore more">
    Browse the [Python SDK guides](/python-sdk/core-concepts), check the [API Reference](/api/imfusion-module), or open ImFusion Suite for interactive exploration.
  </Step>
</Steps>
