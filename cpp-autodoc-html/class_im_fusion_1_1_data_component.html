Detailed Description

template<typename T>
class ImFusion::DataComponent< T >

Main specialization of DataComponentBase for regular types.

If you need serialization, make your class T inherit from DataComponent<T> and implement the copy- constructor (T(const T&)), the copy assignment operator (T& operator=(const T&)), and the equality comparison operator (bool operator==(const T&) const / friend bool operator==(const T&, const T&)). The clone(), assign(), and equals() methods will forward to your class methods.

You also still need to implement id(), configure(), and configuration() to make your class serializable.

Example:

class FooComponent : public DataComponent<FooComponent>
{
public:
// copy-constructor and copy-assignment operator are auto-generated by the compiler
bool operator==(const FooComponent& rhs) const { return rhs.i == i; }
// you still need to implement these methods to make the DataComponent serializable
std::string id() const override;
void configure(const Properties*) override;
void configuration(Properties*) const override;
private:
int i = 0;
};
