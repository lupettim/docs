Detailed Description

Base interface for custom Image types that can be converted from/to MemImages and are supported by SharedImage.

Implementing the CustomImage interface enables you to put custom Image types into a SharedImage and make use of its automatic conversion between different image representations. In addition to the pure virtual convertToMem() function, you must implement the inverse conversion as a function with the following signature: static std::shared_ptr<Image> convertFromSharedMem(const std::shared_ptr<const MemImage>&);. This function may increase the reference count of the input MemImage shared_ptr, but must not modify it.

Minimal example showcasing the compressed storage of pixel data:

class CompressedImage : public CustomImage
{
public:
explicit CompressedImage(const MemImage& mem)
: CustomImage(mem.descriptor())
, compressedData(Compression::zstd::compress(ByteBufferView((char*)mem.data(), mem.byteSize())))
{}
static std::shared_ptr<Image> convertFromMem(const std::shared_ptr<const MemImage>& mem) {
return mem ? std::make_unique<CompressedImage>(*mem) : nullptr;
}
std::unique_ptr<MemImage> convertToMem() const override
{
std::unique_ptr<MemImage> mem = MemImage::create(m_descriptor);
Compression::zstd::Context().decompressTo(compressedData, ByteBufferView((char*)mem->data(), mem->byteSize()));
return mem;
}
private:
ByteBuffer compressedData;
};
