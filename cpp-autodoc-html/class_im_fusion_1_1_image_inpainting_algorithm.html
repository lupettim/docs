Detailed Description

Algorithm that inpaints a masked area of an image.

By default the algorithm will inpaint the areas with mask value == 0, if the parameter p_invertMask is true then the algorithm will inpaint the areas with mask value != 0. The algorithm uses normalized convolutions for inpainting, that means that given an image x, a mask m and a convolution function c() the output of the algorithm is c(x) / (c(m) + eps) where m == 0 and x everywhere else. The convolution filter could be a Gaussian (supported on both CPU and GPU), a box filter or a sum of Gaussians. The "sum of Gaussians" filter is constructed by summing different Gaussians filters with sigma parameters going from 1 to p_kernelSizePix / 3. The idea is that such filter will give more importance to nearby pixels, resulting in less pronounced edges between the image and the inpainted area.
