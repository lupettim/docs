Detailed Description

The GlDeformationGrid visualizes the deformation of an image through the deformation of a grid rendered on top of the image.

Since the deformation of the input image must be inverted internally (see InvertDeformationAlgorithm), this visualization is rather expensive.

int ImFusion::GlDeformationGrid::downsamplingFactor ( ) const

Downsampling factor of the internal inverted GlDenseDeformation.

Higher downsampling results in faster computation but in a less accurate visualization. The initial value depends on the size of the input image.

◆ updateInterval()

std::chrono::milliseconds ImFusion::GlDeformationGrid::updateInterval ( ) const

Duration between two updates of the internal GlDenseDeformation.

A value of 0 updates on every rendering and blocks during the computation of the inverse deformation. A value greater 0 computes the inverse async. The initial value depends on the size of the input image.
