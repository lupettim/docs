Detailed Description

Specialization of Buffer for OpenGL Atomic Counter Objects.

Note
Requires at least OpenGL 4.2 or the ARB_shader_atomic_counters extension.
See also
https://www.khronos.org/opengl/wiki/Atomic_Counter
