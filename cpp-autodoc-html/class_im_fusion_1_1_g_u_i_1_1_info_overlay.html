Detailed Description

Interface for a ViewOverlay to show textual information depending on the mouse cursor position (for instance: pixel information of an image, face/vertex info of a mesh).

The text can either be shown using a regular GlTextOverlay at the selected Anchor or as a tooltip hovering next to the mouse pointer.
