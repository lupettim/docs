Parameters
_eventEvent type.
_buttonButton which triggered the event. For move events this must be NoButton.
_buttonsButton state when the event was generated. The button state is a combination of LeftButton, RightButton, MiddleButton using the OR operator. For mouse move events, this is all buttons that are pressed down. For mouse press and double click events this includes the button that caused the event. For mouse release events this excludes the button that caused the event.
_modifiersKeyboard modifiers when the event was generated. The modifier state is a combination of ShiftModifier, CtrlModifier and AltModifier using the OR operator.
_xx-position of mouse in window coordinates
_yy-position of mouse in window coordinates
_deltaDistance that the mouse wheel is rotated. A single step of the mouse wheel is 120.
