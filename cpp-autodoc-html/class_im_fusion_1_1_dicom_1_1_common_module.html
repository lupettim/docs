Detailed Description

Collection of the essential tags describing a Dicom data set combining.

  • Patient Module (PS 3.3 C.7.1.1)
  • General Study Module (PS 3.3 C.7.2.1)
  • General Series Module (PS 3.3 C.7.3.1)
    See also
    http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.html#sect_C.7.1.1 http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.2.html#sect_C.7.2.1 http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.3.html#sect_C.7.3.1 ImageInfoDataComponent AnatomyDataComponent

◆ load() [1/2]

void ImFusion::Dicom::CommonModule::load ( SharedImageSet & sis) const

Load data from the module and store it in a ImageInfoDataComponent of sis.

Missing values will be set to empty strings in the datacomponent. If the dataset contains anatomical data, it is stored in a AnatomyDataComponent.

◆ load() [2/2]

void ImFusion::Dicom::CommonModule::load ( ImageInfoDataComponent & iidc) const

Load data from the module into the ImageInfoDataComponent.

Missing values will be set to empty strings in the datacomponent. This is automatically called by the above load method for SharedImageSet and should only be used for non-image data (such as PointClouds).

◆ save()

void ImFusion::Dicom::CommonModule::save ( const ImageInfoDataComponent * iidc)

Writes the ImageInfoDataComponent to the dataset.

This is automatically called by the above save method for SharedImageSet and should only be used for non-image data (such as PointClouds).

◆ saveMandatory()

void ImFusion::Dicom::CommonModule::saveMandatory ( const std::string & sopClassUID)

Writes all remaining mandatory tags not covered by save(ImageInfoDataComponent).

This is automatically called by the above save method for SharedImageSet and should only be used for non-image data (such as PointClouds).
