Split edges

Documentation for ImFusion::DisplayLayout::splitEdge(ImFusion::LayoutEdge* e). In order to split an edge another edge needs to cross it. This is the case when 4 nodes are connected by exactly two edges. The examples are always for vertical edges but horizontal edges work the same (just rotate your head 90°...)

Unite Edges

Documentation for ImFusion::DisplayLayout::uniteEdge(ImFusion::LayoutEdge* e, ImFusion::LayoutEdge* e). Two edges can be united if they have the same split ratio and if there a two nodes that are connected by another edge orthogonal to the united edges.

First check if both edges have the about the same split ratio. Then try to find a node connected to edge 1 (red) that is connected with an orthogonal edge (turquoise) to another node connected to edge 2 (green).

...
if(e1->splitRatio < e2->splitRatio+tolerance && e1->splitRatio > e2->splitRatio-tolerance) // 1
{
bool canMerge = false;
for(auto node1: e1->left) // 2
{
for(auto node2: e2->left) // 3
{
if(connectingEdge(node1, node2, !e1->vertical)) // 4
{
canMerge = true;
break;
}
if(canMerge) break;
}
}
