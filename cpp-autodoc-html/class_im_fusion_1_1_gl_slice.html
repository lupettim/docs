Detailed Description

Represents an MPR slice for rendering Data into 2D views.

GlSlice will render MPRs of one or multiple Data instances. While the actual rendering of individual data is delegated to the GlSliceRenderer interface, GlSlice will take care of setting up the OpenGL context and blending the rendered images.

Furthermore, GlSlice can act as GlObject to render the extent (and optionally image content) of the MPR slice into other third-party views.

See also
ImageView2D, GlSliceView, GlSliceRenderer

enum class ImFusion::GlSlice::BlendingMode
strong

Enumeration of the blending modes available in GlSlice.

Enumerator
None 

No blending.

Default 

Default alpha blending.

Color 

Automatic color blending depending on modality (either SingleColor or DualColor blending mode).

Checkerboard 

Checkerboard blending.

DistanceToBorder 

Per-pixel blending based on distance to image border.

SingleColor 

Use no tint color for the first image, use the primary blendingColor for all other images.

DualColor 

Use primary blendingColor for the first image, use secondary blendingColor for all other images.

Enumerator
Nothing 

Don't render MPR cross-sections at all.

Border 

Render the surrounding border of the slice.

Render the actual image content.

Centerlines 

Render the centerlines of the slice.

Plane 

Render the slice as semi-transparent plane.

Crosssection 

Render lines forming a cross where the three MPR planes intersect (only if exactly 3 slice views exist).
