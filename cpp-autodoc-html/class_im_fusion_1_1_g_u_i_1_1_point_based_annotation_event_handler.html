Detailed Description

Event handler for creating and editing point-based annotations.

This event handler can operate in two different modes:

  • In "creation mode" the editing of the existing points is not possible and the only actions you can perform are to append additional points at the end, or to finish or abort the creation. Usually, this is the initial mode when starting with an empty annotation without any points.
  • In "regular mode" the full set of editing actions are available.

If you need an interactively editable GlPointBasedAnnotation you additionally instantiate the PointBasedAnnotationEventHandler and wrap the two together in a TypedViewObject.
