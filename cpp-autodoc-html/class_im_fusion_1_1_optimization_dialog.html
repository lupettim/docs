◆ selectResult

void ImFusion::OptimizationDialog::selectResult ( int which)
signal

Emitted if a different result is selected.

If this signal is not connected from outside, the results of an optimization study are automatically set by calling the evaluate method of the last used optimizer with the corresponding parameters.
