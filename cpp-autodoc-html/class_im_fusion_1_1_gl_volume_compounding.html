Detailed Description

Compounding of multiple volumes (SharedImageSets) with different modalities for the computation of the resulting voxel values (Mode): Mean, Median, Maximum, and Distance.

Note
While the first three modes are self-descriptive, the DISTANCE mode computes distance-maps of the input volumes and employs them to blend the resulting volume. This mode allows to reduce artifacts at the edges. The properties of the resulting volume are taken from the given reference volume (setReferenceVolume()) or inferred from the input volumes.

enum ImFusion::GlVolumeCompounding::WeightedMeanMode

The mode for weighted Mean.

Enumerator

Compute the weight by the distance to the center.

ALONGZ 

Compute the weight by the distance to the center in z direction.

COSINE 

Compute the weight by a smooth cosine along the border covering any overlap.
