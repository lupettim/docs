Detailed Description

Utility class to automatically create DataLink instances between datasets in a DataModel.

This class can manage the following links automatically:

  • If setAutoLinkWindowing() is enabled it will create one LinkWindowing instance for each Data::Modality present in the DataModel so that images of the same modality share their display options.
  • If setAutoLinkPose() is enabled it will create one LinkPose instance for each DataGroup present in the DataModel and add its children to it.
Note
Ownership of the links created by DataModelAutoLinking will still be held by the observed DataModel as usual.
