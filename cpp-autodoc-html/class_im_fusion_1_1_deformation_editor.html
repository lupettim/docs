Detailed Description

The DeformationEditor algorithm modifiers the deformation property of a SharedImageSet.

It can add a specific deformation model or remove all deformations. If the input SharedImageSet contains multiple images, the deformation is expected to have the same properties on all images (e.g. same model and same grid size but not necessarly the same deformation).
