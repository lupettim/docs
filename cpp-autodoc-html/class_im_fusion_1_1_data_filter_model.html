Detailed Description

The DataFilterModel class is a proxy model to filter a DataItemModel.

By default, nothing is filtered. A DataFilterModel will only work correctly if the source model is of type DataItemModel.

Warning
Do not change the filter model from within callbacks of DataModel/DataWidget-changed signals, since this can potentially corrupt the QModelIndex mapping inside.
