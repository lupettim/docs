Detailed Description

Registers two images with a non-linear deformation based on TPSs.

The algorithm modifies the GlTPSDeformation object attached to the moving image (see also setMoving). Because TPS deformations are generally not invertible, only one image can have a deformation at a time.

std::shared_ptr< ParametricDeformation > ImFusion::TPSImageRegistration::deformation ( ) const

Returns the deformation object that is currently modified.

This is a convenience method that casts moving to a SharedImageSet, retrieves its deformation and cast that to a GlTPSDeformation. Returns 0, if any of the casts fail.
