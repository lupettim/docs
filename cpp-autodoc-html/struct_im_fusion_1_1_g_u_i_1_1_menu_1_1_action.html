Detailed Description

Represents an item inside a Menu where a callback function is called when user clicks on it.

Besides the default appearance it can also be of Checkable type where an underlying boolean state is toggled through the action (usually indicated by a checkmark icon), or of ExclusiveCheckable type where multiple actions comprise an enumeration of possible values of which only one can be selected (usually indicated by a radio button icon). By convention, a group of exclusive options is formed by the set of adjacent actions of this type where there is no Separator in-between.
