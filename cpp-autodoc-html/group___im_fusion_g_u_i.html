Detailed Description

Generic Abstraction Layer for Visualization and User Input Events.

The ImFusionGUI component provides a series of classes and interfaces to embed our visualization and rendering stack into an interactive graphical user interface. It consists of the following parts:

  • A platform-agnostic abstraction layer for receiving and handling user input (e.g. keyboard, mouse, touch events).
  • A set of interfaces and base classes for building a visualization viewport inside which you can arrange multiple views, rendering objects, and overlays in a hierarchical fashion.
  • A library of concrete classes implementing the beforementioned interfaces providing all the necessary tools to build a radiology workstation.
  • Some Qt wrapper classes to embed the ImFusionGUI component into a Qt-based GUI applicaiton.
Note
The ImFusionGUI component does not provide high-level GUI elements such as widgets or a window manager. Instead "GUI" refers to the visualization/rendering capabilities of our SDK and their integration into any 3rd-party graphical user interface.
