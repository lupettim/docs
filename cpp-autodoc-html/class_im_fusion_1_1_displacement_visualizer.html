Detailed Description

Algorithm/GlObject hybrid to display an image's displacement as a vector field in an ImageView2D.

The Algorithm interface does not perform anything but enabling instantiation from the data model/algorithm factory. The GlObject interface will render the vector field into any ImageView2D/GlSliceView it is added to.
