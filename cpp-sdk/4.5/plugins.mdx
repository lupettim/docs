The ImFusion C\+\+ SDK ships with a catalog of plugin modules you can enable per license.

| Plugin | Purpose |
| --- | --- |
| DICOM | Read, write, and query DICOM data |
| ML | Deep learning inference and training pipelines |
| Registration | Rigid, affine, and deformable image registration |
| Segmentation | Interactive and automatic segmentation tools |
| Ultrasound | Ultrasound acquisition, reconstruction, and processing |
| CT | Cone-beam and helical CT reconstruction |
| Vision | Camera calibration, feature detection, hand-eye calibration |
| Navigation | Tracking device integration and scene navigation |

Load plugins at runtime via the `PluginManager` or declaratively through your application configuration.
