The ImFusion C\+\+ SDK is a modular, plugin-based framework for building production medical imaging applications. It powers ImFusion Suite and is the foundation for high-performance clinical and research software on Windows, Linux, and macOS.

## What you get

- A unified data model for 2D, 3D, and 4D medical images
- GPU-accelerated image processing and rendering
- A plugin system that exposes DICOM, ML, registration, segmentation, ultrasound, CT, vision, and navigation modules
- Cross-platform build targets and Qt-based UI components

## When to use it

Choose the C\+\+ SDK when you need maximum performance, deep integration with hardware, or you are shipping a native desktop or embedded clinical application. For scripting and rapid prototyping, use the Python SDK instead.
