## Overview

[Compression](/cpp-sdk/4.5/api/namespaces/imfusion-compression) settings for nested ImFusionFile save operations.

Used to pass compression configuration to plugins that save nested data. Different datasets inside the same ImFusionFile can have very different size, structure, and compressibility, so a single global compression choice may be suboptimal. This struct allow defining compression settings for heterogeneous content where a one-mode-fits-all compression approach can either bloat storage/transfer or slow down save/load unnecessarily.
