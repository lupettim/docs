## Public Attributes

- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **tokenEndpoint** — URL that will be used to start the refresh flow.
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **tokenType** — Type of the access token (usually "Bearer").
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **clientId** — ClientID as configured in the OAuth2 server.
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **clientSecret** — Client secret as configured in the OAuth2 server (optional).
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **refreshToken** — The refresh token for the access token that should be refreshed.
