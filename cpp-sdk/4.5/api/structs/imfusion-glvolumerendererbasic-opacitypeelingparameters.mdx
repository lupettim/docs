`#include <ImFusion/GL/GlVolumeRendererBasic.h>`

## Overview

Record of parameters for configuring Opacity Peeling in the [GlVolumeRendererBasic](/cpp-sdk/4.5/api/classes/imfusion-glvolumerendererbasic).

## Public Attributes

- bool **enabled** — Flag whether to enable Opacity Peeling.
- int **layer** — Opacity Peeling layer to display.
- double **lowerThreshold** — Lower threshold of the Opacity Peeling algorithm.
- double **upperThreshold** — Upper threshold of the Opacity Peeling algorithm.
- double **advanceInMm** — Amount of space that should be skipped after each layer in millimeter.
