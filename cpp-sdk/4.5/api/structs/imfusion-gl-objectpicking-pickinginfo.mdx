## Overview

Structure encoding picking information for a single pixel.

## Public Attributes

- uint32_t **objectId** — Object ID, a value of 0xFFFFFFFFU represents an unknown object.
- uint32_t **elementId** — [Element](/cpp-sdk/4.5/api/classes/imfusion-dicom-element) index within the object.
