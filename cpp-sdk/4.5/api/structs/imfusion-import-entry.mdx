## Public Attributes

- QString **m_path**
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **m_ioAlgorithmId** — Id of the [IoAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-ioalgorithm) that should load the data. If empty, it needs to be determined from the path.
- [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) **m_ioAlgorithmConfig** — Configuration of the [IoAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-ioalgorithm). "location" will always be ignored.
- [std::optional](http://en.cppreference.com/w/cpp/utility/optional.html)&lt; size_t &gt; **m_imageIndex**
- QString **m_pathToLabels**
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **m_name** — Will be determined automatically if empty.
