## Member Enumeration Documentation

### Mode

```cpp
enum class ImFusion::GlMeshRendererSurface::DisplayOptions::Material::Mode
```

*strong*

**Enumerator:**

- `Uniform` — uses the colors from setMaterial
- `Color` — uses the colors from setColor or (if m_colors are empty) mesh vertex colors for ambient, diffuse and specular
- `Texture` — uses the mesh vertex texture for ambient, diffuse and specular
- `FaceLabel` — uses the labels of faces
- `VertexLabel` — uses the labels of vertices
