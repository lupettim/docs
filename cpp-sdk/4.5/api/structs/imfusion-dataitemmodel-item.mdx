## Overview

Internal record to implement bookkeeping of the hierarchy in the underlying [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel).

It enables two main things:

1. provide fast lookups of elements and their relationships
2. have a secondary storage backend so that we can call the pairs of Qt's begin/endFooRows() functions correctly. One [Item](/cpp-sdk/4.5/api/structs/imfusion-dataitemmodel-item) instance always corresponds to exactly one element in the QAbstractItemModel. However, the [Item](/cpp-sdk/4.5/api/structs/imfusion-dataitemmodel-item) hierarchy does not necessarily 1-to-1 correspond to the hierarchy in the underlying [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel).
