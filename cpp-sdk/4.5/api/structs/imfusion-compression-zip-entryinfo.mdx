`#include <ImFusion/Core/Zip.h>`

## Overview

Info for an archive entry:

- path on disk
- uncompressed file size
- last-modified-date in UTC
- crc32 checksum

## Public Attributes

- [Filesystem::Path](/cpp-sdk/4.5/api/classes/imfusion-filesystem-path) **path**
- [ByteSize](/cpp-sdk/4.5/api/classes/imfusion-bytesize) **uncompressedSize**
- [std::time_t](http://en.cppreference.com/w/cpp/chrono/c/time_t.html) **lastModifiedUtc**
- [std::uint32_t](http://en.cppreference.com/w/cpp/types/integer.html) **crc32**
