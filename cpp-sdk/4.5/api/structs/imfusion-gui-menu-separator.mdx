`#include <ImFusion/GUI/Menu.h>`

## Overview

Represents a separator to be shown as divider between unrelated groups of actions.

Separators are usually rendered as horizontal lines, optionally with a string describing serving as group header.

## Public Attributes

- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **title** — Optional text to be shown next to the separator and serving as group header.
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **description** — Optional text that explains the subsequent group, to be shown as tooltip/and or in the status bar.
