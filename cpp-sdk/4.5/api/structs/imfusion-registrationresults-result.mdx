## Overview

Represents a complete registration result, composed by multiple [DataResult](/cpp-sdk/4.5/api/structs/imfusion-registrationresults-dataresult) objects.

### restorePointSet

```cpp
bool ImFusion::RegistrationResults::Result::restorePointSet (const std::string & pointSetName) const
```

Restores one stored point set into loaded data entries of this result.

Existing sets with the same name are overwritten by configuration.

**Parameters:**

- `pointSetName` — The name of the point set to restore.

**Returns:** true if restoration succeeded for at least one data entry.

### dataDeleted

### operator

```cpp
bool ImFusion::RegistrationResults::Result::operator== (const Result & other) const
```

Compares two [Result](/cpp-sdk/4.5/api/structs/imfusion-registrationresults-result) objects for equality.

dataResults are compared in order, so the order of data in the list matters. regAlgConfig is ignored in the comparison.
