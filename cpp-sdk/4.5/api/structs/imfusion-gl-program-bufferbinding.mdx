## Overview

Struct to keep track of GlBuffers bound shaders as SSBOs.

## Public Attributes

- const [Buffer](/cpp-sdk/4.5/api/classes/imfusion-gl-buffer) \* **buffer** = nullptr — bound buffer
- uint32_t **blockIndex** = 0 — index of of the resource binding point in the GLSL shader
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **name** — uniform name of the binding point
