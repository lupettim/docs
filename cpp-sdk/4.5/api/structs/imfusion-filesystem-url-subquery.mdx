`#include <ImFusion/Core/Filesystem/Url.h>`

## Overview

A [Subquery](/cpp-sdk/4.5/api/structs/imfusion-filesystem-url-subquery) represents part of a query delimited by '&' or ';'.

- **Subquery** (const [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) &subquery)
- **Subquery** (const char \*subquery)

- **Subquery** (const char \*key, const char \*value)
- **operator std::string** () const
- bool **hasKey** () const — Checks if the subquery is a key-value pair.
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **key** () const — Returns the key if this is a key-value pair, or an empty string.
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **value** () const — Returns the value if this is a key-value pair, or the whole subquery.
