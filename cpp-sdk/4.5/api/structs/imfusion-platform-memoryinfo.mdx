`#include <ImFusion/Core/Platform.h>`

## Overview

Structure storing information about the available main memory as reported Platform::memoryInfo().

## Public Member Functions

- double **usageRatio** () const — Returns the ratio of availableMemory with respect to totalMemory in 0..1 range.

## Public Attributes

- [ByteSize](/cpp-sdk/4.5/api/classes/imfusion-bytesize) **availableMemory** = [ByteSize](/cpp-sdk/4.5/api/classes/imfusion-bytesize)(0) — Currently available main memory.
- [ByteSize](/cpp-sdk/4.5/api/classes/imfusion-bytesize) **totalMemory** = [ByteSize](/cpp-sdk/4.5/api/classes/imfusion-bytesize)(0) — Total main memory.
