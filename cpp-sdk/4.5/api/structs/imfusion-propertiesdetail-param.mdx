`#include <ImFusion/Core/PropertiesDetail.h>`

## Overview

Record of all properties that represent a single param inside a [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) instance.

## Public Attributes

- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **value** — Serialized value of the parameter.
- ParamType **type** = ParamType::Unknown — Optional parameter type, used for instance by [PropertiesWidget](/cpp-sdk/4.5/api/classes/imfusion-propertieswidget).
- [std::optional](http://en.cppreference.com/w/cpp/utility/optional.html)&lt; [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) &gt; **defaultValue** — Optional default value for parameter.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; [Attribute](/cpp-sdk/4.5/api/structs/imfusion-propertiesdetail-attribute) &gt; **attributes** — Optional param attributes, used for instance by [PropertiesWidget](/cpp-sdk/4.5/api/classes/imfusion-propertieswidget).
