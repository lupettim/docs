`#include <ImFusion/GL/PointCorrAlgorithm.h>`

## Overview

Structure encapsulating a single point correspondence item defined by a pair of points.

The point coordinates are either in image or in world coordinates, depending on the m_world flag. The annotation may be set by outside. If it is not null, [PointCorrAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-pointcorralgorithm) will update its coordinates during [updatePoints()](/cpp-sdk/4.5/api/classes/imfusion-pointcorralgorithm).

## Public Attributes

- vec4 **pt1** — Original coordinate of the point in image 1 (storage coordinate space of image 1).
- vec4 **pt2** — Original coordinate of the point in image 2 (storage coordinate space of image 2).
