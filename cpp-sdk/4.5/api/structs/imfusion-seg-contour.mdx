`#include <ImFusion/Seg/ContoursStorageComponent.h>`

## Overview

[Data](/cpp-sdk/4.5/api/classes/imfusion-data)[Structure](/cpp-sdk/4.5/api/classes/imfusion-seg-structure) containing a vector of 3D points as well as metadata such as:

- how the contour was created
- the type of the line defining the contour
- the frame where the contour was drawn
- the label value
- the name of the contour
