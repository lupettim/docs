## Overview

Represents a zip/unzip error.

Type and message can be retrieved via the corresponding members.

## Public Types

- enum class **Type** &#123; **UnspecifiedFailure** = 0 , **InvalidInputPath** = 1 , **InvalidOutputPath** = 2 , **ZipFailure** = 3 , **UnzipFailure** = 4 , **SecurityFailure** = 5 &#125;

## Public Attributes

- Type **type** = Type::UnspecifiedFailure
- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) **message**
