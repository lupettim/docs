## Overview

Struct describing an algorithm registered with a factory.

along with functions to create it and check compatibility. A default constructed descriptor has create and isCompatible functions that always return nullptr and false respectively. An empty descriptor implicitly casts to false, i.e. can check if a descriptor is valid with if (descriptor).

## Public Types

- enum [AlgorithmAttributes](#algorithmattributes) &#123; Internal = 1 &lt;&lt; 0 , Experimental = 1 &lt;&lt; 1 , Priority = 1 &lt;&lt; 2 &#125; — Bitflag enumeration of factory attributes for a registered [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm).

## Public Member Functions

- bool **isEmpty** () const — Returns whether the descriptor has an empty id or represents a known entry in the underlying factory.
- **operator bool** () const — Returns whether the descriptor represents a known entry in the underlying factory.

## Member Enumeration Documentation

### AlgorithmAttributes

```cpp
enum ImFusion::AlgorithmFactory::Descriptor::AlgorithmAttributes
```

Bitflag enumeration of factory attributes for a registered [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm).

**Enumerator:**

- `Internal` — Whether the algorithm is internal.
- `Experimental` — Whether the algorithm is experimental.
- `Priority` — Whether the algorithm should be shown at the top of the list.
