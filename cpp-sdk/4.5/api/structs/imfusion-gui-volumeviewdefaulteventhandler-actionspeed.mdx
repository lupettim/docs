`#include <ImFusion/GUI/VolumeViewDefaultEventHandler.h>`

## Overview

Record to define the speed/change rate at which a mouse/touch movement will modify the underlying view.

## Public Attributes

- double **mouseSpeed** = 1.0 — unitless factor by which the action's base speed is multiplied when using mouse movements
- double **touchSpeed** = 1.0 — unitless factor by which the action's base speed is multiplied when using touch movements
- double **keyboardSpeed** = 1.0 — unitless factor by which the action's base speed is multiplied when using key presses
