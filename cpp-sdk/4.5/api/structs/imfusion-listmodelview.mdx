## Overview

Represents a QListView, and the container [ItemViewWidget](/cpp-sdk/4.5/api/classes/imfusion-itemviewwidget). The model can be accessed in [ItemViewWidget::model](/cpp-sdk/4.5/api/classes/imfusion-itemviewwidget) or QListView::model.

## Public Attributes

- QListView \* **view** = nullptr
- [ItemViewWidget](/cpp-sdk/4.5/api/classes/imfusion-itemviewwidget) \* **itemViewWidget** = nullptr
