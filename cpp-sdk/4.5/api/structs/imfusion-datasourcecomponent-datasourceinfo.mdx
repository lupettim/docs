## Overview

Simple struct for storing the original filename of a data, how it has been loaded and the history of its sources.

The [DataSourceComponent](/cpp-sdk/4.5/api/classes/imfusion-datasourcecomponent) will store a vector of such structures, typically one per frame of an image set. This might seem unnecessary because most multi-frame datasets have the same origin but this is useful in the context of machine learning pipelines, where we assemble batches of multiple images, each one coming from a different source and its own history.
