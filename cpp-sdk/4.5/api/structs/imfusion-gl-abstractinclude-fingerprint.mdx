`#include <ImFusion/Core/GL/AbstractInclude.h>`

## Overview

AbstractIncludes provide a fingerprint to simplify duplicate detection.

If two finger prints are equal this means that the two includes provide the very same code substitution. If two includes do NOT provide the same code substitution they MUST not have the same [Fingerprint](/cpp-sdk/4.5/api/structs/imfusion-gl-abstractinclude-fingerprint).
