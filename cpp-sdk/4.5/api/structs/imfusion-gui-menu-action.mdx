## Overview

Represents an item inside a [Menu](/cpp-sdk/4.5/api/classes/imfusion-gui-menu) where a callback function is called when user clicks on it.

Besides the default appearance it can also be of Checkable type where an underlying boolean state is toggled through the action (usually indicated by a checkmark icon), or of ExclusiveCheckable type where multiple actions comprise an enumeration of possible values of which only one can be selected (usually indicated by a radio button icon). By convention, a group of exclusive options is formed by the set of adjacent actions of this type where there is no [Separator](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-separator) in-between.

## Member Enumeration Documentation

### Type

```cpp
enum class ImFusion::GUI::Menu::Action::Type
```

*strong*

Enumeration of available [Action](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-action) variants.

**Enumerator:**

- `Default` — Regular menu entry where some function is called when selected.
- `Checkable` — [Menu](/cpp-sdk/4.5/api/classes/imfusion-gui-menu) entry representing a boolean state that is toggled when clicked.
- `ExclusiveCheckable` — [Menu](/cpp-sdk/4.5/api/classes/imfusion-gui-menu) entry representing one of multiple mutually exclusive options that is selected when clicked.
A group of exclusive options is formed by the set of adjacent actions of this type where there is no [Separator](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-separator) in-between.
