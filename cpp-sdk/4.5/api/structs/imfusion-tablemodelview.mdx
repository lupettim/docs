## Overview

Represents a QTableView, and the container [ItemViewWidget](/cpp-sdk/4.5/api/classes/imfusion-itemviewwidget). The model can be accessed in [ItemViewWidget::model](/cpp-sdk/4.5/api/classes/imfusion-itemviewwidget) or QTableView::model.

## Public Attributes

- QTableView \* **view** = nullptr
- [ItemViewWidget](/cpp-sdk/4.5/api/classes/imfusion-itemviewwidget) \* **itemViewWidget** = nullptr
