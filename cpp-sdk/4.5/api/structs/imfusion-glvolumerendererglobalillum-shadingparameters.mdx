`#include <ImFusion/GL/GlVolumeRendererGlobalIllum.h>`

## Overview

Structure encapsulating a Phong shading configuration.

## Public Attributes

- vec3 **ambientColor** — Ambient light color and intensity.
- vec3 **diffuseColor** — Diffuse shading color and intensity.
- vec3 **specularColor** — Specular shading color and intensity.
- double **shininess** — Specular shading shininess, must be &gt; 0.
