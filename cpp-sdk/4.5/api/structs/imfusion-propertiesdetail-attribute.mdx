`#include <ImFusion/Core/PropertiesDetail.h>`

## Overview

Attributes allow for attaching custom data to individual parameters or an entire [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) instance.

They exist only at runtime and can be used for instance to customize the automatically generated [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui).
