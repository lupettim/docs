## Overview

Structure encapsulating all information needed to connect to a PACS instance.

Also provides some utility methods to read/write the configuration from/to the global settings and generate/parse distinct URI describing series/images on the PACS.

**Examples:** [DicomWorklist.cpp](/cpp-sdk/4.5/api/examples/dicom-worklist-8cpp), and [PacsExample.cpp](/cpp-sdk/4.5/api/examples/pacs-8cpp).

## Member Function Documentation

### fromUri

```cpp
DicomPacsConfiguration ImFusion::DicomPacsConfiguration::fromUri (const Url & uri)
```

*static*

Converts a URI to a DICOM image located on a PACS server into its separate parts.

URI must be of the form "pacs://&lt;Hostname&gt;:&lt;Port&gt;/&lt;HostAE&gt;?&lt;seriesUID&gt;#&lt;sopInstanceUID&gt;". If the optional cget-retrieve is a non-zero integer, useCGet will be set to true.

**See also:** [generateUri()](#generateuri)
