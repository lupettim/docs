## Overview

Collection of algorithms for post-processing meshes.

## Namespaces

- namespace [Cleaning](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-cleaning) — [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) processing algorithms related to cleaning of the mesh.
- namespace [ColorsTexturing](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-colorstexturing) — [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) processing algorithms related to mesh colors and texture processing.
- namespace [ConnectedComponents](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-connectedcomponents) — [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) processing algorithms based on connected component analysis.
- namespace [Geometry](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-geometry) — [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) processing algorithms related to geometrical operations.
- namespace [Misc](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-misc) — Miscellaneous mesh processing functions.
- namespace [Remeshing](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-remeshing) — [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) processing algorithms related to mesh remeshing (smoothing, decimation, subdivision).
- namespace [Statistics](/cpp-sdk/4.5/api/namespaces/imfusion-meshprocessing-statistics) — Methods for computing statistics on Meshes.

### labelConnectedComponents

```cpp
void ImFusion::MeshProcessing::labelConnectedComponents (Mesh * mesh, std::vector < int > & labels, std::vector < int > & labelsSize, std::vector < float > & labelsArea, std::vector < ImFusion::Geometry::AlignedBox > * labelsBoundingBox)
```

Determines connected components and assigns an id of connected component to each face using an integer "labelProperty" accessible through an underlying [TriMesh](/cpp-sdk/4.5/api/classes/imfusion-trimesh).

****Deprecated**:** "MeshProcessing::labelConnectedComponents is deprecated, consider using MeshProcessing::ConnectedComponents::labelConnectedComponents API instead"

**Parameters:**

- `mesh` — [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh)
- `labels` — mapping from the "labelProperty" to an array index (for the next 3 outputs)
- `labelsSize` — list of triangle count per connected component, ordered according to area from smallest to largest
- `labelsArea` — list of area of triangles per connected component, ordered from smallest to largest
- `labelsBoundingBox` — list of bounding boxes of connected components, ordered according to area from smallest to largest
