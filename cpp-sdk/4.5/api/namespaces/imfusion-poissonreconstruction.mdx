## Enumeration Type Documentation

### DensityThresholdMode

```cpp
enum ImFusion::PoissonReconstruction::DensityThresholdMode
```

**Enumerator:**

- `None` — No thresholding is applied on the vertices, all contributes to the final mesh.
- `QuantileDensity` — Applies alpha trimming on the vertices that are below a certain quantile in the density histogram. The quantile value is controlled by [PoissonParams.densityThreshold](/cpp-sdk/4.5/api/structs/imfusion-poissonreconstruction-poissonparams).
- `MedianDensity` — Applies binary thresholding on the vertices that have a lower density than a certain percentage of the median value of the density histogram. The percentage value is controlled by PoissonParams.medianDensityPercentageThreshold.
- `AbsoluteDensity` — Uses absolute density threshold. The value is controlled by PoissonParams.absoluteDensityThreshold.
