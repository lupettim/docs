### decompress

```cpp
ByteBuffer ImFusion::Compression::zlib::decompress (const ByteBufferView & inputBuffer)
```

Decompress the given data using zlib into a newly created buffer.

Use [decompressTo()](#decompressto) if you know the output size in advance and want to decompress into an already existing buffer. Returns an empty buffer on failure.

Decompress the given data using zlib into an already allocated buffer.
