## Function Documentation

### migrateQt

```cpp
bool ImFusion::SettingsMigration::migrateQt (const std::string & oldName, const std::string & newName, const Properties & p)
```

If version(*newName*) &lt; 1, attempts to migrate the settings for *newName* from the Qt backend to ImFusion::GlobalSettings, sets the version to 1, and returns true.

Otherwise, returns false.

**Parameters:**

- `oldName` — The name of the settings group in the Qt backend. Can be empty to signify the root group.
- `newName` — The name of the settings in ImFusion::GlobalSettings.
- `p` — The template properties of the settings to migrate.

### migrateQt

```cpp
template<typename T> bool ImFusion::SettingsMigration::migrateQt (const std::string & oldName)
```

**Parameters:**

- `oldName` — The name of the settings group in the Qt backend. Can be empty to signify the root group.
