## Overview

Collection of utility functions to work on or with strings.

**See also:** String Processing

## Enumerations

- enum SplitOptions &#123; Default = 0 , AnyOf = 1 &lt;&lt; 0 , IgnoreEmptyParts = 1 &lt;&lt; 1 , TrimWhitespace = 1 &lt;&lt; 2 &#125; — Bitflag enumeration for String::split().
- enum CaseSensitivity &#123; **CaseSensitive** = 0 , **CaseInsensitive** = 1 &#125; — Enumeration for levenshteinDistance.

- int levenshteinDistance (const [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) &source, const [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) &target, CaseSensitivity sensitivity=CaseSensitive) — Computes the Levenshtein distance (also known as Edit distance) between two strings.

- [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) rtrimCopy ([std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) str, [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) whitespace=" \t\f\v\r\n") — Returns a copy of *str* with all trailing occurrences of the characters in *whitespace* removed.
- [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) trimRef ([std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) str, [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) whitespace=" \t\f\v\r\n") — Returns a [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) into the input string where all leading and trailing occurrences of *whitespace* are removed.

- [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) rtrimRef ([std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) str, [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) whitespace=" \t\f\v\r\n") — Returns a [std::string_view](http://en.cppreference.com/w/cpp/string/basic_string_view.html) into the input string where all trailing occurrences of *whitespace* are removed.
