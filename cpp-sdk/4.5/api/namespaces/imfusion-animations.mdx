## Enumeration Type Documentation

### EasingCurve

```cpp
enum class ImFusion::Animations::EasingCurve
```

*strong*

Enumeration of built-in easing curves describing the rate of change of the interpolation between 0.0 and 1.0.

**Enumerator:**

- `Linear` — Constant velocity.
- `InCubic` — Cubic function accelerating from zero velocity.
- `OutCubic` — Cubic function decelerating to zero velocity.
- `InOutCubic` — Cubic function accelerating from zero velocity until halfway, then decelerating to zero velocity.
- `Sinusoid` — Sinusoid acceleration from zero velocity until halfway, then sinusoid deceleration to zero velocity.
