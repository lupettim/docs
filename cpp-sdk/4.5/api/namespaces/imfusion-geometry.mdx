## Overview

Classes to represent geometric primitives like lines or polygons, as well as functions dealing exclusively with such objects.

**Note:** This namespace contains improved versions of what is in the original and now deprecated GeometryLegacy namespace.

**See also:** ImFusionGeometry

## Classes

- class [AlignedBox](/cpp-sdk/4.5/api/classes/imfusion-geometry-alignedbox) — Class representing an axis aligned rectangular cuboid in 3D space.
- class [Circle](/cpp-sdk/4.5/api/classes/imfusion-geometry-circle) — Class representing a circle (actually a disk, so including the inner part) in 3D space.
- class [Ellipse2D](/cpp-sdk/4.5/api/classes/imfusion-geometry-ellipse2d) — Class representing an ellipse in 2D space.
- class [LineBase](/cpp-sdk/4.5/api/classes/imfusion-geometry-linebase) — Common interface of [Line](/cpp-sdk/4.5/api/classes/imfusion-geometry-line) and [LineSegment](/cpp-sdk/4.5/api/classes/imfusion-geometry-linesegment).
- class [Line](/cpp-sdk/4.5/api/classes/imfusion-geometry-line) — Class representing a full line, i.e.
- class [LineSegment](/cpp-sdk/4.5/api/classes/imfusion-geometry-linesegment) — Class representing a line segment, i.e.
- class [Plane](/cpp-sdk/4.5/api/classes/imfusion-geometry-plane) — Class representing an oriented plane in 3D space defined by a base point and a normal direction.
- class [Polygon2D](/cpp-sdk/4.5/api/classes/imfusion-geometry-polygon2d) — Class representing a planar polygon.
- class [AlignedRectangle2D](/cpp-sdk/4.5/api/classes/imfusion-geometry-alignedrectangle2d) — Class representing an axis aligned rectangle in 2D space.
- class [Rectangle](/cpp-sdk/4.5/api/classes/imfusion-geometry-rectangle) — Arbitrary rectangle in 3D space.
- class [Triangle](/cpp-sdk/4.5/api/classes/imfusion-geometry-triangle) — Class representing a triangle in 3D space.
- struct [SpherePivotResult](/cpp-sdk/4.5/api/structs/imfusion-geometry-spherepivotresult)

### fitLineRANSAC

```cpp
double ImFusion::Geometry::fitLineRANSAC (const std::vector < vec3 > & pts, vec3 & lineOutOrigin, vec3 & lineOutDir, std::vector < vec3 > & inliersOut, double inlierThreshold)
```

Fit line to inlier points.

**Parameters:**

- `pts` — input points
- `lineOutOrigin` — point belonging to the line
- `lineOutDir` — direction of the line, thus any point on the line can be estimated as: &lt;x, y, z&gt; = lineOutOrigin + t\*lineOutDir
- `inliersOut` — input points that lie within inlierThreshold distance from the estimated line
- `inlierThreshold` — threshold for distance between an inlier point and an estimated line

**Returns:** average distance between inlier point and the line
