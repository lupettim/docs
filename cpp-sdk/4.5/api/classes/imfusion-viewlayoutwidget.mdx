## Overview

The [ViewLayoutWidget](/cpp-sdk/4.5/api/classes/imfusion-viewlayoutwidget) class provides controls to show/hide views and change the layout.

It automatically adapts to the features of the layout e.g. disables the 'Show 3D View' button if the layout does not support the 3D view.
