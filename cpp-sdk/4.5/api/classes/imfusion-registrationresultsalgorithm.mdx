## Overview

Store registration results for data combinations.

This class allows for storage of registration results, where a result includes the transformation matrix of one or multiple (image) data sets - the most typical scenario will be data pairs. In addition, deformation instances and parameters can be kept track of. No input data is provided here, pointers to the data instances are provided through the corresponding controller. Saving and loading through workspaces / properties uses the name string of each involved data set.
