`#include <ImFusion/GUI/RadiologyDisplay.h>`

## Overview

Specialization of a [Display](/cpp-sdk/4.5/api/classes/imfusion-gui-display) that provides an off-the-shelf setup typically present in radiology workstations.

This class serves as convenience wrapper aggregating several helper classes and providing syntactic sugar:

- It has a [RadiologyViewGroup](/cpp-sdk/4.5/api/classes/imfusion-gui-radiologyviewgroup) so that a 2D image view, as well as 3 MPR views and a [VolumeView](/cpp-sdk/4.5/api/classes/imfusion-gui-volumeview) are present.
- It has a [RadiologyViewOverlays](/cpp-sdk/4.5/api/classes/imfusion-gui-radiologyviewoverlays) instance so that you can easily add and maintain common ViewOverlays. The [RadiologyDisplay](/cpp-sdk/4.5/api/classes/imfusion-gui-radiologydisplay) constructor will **not** ask it to create any overlays, thus this has to be done by the user.
- It has a [DisplayAutoLayouter](/cpp-sdk/4.5/api/classes/imfusion-gui-displayautolayouter) so that all views are automatically shown in a useful default way.
- You can optionally enable setAutomaticallyShowHideViews() so that only Views with active datasets are shown.
- It provides a set of convenience getter functions to directly access important entities inside.

```cpp
GUI::RadiologyDisplay display;

// create common ViewOverlays as needed
display.mainViewOverlays().addOverlays(display.views(), RadiologyViewOverlays::OverlayType::AllOverlays);

// select a particular view layout
display.layouter().setCurrentMode(DisplayAutoLayouter::Mode::FocusPlusStack);
display.layouter().setFocusedView(&display.main3dView());

// show a dataset
display.mainViewGroup().showData(image);
```

## Member Function Documentation

### requestViewVisibility

```cpp
void ImFusion::GUI::RadiologyDisplay::requestViewVisibility (const std::vector < const View * > views, bool value)
```

Sets whether the given set of *views* should be shown by the layouter() if applicable or not.

This is a convenience function simply forwarding to layouter-&gt;setViewsHidden([views](/cpp-sdk/4.5/api/classes/imfusion-gui-display), !value).

**Note:** The underlying layouting algorithm may still decide to not show a view even if it was set to visible here. See [DisplayAutoLayouter::setViewHidden()](/cpp-sdk/4.5/api/classes/imfusion-gui-displayautolayouter) for a detailed explanation.
