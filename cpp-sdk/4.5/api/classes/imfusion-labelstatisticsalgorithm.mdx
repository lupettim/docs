## Overview

The [LabelStatisticsAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-labelstatisticsalgorithm) computes image statistics depending on several label maps.

For each labelmap, the algorithm computes statistics on the image intensities depending on the labels in the labelmap. The algorithm computes the following statistics:

- Minimum
- Maximum
- Mean
- Standard Deviation
- Area/Volume
- Number of Pixels [Statistics](/cpp-sdk/4.5/api/structs/imfusion-statistics) are printed on the console or optionally saved to a CSV file.

### statistics

```cpp
Statistics ImFusion::LabelStatisticsAlgorithm::statistics (int labelMap, int label) const
```

Returns the statistics for the given label in the given labelMap.

The labelMap index depends on the order in which the were passed to the algorithm, e.g. the first label has index 1. The labelMap with index 0 returns the statistics on the complete image. If the combination of labelMap and label does not exists, a default constructed [Statistics](/cpp-sdk/4.5/api/structs/imfusion-statistics) object is returned.
