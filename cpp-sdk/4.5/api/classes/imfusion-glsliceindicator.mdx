## Overview

Renderer to show the position/orientation of [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) instances in context of other Views.

This [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) will visualize existing [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice)[View]s in other SliceViews and VolumeViews by rendering indicators at the location where the slices are in the coordinate system of the parent view. By default this will be lines the slices intersect in SliceViews and an outline/border in VolumeViews. The appearance can be customized: the [RenderMode2d](#rendermode2d) applies to all slices(), while the [RenderMode3d](#rendermode3d) is a per-slice setting.

**See also:** - [GUI::SliceIndicator](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceindicator) for a wrapper class that add input event handling to manipulate the [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) position and orientation by dragging the intersection lines in other SliceViews.

## Member Enumeration Documentation

### RenderMode2d

```cpp
enum class ImFusion::GlSliceIndicator::RenderMode2d
```

*strong*

Enumeration for selecting how to render the cross-section between this MPR and other MPR slice views.

**Enumerator:**

- `Nothing` — Don't render MPR cross-sections at all.
- `Line` — Render the actual MPR cross-section lines.
- `Thickness` — Render slice thickness indicators in MPR cross-sections.
- `OrientationMarkers` — Render orientation markers in MPR cross-sections.

### RenderMode3d

```cpp
enum class ImFusion::GlSliceIndicator::RenderMode3d
```

*strong*

Enumeration for selecting what to render inside a [GlVolumeView](/cpp-sdk/4.5/api/classes/imfusion-glvolumeview).

**Enumerator:**

- `Nothing` — Don't render MPR cross-sections at all.
- `Border` — Render the surrounding border of the slice.
- `ImageContent` — Render the actual image content.
- `Centerlines` — Render the centerlines of the slice.
- `Plane` — Render the slice as semi-transparent plane.
