## Overview

template&lt;typename T&gt;

class ImFusion::VectorParameterWrapper&lt; T &gt;
Wrap a vector into a [ParameterWrapperBase](/cpp-sdk/4.5/api/classes/imfusion-parameterwrapperbase) instance.

This class is templated to allow for vec2, vec3, vec4, [std::vector&lt;\*&gt;](http://en.cppreference.com/w/cpp/container/vector.html) of such, and the corresponding [Parameter&lt;\*&gt;](/cpp-sdk/4.5/api/classes/imfusion-parameter) types.

**Warning:** : For [Parameter&lt;\*&gt;](/cpp-sdk/4.5/api/classes/imfusion-parameter) types, each deserialize may trigger a signalChanged
