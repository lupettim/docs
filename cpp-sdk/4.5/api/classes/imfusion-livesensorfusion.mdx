## Overview

Abstract base class for 9DOF sensor fusion algorithms.

**Note:** As usual, thread safety is responsibility of the caller

## Public Member Functions

- virtual void [update](#update) (const vec3 &angularVelocityDegPerSec, const vec3 &linAcc, const vec3 &mag, double timestampSeconds)=0 — Propagates the physical state using a new measurement.
- virtual Eigen::Quaterniond [orientationQuaternion](#orientationquaternion) () const =0 — Returns the orientation of the most recent state.
- virtual vec3 [position](#position) () const =0 — Returns the position in world coords.
- virtual void [reset](#reset) ()=0 — Restores the initial state, which is at rest in the origin and oriented according to Eigen defaults.
