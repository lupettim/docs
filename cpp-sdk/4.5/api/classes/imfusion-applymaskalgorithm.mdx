
*override, virtual*

Processes a single image without modifying the algorithm's status.

For performance reasons, it is possible to do this in-place, i.e. with m_createNew=false, in which case the input image is const-cast and modified directly. Proceed with care. If you intend to run this algorithm with the StreamAlgorithmExecutor, ensure that m_createNew=true (default).
