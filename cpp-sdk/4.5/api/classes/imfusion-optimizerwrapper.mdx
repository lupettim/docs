## Overview

Wrapper around [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) and [CostFunction](/cpp-sdk/4.5/api/classes/imfusion-costfunction) classes to work on simple lambda functions.

Example usage:

```cpp
auto costFunction = [&](int n, const double* x) { return ...; };
OptimizerWrapper ow(costFunction, 3);
ow.compute();
```
 fragment

- **OptimizerWrapper** ([std::function](http://en.cppreference.com/w/cpp/utility/functional/function.html)&lt; double(int, const double \*)&gt; cf, int dim, int type=34) — Constructor with lambda cost function, dimension and optional NLopt type (BOBYQA by default).
