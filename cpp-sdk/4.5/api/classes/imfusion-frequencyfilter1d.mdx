## Overview

Apply arbitrary frequency filters to 1D input.

This class takes a filter response defined by a given lambda function and applies it to 1D time series. Filter response is defined as a spectral density multiplier for the given frequency in units of oscillations per signal duration. This function is sampled according to the resolution of the forward [FFT](/cpp-sdk/4.5/api/classes/imfusion-fft) of the input signal. The required Fourier transformations to and from frequency space are handled internally using the [CpuFFT](/cpp-sdk/4.5/api/namespaces/imfusion-cpufft) class.
