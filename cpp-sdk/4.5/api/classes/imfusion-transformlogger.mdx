## Overview

Support for logging transforms as coordinate systems in the Display.

Run the [TransformLogger](/cpp-sdk/4.5/api/classes/imfusion-transformlogger) algorithm in the ImFusionSuite to start displaying the logged transforms. Use addTranform with a matrix to display a [GlCoordinateSystem](/cpp-sdk/4.5/api/classes/imfusion-glcoordinatesystem) in the Display, with the given name

**See also:** [DataLogger](/cpp-sdk/4.5/api/classes/imfusion-datalogger)
