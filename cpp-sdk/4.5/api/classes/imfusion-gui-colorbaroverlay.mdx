## Member Function Documentation

### setLabels

```cpp
void ImFusion::GUI::ColorBarOverlay::setLabels (const std::vector < std::string > & labels)
```

Sets labels for the colormap.

The labels will be distributed evenly along the colormap, starting from the bottom.

### setLabelHeights

```cpp
void ImFusion::GUI::ColorBarOverlay::setLabelHeights (const std::vector < double > & labelHeights)
```

Sets label heights for the colormap The labels will be distributed according to the ratio w.r.t histogram height.

Each label should have a corresponding height if this is to be used.
