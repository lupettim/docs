## Overview

Converts a LabelMap to RTStructures.

An RTStructure is a [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) with an [RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent). Per label in the input, one [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) is generated.

*override, virtual*

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
