`#include <ImFusion/GUI/RadiologyViewGroup.h>`

## Overview

Convenience class to setup a set of views typically available in radiology workstations.

This class will create a total of 5 views and add them to the parent display passed during construction:

- 1 [SliceView](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview) for showing 2D data
- 3 SliceViews for showing 3D data as axial, sagittal, and coronal MPRs
- 1 [VolumeView](/cpp-sdk/4.5/api/classes/imfusion-gui-volumeview) for showing 3D data

In addition it provides a series of convenience functionality for these views:

- A set of named accessor functions to directly retrieve a specific view.
- A set of functions to modify the visible datasets in all views at once.
- A set of helper functions to perform advanced actions such as centering the views at a specific location or align the MPR views to the voxel grid.
- Customizable property linking of the 3 MPR views so that they all share the same appearance.

The relationship between [RadiologyViewGroup](/cpp-sdk/4.5/api/classes/imfusion-gui-radiologyviewgroup) and its views is bi-directional: you can call [SliceView::viewGroup()](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview) and [VolumeView::viewGroup()](/cpp-sdk/4.5/api/classes/imfusion-gui-volumeview) to query its siblings.

Since it is such a common use case, the [Display](/cpp-sdk/4.5/api/classes/imfusion-gui-display) interface provides a virtual member function Display::mainViewGroup() to be implemented by subclasses. This enables generic code to show data on the main views if available.

- int [showData](#showdata) ([Data](/cpp-sdk/4.5/api/classes/imfusion-data) \*dataset) — Appends *data* to the visible data on all compatible views of this group by calling [showData()](#showdata) on them.

- void **replaceData** ([Data](/cpp-sdk/4.5/api/classes/imfusion-data) \*which, [Data](/cpp-sdk/4.5/api/classes/imfusion-data) \*with) — Replaces all occurrence of *which* in the set of visible data with *with* in all views comprising this group.
- void **setVisibleData** (const [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) &datasets) — Replace the visible data of all views with the one in *datasets*.
- [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) **visibleData** () const — Returns the union of the currently visible data in all views comprising this group.

## Member Enumeration Documentation

### Link

```cpp
enum class ImFusion::GUI::RadiologyViewGroup::Link
```

*strong*

Enumeration of properties that can be linked between the axial, sagittal, and coronal MPR views.

**Enumerator:**

- `None` — Synchronize none of the below.
- `Everything` — Synchronize all of the below.
- `ZoomFactor` — Synchronize [SliceView::zoomFactor()](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview).
- `ZoomMode` — Synchronize [SliceView::zoomMode()](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview).
- `Interpolation` — Synchronize [SliceView::interpolationMode()](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview).
- `Flip` — Synchronize SliceView::flipY().
- `BlendingMode` — Synchronize [SliceView::blendingMode()](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview).
- `BlendingFactor` — Synchronize [SliceView::blendingFactor()](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview).

### setMprViewMatrices

```cpp
void ImFusion::GUI::RadiologyViewGroup::setMprViewMatrices (const mat4 & matrix, bool suppressAnimation = false)
```

Sets the matrix of each MPR view according to their original plane.

The rotation part of *matrix* sets the rotation of the axial view, the sagittal and coronal views according to their [GlSliceView::originalPlaneMatrix()](/cpp-sdk/4.5/api/classes/imfusion-glsliceview). The tranlational part of *matrix* defines the intersection point of the three MPR views.
