## Signals

- void **checked** (const [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) \*object, const vec3 &color) — Emitted when the checkbox is checked, for adding a corresponding bar in the [GlLiftChartOverlay](/cpp-sdk/4.5/api/classes/imfusion-glliftchartoverlay).

- void **colorChanged** (const [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) \*object, const vec3 &color) — Emitted only when the color changes.
