`#include <ImFusion/GL/ParametricDeformation.h>`

## Overview

[Base](/cpp-sdk/4.5/api/structs/imfusion-dicom-base) interface for a parametric deformation in 2D or 3D.

Derived classes will implement actual parametric deformation models. Additionally to the displacement method, the shader include is expected to have the following two methods:

```cpp
// Returns the control point with the index i
vec3 controlPointLocation(int i);
// Returns the displacement of the control point with index i
vec3 controlPointDisplacement(int i);
```

- virtual void [setDisplacement](#setdisplacement) (const [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; vec3f &gt; &d) — Set displacement in mm for all control points if applicable For 2D images the z-coordinate of the displacement is ignored.

- virtual [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; vec3 &gt; [controlPoints](#controlpoints) () const — Returns a list of all control points in world coordinates (might be empty for certain models).
- virtual int [numberOfControlPoints](#numberofcontrolpoints) () const — Returns the number of control points (may be zero for certain deformations).
- virtual int [numberOfParameters](#numberofparameters) () const — Returns the number of parameters e.g. for optimization.
