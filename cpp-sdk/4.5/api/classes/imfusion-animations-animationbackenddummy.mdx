`#include <ImFusion/GUI/Animations.h>`

## Overview

Dummy implementation of the [AnimationBackendBase](/cpp-sdk/4.5/api/classes/imfusion-animations-animationbackendbase) interface that will not execute any animation but only call its update() function exactly once with t=1.0.

You can use this backend in the absence of a suitable event loop
