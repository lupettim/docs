`#include <ImFusion/GL/BakeTransformationAlgorithm.h>`

## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for baking in an image transformation while adjusting the size to encompass the full transformation.

## Public Member Functions

- **BakeTransformationAlgorithm** ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) \*imgs) — Creates the algorithm instance with a data.
- **~BakeTransformationAlgorithm** () override — Deletes owned resources.
- void [configure](#configure) (const [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) \*p) override — Apply configuration.
- void [configuration](#configuration) ([Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) \*p) const override — Configures the algorithm.
- void **setInput** ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) \*newInput) — Replaces the input of the algorithm.
- void **setNearestInterpolation** (bool val=false) — Configures whether nearest interpolation is used, or linear in case of false.
- void **setCreateNew** (bool createNew) — Configures whether the algorithm should create a new image.
- void **setForceCPU** (bool forceCPU) — Configures whether the algorithm should run on the CPU (by default it will be executed on the GPU).
- void **setAdjustSize** (bool val=true) — Configures whether the resulting image should adjust its size to encompass the transformation.
- void **setSamplerWrapping** ([GL::Sampler::Wrap](/cpp-sdk/4.5/api/classes/imfusion-gl-sampler) samplerWrapping) — Set sampler wrapping.
- void **setBorderColor** (const vec4f &color) — Set border color in case sampler wrapping is ClampToEdge.
- [std::unique_ptr](http://en.cppreference.com/w/cpp/memory/unique_ptr.html)&lt; [SharedImage](/cpp-sdk/4.5/api/classes/imfusion-sharedimage) &gt; **applyTo** (int i) — Apply the transformation to [SharedImage](/cpp-sdk/4.5/api/classes/imfusion-sharedimage) i in the [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) given in the constructor returns a pointer to a newly created [SharedImage](/cpp-sdk/4.5/api/classes/imfusion-sharedimage), ensure that it gets owned by a unique_ptr.

## Static Public Member Functions

- static bool **hasRotation** ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) &img) — Function to see if supplied shared image set actually has a rotation, if not algorithm is not needed.
- static bool [canBakeTransformation](#canbaketransformation) ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) &img) — Returns true if the algorithm is able to bake the transformation of the input [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset).
