## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to generate learning data (heatmaps or label maps) from the position of landmarks in images.

If matrices are provided in the constructors, input point clouds are assumed to be in world space coordinates. Otherwise, they will be treated as image coordinates (still in mm).

## Member Enumeration Documentation

### Mode

```cpp
enum class ImFusion::LandmarksToImageAlgorithm::Mode
```

*strong*

**Enumerator:**

- `Labels` — Binary label map of spheres with specified radius around each landmark.
- `Weights` — The above followed by Gaussian smoothing, as well as chosen background value.
- `Vectors` — Voxel-wise vectors.
- `WeightsPrecise` — Smooth blob creation by setting single pixels and smoothing them, creating equally sized blobs.
