## Member Function Documentation

### registerRenderer

```cpp
void ImFusion::RendererFactory::registerRenderer (SliceRendererFactoryFunction func)
```

*static*

Registers the given [GlSliceRenderer](/cpp-sdk/4.5/api/classes/imfusion-glslicerenderer) to be used in GlSliceViews.

Example usage:

```cpp
RendererFactory::registerRenderer([](MaskRenderingDecorator& maskDecorator, bool is3d) -> std::unique_ptr<GlSliceRenderer> {
 return (is3d ? std::make_unique<GlSliceRendererSweep>(maskDecorator) : return nullptr);
});
```
 fragment

**Note:** The selection of the renderer at runtime is handled by [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) evaluating [GlSliceRenderer::accepts()](/cpp-sdk/4.5/api/classes/imfusion-glslicerenderer). Individual images can be rendered by different renderers.

### registerRenderer

```cpp
void ImFusion::RendererFactory::registerRenderer (const std::string & name, VolumeRendererFactoryFunction factoryFunc, CapabilityCheckFunction capabilityCheckFunc = []() { return true;})
```

*static*

Registers the given [GlVolumeRenderer](/cpp-sdk/4.5/api/classes/imfusion-glvolumerenderer) to be used in GlVolumeViews.

Example usage:

```cpp
RendererFactory::registerRenderer("Global Illumination Volume Renderer",
    []() -> std::unique_ptr<GlVolumeRenderer> { return std::make_unique<GlVolumeRendererGlobalIllum>(); },
    []() { return GLAD_GL_VERSION_4_3 > 0; });
```
 fragment

**Note:** The selection of the renderer at runtime is handled by the user. All images are rendered by the same renderer in a single pass.
