`#include <ImFusion/GUI/ScaleOverlay.h>`

## Overview

Wraps a GlScaleOverlay into a [ViewOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-viewoverlay) so that it can be added to a [GUI::View](/cpp-sdk/4.5/api/classes/imfusion-gui-view).

Currently does not provide any user interaction.

- void **setVertical** (bool state) — Set whether this scale uses vertical orientation.
