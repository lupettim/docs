## Overview

VOI LUT [Module](/cpp-sdk/4.5/api/classes/imfusion-dicom-module) (PS 3.3 C.11.2) Loads and saves windowing related values to and from [DisplayOptions2d](/cpp-sdk/4.5/api/classes/imfusion-displayoptions2d).

**See also:** [http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.11.2.html](http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.11.2.html)[DisplayOptions2d](/cpp-sdk/4.5/api/classes/imfusion-displayoptions2d)

*override, virtual*

Loads windowing parameters and stores them in the [DisplayOptions2d](/cpp-sdk/4.5/api/classes/imfusion-displayoptions2d) for the given image.

Per-frame windowing is not supported and only the windowing of the center frame of a volume will be used.
