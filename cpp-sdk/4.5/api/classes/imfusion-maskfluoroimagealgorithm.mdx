## Member Enumeration Documentation

### Mode

```cpp
enum class ImFusion::MaskFluoroImageAlgorithm::Mode
```

*strong*

**Enumerator:**

- `ImageIntensifier` — Images from image intensifier based C-Arms have a circular shape.
- `FlatField` — Images from flat-field based systems have a square shape, with potentially additional collimation.

*override, virtual*

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
