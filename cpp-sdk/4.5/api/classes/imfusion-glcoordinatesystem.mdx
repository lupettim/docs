`#include <ImFusion/GL/GlCoordinateSystem.h>`

## Overview

[GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) to render an illustration of a coordinate system into the scene.

For each of the three main axes, renders a 3D arrow as well as a label.

## Public Member Functions

- double **scaling** () const — Returns the scaling factor for the rendered axes.
- void **setScaling** (double value) — Sets the scaling factor for the rendered axes.
- bool **showLabels** () const — Returns whether the labels are shown.
- void **setShowLabels** (bool value) — Sets whether to show the labels.
- bool **drawAxesNegative** () const — Returns whether the axes are drawn in the negative range.
- void **setDrawAxesNegative** (bool value) — Sets whether to draw the axes in the negative range.
- void **setColor** (int idx, const vec3 &c) — Sets the rendering color of the specified axis and its label.
