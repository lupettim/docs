## Overview

## Member Enumeration Documentation

### GradientFilter

```cpp
enum ImFusion::FiducialExtraction::GradientFilter
```

**Enumerator:**

- `None` — Use all gradients.
- `Outward` — Only use gradients pointing away from the patch center.
- `Inward` — Only use gradients pointing towards the patch center.
