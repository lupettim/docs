## Overview

[DataDisplayHandler](/cpp-sdk/4.5/api/classes/imfusion-datadisplayhandler) for PointClouds that represent DICOM RT Structs.

A [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) is considered an RT Struct if it has a [RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent).

- bool [canBeShownInView](#canbeshowninview) (const [Data](/cpp-sdk/4.5/api/classes/imfusion-data) \*data, const [GUI::View](/cpp-sdk/4.5/api/classes/imfusion-gui-view) &view) const override — Only supports [ImageView2D](/cpp-sdk/4.5/api/classes/imfusion-imageview2d) and [ImageView3D](/cpp-sdk/4.5/api/classes/imfusion-imageview3d).
- void [show](#show) ([Data](/cpp-sdk/4.5/api/classes/imfusion-data) \*data, [GUI::View](/cpp-sdk/4.5/api/classes/imfusion-gui-view) &view) override — Adds RTStructureAnnotations to [ImageView2D](/cpp-sdk/4.5/api/classes/imfusion-imageview2d) and [ImageView3D](/cpp-sdk/4.5/api/classes/imfusion-imageview3d).
