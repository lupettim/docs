`#include <ImFusion/Base/CombineImagesAlgorithm.h>`

## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to combine multiple images into a new shared image set.

**Note:** [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) and DataComponents of the first input image are copied to the output image.
