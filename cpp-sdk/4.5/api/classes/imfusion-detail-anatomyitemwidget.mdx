## Signals

- void **checked** (int64_t anatomyId) — Emitted when the checkbox is checked, for adding the anatomy to the current list.
- void **unchecked** (int64_t anatomyId) — Emitted when the checkbox is unchecked or when the remove button is clicked, for removing the anatomy from the current list.
