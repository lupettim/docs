`#include <ImFusion/Dicom/DicomModule.h>`

## Overview

A stub that allows to use a [Module](/cpp-sdk/4.5/api/classes/imfusion-dicom-module) with a DcmItem alone (without deriving some class from [ModuleList](/cpp-sdk/4.5/api/classes/imfusion-dicom-modulelist)).

```cpp
DcmItem& item = ...;
Dicom::DatasetModuleList moduleParent(item);
Dicom::CommonModule common(moduleParent, Dicom::Module::Type::UserOption);
```
