## Overview

Collection of the essential tags describing a [Dicom](/cpp-sdk/4.5/api/namespaces/imfusion-dicom) data set combining.

- [Patient](/cpp-sdk/4.5/api/structs/imfusion-dicom-patient)[Module](/cpp-sdk/4.5/api/classes/imfusion-dicom-module) (PS 3.3 C.7.1.1)
- General [Study](/cpp-sdk/4.5/api/structs/imfusion-dicom-study)[Module](/cpp-sdk/4.5/api/classes/imfusion-dicom-module) (PS 3.3 C.7.2.1)
- General [Series](/cpp-sdk/4.5/api/structs/imfusion-dicom-series)[Module](/cpp-sdk/4.5/api/classes/imfusion-dicom-module) (PS 3.3 C.7.3.1)

  **See also:** [http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.html#sect_C.7.1.1](http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.html#sect_C.7.1.1)[http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.2.html#sect_C.7.2.1](http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.2.html#sect_C.7.2.1)[http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.3.html#sect_C.7.3.1](http://dicom.nema.org/medical/dicom/current/output/chtml/part03/sect_C.7.3.html#sect_C.7.3.1)[ImageInfoDataComponent](/cpp-sdk/4.5/api/classes/imfusion-imageinfodatacomponent)[AnatomyDataComponent](/cpp-sdk/4.5/api/classes/imfusion-anatomydatacomponent)

## Member Function Documentation

### load

```cpp
void ImFusion::Dicom::CommonModule::load (SharedImageSet & sis) const
```

Load data from the module and store it in a [ImageInfoDataComponent](/cpp-sdk/4.5/api/classes/imfusion-imageinfodatacomponent) of *sis*.

Missing values will be set to empty strings in the datacomponent. If the dataset contains anatomical data, it is stored in a [AnatomyDataComponent](/cpp-sdk/4.5/api/classes/imfusion-anatomydatacomponent).

### load

```cpp
void ImFusion::Dicom::CommonModule::load (ImageInfoDataComponent & iidc) const
```

Load data from the module into the [ImageInfoDataComponent](/cpp-sdk/4.5/api/classes/imfusion-imageinfodatacomponent).

Missing values will be set to empty strings in the datacomponent. This is automatically called by the above load method for [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) and should only be used for non-image data (such as PointClouds).

### save

```cpp
void ImFusion::Dicom::CommonModule::save (const ImageInfoDataComponent * iidc)
```

Writes the [ImageInfoDataComponent](/cpp-sdk/4.5/api/classes/imfusion-imageinfodatacomponent) to the dataset.

This is automatically called by the above save method for [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) and should only be used for non-image data (such as PointClouds).

### saveMandatory

```cpp
void ImFusion::Dicom::CommonModule::saveMandatory (const std::string & sopClassUID)
```

Writes all remaining mandatory tags not covered by save(ImageInfoDataComponent).

This is automatically called by the above save method for [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) and should only be used for non-image data (such as PointClouds).
