## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for computing solutions to (autonomous) ordinary differential equations (ODEs).

This class is designed to solve ODEs of type x'(t) = f(x(t)), where the right-hand side is given by a 2D or 3D vector field. The flow of this vector field (i.e. the solutions of this ODE, with initial value given by pixel centers) is computed and stored in a displacement field. Optionally, the vector field can be post-composed with another deformation.

## Constructor & Destructor Documentation

### VectorFieldFlowAlgorithm

```cpp
ImFusion::VectorFieldFlowAlgorithm::VectorFieldFlowAlgorithm (SharedImageSet & imgIn, SharedImageSet * output, GlDenseDeformation * initialValues = nullptr, GlDenseDeformation * other = nullptr)
```

Constructor.

**Parameters:**

- `imgIn` — [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) input with vector field representing the right-hand side of the ODE.
- `output` — Output [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) where the computed solution is stored. (If nullptr is provided, an explicit output will be returned.)
- `initialValues` — Optional initial values for the ODE solution.
- `other` — Optional additional deformation to be composed with the vector field. The constructor requires imgIn to have a deformation and output to have a [GlDenseDeformation](/cpp-sdk/4.5/api/classes/imfusion-gldensedeformation), or it will throw an exception. The constructor will also throw an exception, if the compilation of the shader fails. Note, the existing deformation of output will be ignored. If the deformation should be taken into account, it has to be passed via the initialValues argument.

## Member Function Documentation

### compute

```cpp
void ImFusion::VectorFieldFlowAlgorithm::compute ()
```

*override, virtual*

Solve the ODE with rhs given by m_vectorField and initial values m_initialValues.

Optionally post-compose the vector field with the deformation in m_other. This function implements an approximation to the operator other \* exp(vectorField) \* initialValues, where \* denotes composition of deformations and exp() is the time-1 flow operator.

### setSharedImageSetForMask

### takeOutput

```cpp
OwningDataList ImFusion::VectorFieldFlowAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
