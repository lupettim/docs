## Protected Attributes

- [GL::ImageProgram](/cpp-sdk/4.5/api/classes/imfusion-gl-imageprogram) \* **m_kernelProgram** — Shader program for computation.
- [GlImage](/cpp-sdk/4.5/api/classes/imfusion-glimage) \* **m_guide** — Optional guide image for joint bilateral filtering (see refs. in BilateralFilterAlgorithm.h). Not owned.
- [GlImage](/cpp-sdk/4.5/api/classes/imfusion-glimage) \* **m_imgAux** — Auxiliary image for in-place filtering.
- [GL::Sampler](/cpp-sdk/4.5/api/classes/imfusion-gl-sampler) **m_samplingParameters** — Sampling settings for the input and guide textures.
- double **m_sigmaIntensity** — Width of intensity Gaussian weighting function.
- double **m_sigmaPosition** — Width of spatial Gaussian weighting function (in pixels).
- double **m_kernelRadiusScaling** — Defines ration of spatial kernel radius and sigmaPosition.
- bool **m_adaptive**
- double **m_adaptiveOffset**
- double **m_adaptiveIncline**
- bool **m_skipOutsidePixels** = true — If enabled, areas outside the image have zero weight.
