`#include <ImFusion/GUI/PropertiesListWidget.h>`

## Overview

The [PropertiesListWidget](/cpp-sdk/4.5/api/classes/imfusion-propertieslistwidget) shows as sub-properties as a list.

All sub-properties of a [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) object are treated as items of the list and are expected to have the same parameter structure. List items can be removed and edited. New list items can be added. Further sub-properties of the sub-properties are not represented by further PropertiesListWidgets.

## Member Function Documentation

### setAddTemplate

```cpp
void ImFusion::PropertiesListWidget::setAddTemplate (const Properties & p)
```

Sets a template that is used when adding a new item.

If the template is empty (the default), adding new items is not possible.

### setForceTemplateParamType

```cpp
void ImFusion::PropertiesListWidget::setForceTemplateParamType (bool enable)
```

If enabled, each params type and type attribute is overridden with the corresponding value from the template (if available).

Default is disabled.

### configure
