## Overview

The [GlDeformationGrid](/cpp-sdk/4.5/api/classes/imfusion-gldeformationgrid) visualizes the deformation of an image through the deformation of a grid rendered on top of the image.

Since the deformation of the input image must be inverted internally (see [InvertDeformationAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-invertdeformationalgorithm)), this visualization is rather expensive.

### downsamplingFactor

```cpp
int ImFusion::GlDeformationGrid::downsamplingFactor () const
```

Downsampling factor of the internal inverted [GlDenseDeformation](/cpp-sdk/4.5/api/classes/imfusion-gldensedeformation).

Higher downsampling results in faster computation but in a less accurate visualization. The initial value depends on the size of the input image.

### updateInterval

```cpp
std::chrono::milliseconds ImFusion::GlDeformationGrid::updateInterval () const
```

Duration between two updates of the internal [GlDenseDeformation](/cpp-sdk/4.5/api/classes/imfusion-gldensedeformation).

A value of 0 updates on every rendering and blocks during the computation of the inverse deformation. A value greater 0 computes the inverse async. The initial value depends on the size of the input image.
