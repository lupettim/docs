### takeOutput

```cpp
OwningDataList ImFusion::NiftiIoAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).

```cpp
bool ImFusion::NiftiIoAlgorithm::didLoadLeftHandedCoordinatesFile () const
```

*inline*

Returns whether the file loaded was using a left-handed coordinate system The result of this function is only meaningful when.

1. using this algorithm in Read mode
2. [compute()](#compute) was already called and the loading actually succeeded

### canLoad

```cpp
bool ImFusion::NiftiIoAlgorithm::canLoad (const Filesystem::Path & filepath) const
```

*override, virtual*

Returns whether the algorithm is suitable for loading the indicated file.

Default implementation simply compares the file extension. If m_forceCannotLoad is true, it returns false, which allows disabling the [IoAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-ioalgorithm) in the global settings.
