## Overview

Annotation that renders a [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) with a [RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent).

All the rendering data is stored in a [RTStructureRenderingDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructurerenderingdatacomponent), so this object is cheap to create. In a slice view, the annotation renders the original outline if the slice matches the plane of the outline. Otherwise it will render a mesh if available. In the 3D view, it will always render the mesh if available.
