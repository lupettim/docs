## Overview

Basic sensor fusion.

Basic version, plain integration of the gyro without any magnetometer information. Requires a recently zeroed in gyro, or will run away rather quickly

## Public Member Functions

- void [update](#update) (const vec3 &angularVelocityDegPerSec, const vec3 &linAcc, const vec3 &mag, double timestampSeconds) override — Propagates the physical state using a new measurement. Ignores mag.
- Eigen::Quaterniond [orientationQuaternion](#orientationquaternion) () const override — Returns the orientation of the most recent state.
- vec3 [position](#position) () const override — Returns the position in world coords.
- void [reset](#reset) () override — Restores the initial state.
