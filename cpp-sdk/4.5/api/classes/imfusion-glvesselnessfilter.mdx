## Overview

Computes a multi-scale vesselness measure based on Frangi's filter.

The method is based on analysing the hessian eigenvalues computed at each voxel at multiple scales. The likelihood that a given voxel belongs to a vessel-like structure is formulated as a function of the hessian eigenvalues (Frangi's score).

The multi-scale analysis is based on space-scale derivatives theory. Instead of resampling the image at the desired scale and computing the hessian, one smooths the image with a Gauss kernel with a size of the corresponding scale and then computes the derivative. Since Gauss convolutions commute with derivatives, one can simply filter the image with a derivative of the Gauss kernel at the desired scale.

The final vesselness score is obtained by maxing the set of scores for each scale.

A. Frangi et al. "Multiscale vessel enhancement filtering", MICCAI '98
