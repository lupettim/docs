`#include <ImFusion/Seg/GlIsolineRenderer.h>`

## Overview

[GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) to render a label map into a [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview) using isolines.

The current implementation only supports label maps of unsigned byte type.

**Note:** This class trades performance for accuracy and will only render an approximation of the isolines. If you need pixel-perfect isolines, you will need to run marching squares yourself and render the resulting geometry as lines.
