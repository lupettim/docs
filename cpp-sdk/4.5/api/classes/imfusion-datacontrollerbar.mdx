## Overview

The [DataControllerBar](/cpp-sdk/4.5/api/classes/imfusion-datacontrollerbar) displays [DataController](/cpp-sdk/4.5/api/classes/imfusion-datacontroller) in toolbar-mode.

This class is usually not used directly but automatically by the [DataController](/cpp-sdk/4.5/api/classes/imfusion-datacontroller). If it was created with a QToolBar, the [DataControllerBar](/cpp-sdk/4.5/api/classes/imfusion-datacontrollerbar) will automatically place DataControllers into the QToolBar if it has sufficient free space.
