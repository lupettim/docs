`#include <ImFusion/Mesh/MeshBooleanOperationsAlgorithm.h>`

## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for performing boolean operations on meshes.

Meshes must be watertight. The algorithm internally will only check if the meshes are manifold and closed but it will not check for self-intersections. When in doubt, use [Mesh::isSelfIntersecting](/cpp-sdk/4.5/api/classes/imfusion-mesh)

## Member Enumeration Documentation

### Mode

```cpp
enum ImFusion::MeshBooleanOperationsAlgorithm::Mode
```

Enumeration listing supported boolean operations.

**Enumerator:**

- `UNION` — Compute union between first and second mesh.
- `DIFFERENCE` — Subtract second from first mesh.
- `INTERSECTION` — Extract portion common to both, first and second mesh.

### StatusExt

```cpp
enum class ImFusion::MeshBooleanOperationsAlgorithm::StatusExt
```

*strong*

Enumeration listing possible errors during the computation.

**Enumerator:**

- `NonManifoldInput` — At least one of the input meshes is non-manifold.
- `EmptyOutput` — The boolean operation applied to meshes resulted in an empty mesh.
