## Overview

Hierarchical model of DICOM patients.

The model has the following hierarchy:

- patients
  - studies
    - series
      - images

[Series](/cpp-sdk/4.5/api/structs/imfusion-dicom-series) can be selected with their Qt::CheckStateRole. Patients and Studies reflect the checked states of their children, e.g. can be partially checked. The actual scanning and loading of DICOM data is performed internally by a [Dicom::Scanner](/cpp-sdk/4.5/api/classes/imfusion-dicom-scanner).

Minimal Example:

```cpp
#include <ImFusion/Dicom/GUI/DicomBrowserModel.h>

#include <QtWidgets/QBoxLayout>
#include <QtWidgets/QDialog>
#include <QtWidgets/QProgressBar>
#include <QtWidgets/QTreeView>

class DicomBrowser : public QDialog
{
    Q_OBJECT
public:
    DicomBrowser(const QString& directory, QWidget* parent = nullptr);
};
```
 fragment
```cpp
DicomBrowser::DicomBrowser(const QString& directory, QWidget* parent)
    : QDialog(parent)
{
 auto model = new ImFusion::DicomBrowserModel(nullptr, this);

 auto view = new QTreeView(this);
    view->setModel(model);

 auto progress = new QProgressBar(this);
    connect(model, &ImFusion::DicomBrowserModel::scanProgressRangeChanged, progress, &QProgressBar::setRange);
    connect(model, &ImFusion::DicomBrowserModel::scanProgressValueChanged, progress, &QProgressBar::setValue);
    model->scanDirectoryAsync(directory, true, false, true);

 auto layout = new QVBoxLayout(this);
    layout->addWidget(view);
    layout->addWidget(progress);
    setLayout(layout);
}
```

### Roles

```cpp
enum ImFusion::DicomBrowserModel::Roles
```

**Enumerator:**

- `ParameterRole` — Role to retrieve all parameters of an index as a ParameterList.
- `SortRole` — Special role used to compare indices (e.g. patients by name). Values should be unique to assure stable sorting.
- `PatientRole` — Retrieves the Patient\* object from an index or QVariant() if index is not a patient.
- `StudyRole` — Retrieves the Study\* object from an index or QVariant() if index is not a study.
- `SeriesRole` — Retrieves the Series\* object from an index or QVariant() if index is not a series.
- `ImageRole` — Retrieves the Image\* object from an index or QVariant() if index is not an image.
- `PacsSeriesProxyRole` — Retrieves the PacsSeriesPlaceholder\* object from an index or QVariant() if index is not an image.

### setUpdateSeriesDimensionsAfterScan

```cpp
void ImFusion::DicomBrowserModel::setUpdateSeriesDimensionsAfterScan (bool enable)
```

Sets whether the kind field of each [Series](/cpp-sdk/4.5/api/structs/imfusion-dicom-series) is updated automatically after scanning.

Since this needs to iterate over all files of each series, this can take about as long as scanning itself. In the case of the scan...Async methods, the scanProgress will be used to report the progress of the update (starting from 0 again after the scan has finished but with a different text). Disabled by default.

### scanDirectoryAsync

```cpp
void ImFusion::DicomBrowserModel::scanDirectoryAsync (const QString & directory, bool recursive, bool useDicomdirOnly, bool checkDicomHeader)
```

*slot*

Scans a directory for dicom files.

This method is executed async and returns immediately. When the scanner finds a series, it is not instantly added to the model but only the next time the Qt mainthread processes events. This means that the control flow must pass to the Qt main event loop at some point, otherwise patients() might still be empty even when isScanning() returns false. If you cannot return to the Qt main event loop use scanDirectoryBlocking instead. Check the class documentation for a common usage example.

### scanPacsAsync

```cpp
void ImFusion::DicomBrowserModel::scanPacsAsync (DicomPacsConfiguration pacs, const QString & patientNameFilter, const QString & studyFilter)
```

*slot*

Scans the PACS server with the given configuration for existing patients and studies.

Will only download Patient and [Study](/cpp-sdk/4.5/api/structs/imfusion-dicom-study) data. [Series](/cpp-sdk/4.5/api/structs/imfusion-dicom-series) and [Image](/cpp-sdk/4.5/api/classes/imfusion-image) data will not be received in order to preserve bandwidth and time. Use getPacsSeries() to receive all [Series](/cpp-sdk/4.5/api/structs/imfusion-dicom-series) for a given [Study](/cpp-sdk/4.5/api/structs/imfusion-dicom-study).
