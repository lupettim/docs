## Overview

[Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) manipulator with 3d controls.

The [MeshManipulator](/cpp-sdk/4.5/api/classes/imfusion-meshmanipulator) can modify the translation, rotation and scaling of a mesh. For each transformation mode it offers controls that are displayed around the mesh center.

For proper use the [MeshManipulator](/cpp-sdk/4.5/api/classes/imfusion-meshmanipulator) object must be wrapped inside an [InteractiveObject](/cpp-sdk/4.5/api/classes/imfusion-interactiveobject) before being added to the corresponding view, e.g.

```cpp
InteractiveView* view = ...;
Mesh* mesh = ...
auto manip = std::make_unique<MeshManipulator>(mesh);
auto interactiveObject = std::make_unique<InteractiveObject>(&mesh, manip.get(), false);
view->addObject(interactiveObject.get());
```
 fragment
The manipulator must be removed again before deleting!

**See also:** [InteractiveObject](/cpp-sdk/4.5/api/classes/imfusion-interactiveobject)
