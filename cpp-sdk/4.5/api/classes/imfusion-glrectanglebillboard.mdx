`#include <ImFusion/GL/GlRectangleBillboard.h>`

## Overview

Renders a fixed size rectangle billboard in the world, always facing towards the camera.

The size of the rectangle is specified in pixels, for the size of the side of the rectangle. It is always looking towards the view, without depth test, at the position in the world defined with [GlObject::setMatrix](/cpp-sdk/4.5/api/classes/imfusion-globject). It can be rotated in the view by applying a rotation along the Z axis on the given matrix.
