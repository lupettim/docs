## Overview

Anisotropic diffusion filter using the explicit scheme.

## Public Member Functions

- **GlAnisotropicDiffusionFilter** ([GL::Framebuffer](/cpp-sdk/4.5/api/classes/imfusion-gl-framebuffer) \*fbo=0)
- bool **setSmoothingKernelSize** (int size) — Set smoothing kernel size. Default is 7.
- void **setIterations** (int iterations) — Set iterations. Default is 30.
- void **setTimeStep** (float step) — Set time step. Has to be smaller than 0.25 for 2D and smaller than 0.167 for 3D. Default is 0.20.
- void **setContrast** (float contrast) — Set contrast in normalized intensities. Default is 0.01.
- bool **setSharpenKernelSize** (int size, float strength) — Set sharpening kernel size and strength. Default is 7 and 0 (= sharpen disabled).
- void [compute](#compute) (const [GlImage](/cpp-sdk/4.5/api/classes/imfusion-glimage) &imgIn, [GlImage](/cpp-sdk/4.5/api/classes/imfusion-glimage) &imgOut) override — Execute the filter on the image.
- bool [good](#good) () const override — Return if the filter can be used, in particular if the shaders are available.
- bool **avoidsGlDriverTimeout** () const — Returns the flag whether to avoid a OpenGL driver timeout when using large kernels.
- void **setAvoidGlDriverTimeout** (bool value) — Sets the flag whether to avoid a OpenGL driver timeout when using large kernels.
