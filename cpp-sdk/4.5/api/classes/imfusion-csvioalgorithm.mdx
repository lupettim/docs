`#include <ImFusion/IO/CsvIoAlgorithm.h>`

## Overview

IO algorithm for moving tensors to and from CSV files.

Input needs to be rank 1 and will be converted to double internally. Supports multiple tensors at once, as long as all have the same size. Each column is one tensor, separated by semicolon.

### takeOutput

```cpp
OwningDataList ImFusion::CsvIoAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
