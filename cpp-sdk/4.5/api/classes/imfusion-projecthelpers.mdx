## Member Function Documentation

### copyLocalProject

```cpp
std::unique_ptr < Project > ImFusion::ProjectHelpers::copyLocalProject (const QString & from, const QString & to, const std::optional < std::set < QString > > & identifiersToPreserve = std::nullopt)
```

*static*

Copies a local project from one folder (from) to another folder (to) A list of identifiers to be copied can be passed.

If std::nullopt is passed, all identifiers are copied. If an empty set is passed, none of the identifiers are copied.

### splitProjectPathIntoServerAndId

```cpp
std::pair < std::string, int > ImFusion::ProjectHelpers::splitProjectPathIntoServerAndId (const std::string & path)
```

*static*

Splits the projectPath of a remote [Project](/cpp-sdk/4.5/api/classes/imfusion-project) into its server url and the ID of the project.

Must only be used with projectPath in the form of: http(s)://&lt;hostname-and-port&gt;/api/v1/projects/&lt;project_id&gt; It returns the server url (without the /api... path) and the project id.
