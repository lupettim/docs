## Overview

QWidget allowing for selecting 2 components of the HSV color space.

Renders the color gradient of the two selected components and offers user interaction to select the corresponding color underneath. The third component of the HSV color space is fixed.

**See also:** [HsvHueSlider](/cpp-sdk/4.5/api/classes/imfusion-hsvhueslider), [OpacitySlider](/cpp-sdk/4.5/api/classes/imfusion-opacityslider), [ColorPickerWidget](/cpp-sdk/4.5/api/classes/imfusion-colorpickerwidget), [ColorButton](/cpp-sdk/4.5/api/classes/imfusion-colorbutton)
