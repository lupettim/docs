`#include <ImFusion/GL/GlFlattenTrackingDeformation.h>`

## Overview

[Deformation](/cpp-sdk/4.5/api/classes/imfusion-deformation) model based on flattening a tracking sequence.

This class takes a tracking sequence, assumes every matrix to represent a plane, and creates a 3D deformation that maps them to parallel equidistant planes along the z dimension. Use cases are for example to flatten a 3D CT/MRI volume along a catheter with ultrasound imaging or video, similar to curved MPRs.
