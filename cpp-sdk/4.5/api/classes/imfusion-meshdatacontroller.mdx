### supportsToolBar

```cpp
bool ImFusion::MeshDataController::supportsToolBar () const
```

*inline, override, protected, virtual*

Subclasses can return true if they support showing their [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) in [Location::ToolBar](/cpp-sdk/4.5/api/classes/imfusion-controller).

This interface is used by the default implementation of [supportedLocations()](/cpp-sdk/4.5/api/classes/imfusion-datacontroller) and [preferredLocation()](/cpp-sdk/4.5/api/classes/imfusion-datacontroller) to decide whether to use [Location::ToolBar](/cpp-sdk/4.5/api/classes/imfusion-controller).
