## Overview

Registers two images with a non-linear deformation based on TPSs.

The algorithm modifies the [GlTPSDeformation](/cpp-sdk/4.5/api/classes/imfusion-gltpsdeformation) object attached to the moving image (see also setMoving). Because TPS deformations are generally not invertible, only one image can have a deformation at a time.

### deformation

```cpp
std::shared_ptr < ParametricDeformation > ImFusion::TPSImageRegistration::deformation () const
```

Returns the deformation object that is currently modified.

This is a convenience method that casts moving to a [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset), retrieves its [deformation](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) and cast that to a [GlTPSDeformation](/cpp-sdk/4.5/api/classes/imfusion-gltpsdeformation). Returns 0, if any of the casts fail.
