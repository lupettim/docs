## Overview

[HistogramIntensityClusteringAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-histogramintensityclusteringalgorithm).

This algorithm computes a segmentation of the input image based on a intensity clustering. In order to perform this the histogram of the image is computed and local minima in the histogram are used to split the intensity clusters. Before the detection the histogram is smoothed and downsampled in order to make the detection more robust w.r.t. outliers. The labels are sorted in ascending order w.r.t. the intensities, i.e. the label 0 corresponds to the cluster with the lowest intensities. In addition the algorithm also provides a getter for the label indices sorted ascending w.r.t. the label size.
