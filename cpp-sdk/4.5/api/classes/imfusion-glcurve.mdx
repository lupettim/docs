`#include <ImFusion/GL/GlCurve.h>`

## Overview

[Spline](/cpp-sdk/4.5/api/classes/imfusion-spline) annotation.

Renders a spline. Splines are parameterized in the range [0,1]. Uses a y(x) cubic Hermite spline.
