`#include <ImFusion/GUI/HistogramOverlay.h>`

## Overview

[ViewOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-viewoverlay) to show an intensity histogram of the first shown image in a [SliceView](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview).

Additionally, it shows a the visual representation of the datasets window/level as configured in its [DisplayOptions2d](/cpp-sdk/4.5/api/classes/imfusion-displayoptions2d). Basic mouse interaction allows for manipulating window/level as well for configuring the overlay itself through a context menu.

**See also:** GlHistogramOverlay
