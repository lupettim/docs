### dimensionIndexValues

```cpp
Element < std::vector <int> > ImFusion::Dicom::EnhancedMultiFrameImageIOD::FunctionalGroupMacro::FrameContentSequenceItem::dimensionIndexValues
```

DimensionIndexSequence indices of the frame.

These means that frames with the same number in a slot are equivalent in what this [DimensionIndexSequenceItem](/cpp-sdk/4.5/api/classes/imfusion-dicom-enhancedmultiframeimageiod-dimensionindexsequenceitem) represents, e.g. they show the same segmentation label if that [DimensionIndexSequenceItem](/cpp-sdk/4.5/api/classes/imfusion-dicom-enhancedmultiframeimageiod-dimensionindexsequenceitem) points to ReferencedSegmentNumber. First number in this vector corresponds to the first element in DimensionIndexSequence and so on. The values for each "dimension" are starting at 1 and consecutive.
