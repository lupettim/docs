## Overview

Specialization of [std::istream](http://en.cppreference.com/w/cpp/io/basic_istream.html) to easily extract data from a [ByteBufferView](/cpp-sdk/4.5/api/classes/imfusion-bytebufferview).

```cpp
ByteBuffer buffer = ...
ByteBufferStream bufferStream(input.view());
std::string line;
while (std::getline(bufferStream, line)) {
    ...
}
```
