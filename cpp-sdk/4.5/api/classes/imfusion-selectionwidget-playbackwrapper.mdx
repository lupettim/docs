## Overview

This class provides an interface for creating wrappers that allow playback functionality to a type.

To make a type "Playable" with the [SelectionWidget](/cpp-sdk/4.5/api/classes/imfusion-selectionwidget), implement this interface in a concrete wrapper class, and register it with the [Selection](/cpp-sdk/4.5/api/classes/imfusion-selection) widget using the [registerPlaybackWrapper](/cpp-sdk/4.5/api/classes/imfusion-selectionwidget) function.

### setUpdateGuiCallback

```cpp
virtual void ImFusion::SelectionWidget::PlaybackWrapper::setUpdateGuiCallback (Data * data, const std::function < void(void)> & callback)
```

*pure virtual*

Set a callback function for updating the [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui).

The [SelectionWidget](/cpp-sdk/4.5/api/classes/imfusion-selectionwidget) will provide a callback function that will run on the [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) thread via this function.

**Parameters:**

- `callback` — Function for updating the [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) on the main thread.
