## Member Function Documentation

### computeTransformation

```cpp
bool ImFusion::Dicom::ImagePlane::computeTransformation (const ImageDescriptor & desc, mat4 & matrixOut) const
```

Compute plane transformation from the position and orientation.

The descriptor must already contain the final spacing in 2D. If the plane transformation could be determined, matrix will be set to the transformation and true is returned. Otherwise the matrix will be set to identity and false is returned. See also [ImFusion::Slice](/cpp-sdk/4.5/api/classes/imfusion-slice) on how the properties (e.g. normal or position) of the plane can be extracted.
