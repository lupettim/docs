`#include <ImFusion/GUI/ControllerDecorator.h>`

## Overview

[Base](/cpp-sdk/4.5/api/structs/imfusion-dicom-base) interface for wrapper classes to decorate [Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) instances and place them inside a [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase).

This interface defines a set of generic functionality that must be implemented by container widgets. Subclasses can extend it with extra logic for decoration, docking, or other functionality needed for their intended [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui).

Instances of this interface are usually not created manually by SDK users but rather instantiated and maintained by [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase) after calling [MainWindowBase::addController()](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase).

**See also:** [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase), [Controller](/cpp-sdk/4.5/api/classes/imfusion-controller), [DefaultControllerDecorator](/cpp-sdk/4.5/api/classes/imfusion-defaultcontrollerdecorator)
