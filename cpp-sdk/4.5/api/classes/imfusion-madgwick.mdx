## Overview

[Madgwick](/cpp-sdk/4.5/api/classes/imfusion-madgwick) sensor fusion.

Adapted from [https://github.com/PaulStoffregen/MadgwickAHRS](https://github.com/PaulStoffregen/MadgwickAHRS)

## Public Member Functions

- void **setBeta** (double val) — Explicit parameter override, use with care.
- void **setDescentsPerStep** (int val) — Define descents per step.
- void **setBetaFromGyroError** (double gyroSigmaInDegreesPerSec) — Conversion as used in the original paper: [http://x-io.co.uk/res/doc/madgwick_internal_report.pdf](http://x-io.co.uk/res/doc/madgwick_internal_report.pdf).
- void [update](#update) (const vec3 &angularVelocityDegPerSec, const vec3 &linAcc, const vec3 &mag, double timestampSeconds) override — Propagates the physical state using a new measurement.
- Eigen::Quaterniond [orientationQuaternion](#orientationquaternion) () const override — Returns the orientation of the most recent state.
- vec3 [position](#position) () const override — Returns the position in world coords.
- void [reset](#reset) () override — Restores the initial state.
