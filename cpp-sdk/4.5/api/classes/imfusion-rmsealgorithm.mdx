### createCompatible

```cpp
bool ImFusion::RmseAlgorithm::createCompatible (const DataList & data, Algorithm ** a = 0)
```

*static*

Evaluate if data is compatible with the algorithm requirements and create and algorithm instance, resp.

Possible scenarios: a) Input are two compatible SharedImageSets of equal size. The RMSE is computed pairwise. b) Input is a [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) of size 1 and another, compatible [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset). The RMSE of each image in the second SIS is computed w.r.t. the single ("guide") image. c) Input are several compatible SharedImageSets of size 1. The first one is treated as "guide" image. A [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) is created from all other images, the computation is continued like in b). Note that the "raw" SIS needs to be deleted in the end.
