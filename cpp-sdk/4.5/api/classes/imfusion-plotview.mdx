## Member Function Documentation

### acceptsData

```cpp
bool ImFusion::PlotView::acceptsData (const Data * data) const
```

*override, virtual*

Returns whether this view is capable of displaying the given data.

The default implementation checks whether there is any registered [DataDisplayHandler](/cpp-sdk/4.5/api/classes/imfusion-datadisplayhandler) that accepts this data for this type of view.

**See also:** [setVisibleData()](/cpp-sdk/4.5/api/classes/imfusion-gui-view), [showData()](/cpp-sdk/4.5/api/classes/imfusion-gui-view)
