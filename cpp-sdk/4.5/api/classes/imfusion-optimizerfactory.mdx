- static [std::shared_ptr](http://en.cppreference.com/w/cpp/memory/shared_ptr.html)&lt; [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) &gt; **create** (const [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) \*p) — Create and configure an [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) through [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties).

## Member Function Documentation

### create

```cpp
std::shared_ptr < Optimizer > ImFusion::OptimizerFactory::create (int type, int dimension)
```

*static*

Create an [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) class with a type value and dimensions.

Currently the following type values are supported: 0..41 The optimizers from the NlOpt library, see nlopt.h 42 ImFusion implementation of DIRECT ([OptimizerDiRect](/cpp-sdk/4.5/api/classes/imfusion-optimizerdirect) class) 43 ImFusion implementation of Nelder-Mead simplex ([OptimizerNelderMead](/cpp-sdk/4.5/api/classes/imfusion-optimizerneldermead) class) 44 [Series](/cpp-sdk/4.5/api/structs/imfusion-dicom-series) of other optimizations; not enumerated in methods below because it can not be configured through the [OptimizationDialog](/cpp-sdk/4.5/api/classes/imfusion-optimizationdialog)[GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) but only with [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties)
