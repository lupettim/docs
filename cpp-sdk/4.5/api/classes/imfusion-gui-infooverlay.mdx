`#include <ImFusion/GUI/InfoOverlay.h>`

## Overview

Interface for a [ViewOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-viewoverlay) to show textual information depending on the mouse cursor position (for instance: pixel information of an image, face/vertex info of a mesh).

The text can either be shown using a regular [GlTextOverlay](/cpp-sdk/4.5/api/classes/imfusion-gltextoverlay) at the selected [Anchor](/cpp-sdk/4.5/api/classes/imfusion-gui-viewoverlay) or as a tooltip hovering next to the mouse pointer.
