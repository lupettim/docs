`#include <ImFusion/GUI/PointsOnDataManipulator.h>`

## Overview

Interface class that handles creation and manipulation of points on [Data](/cpp-sdk/4.5/api/classes/imfusion-data).

Implementations of this interface should rely on [PointsOnData](/cpp-sdk/4.5/api/classes/imfusion-pointsondata) for handling the points and react to the events emitted by it.
