## Overview

Represents a pyramid of images with decreasing resolution.

The first level will always be the original image. All other levels are generated and owned by the object.

## Member Function Documentation

### createLevels

```cpp
int ImFusion::ImagePyramid::createLevels (int numLevels)
```

Creates and initializes the levels of the pyramid.

Existing levels are deleted. The minimum number of levels will always be one. Each level is computed by downsampling the previous level by factor 2. Once a dimension reaches a size of 2 pixels, it will not be downsampled any further. When all dimension reach this minimum before the desired number of levels is reached, no more levels will be added. If the input image has a mask, the lower levels will be assigned either a downsampled [ExplicitMask](/cpp-sdk/4.5/api/classes/imfusion-explicitmask) (independend of the input mask type), a mask that evaluates the original mask directly on the dowsampled image (

**See also:** m_resampleMask) The mask of the input remains unchanged.

**Returns:** the actual number of levels.

### propagateUp

```cpp
void ImFusion::ImagePyramid::propagateUp (int level)
```

Upsamples the contents of a level to its direct (higher-resolution) predecessor.

This method only propagates to the next level and NOT recursively to any further levels. Masks will not be propagated.

**Parameters:**

- `level`

### explicitlyResampleMask
