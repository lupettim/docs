## Overview

Simple class to make it easy to select by hand a portion of a volume.

- const [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) \* **selectedVolume** () const — Get the selection volume Values of 0 are considered to be selected, values of 255 are not selected May be nullptr if nothing has been selected so far.
