`#include <ImFusion/Reg/DisplacementVisualizer.h>`

## Overview

Algorithm/GlObject hybrid to display an image's displacement as a vector field in an [ImageView2D](/cpp-sdk/4.5/api/classes/imfusion-imageview2d).

The [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) interface does not perform anything but enabling instantiation from the data model/algorithm factory. The [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) interface will render the vector field into any ImageView2D/GlSliceView it is added to.
