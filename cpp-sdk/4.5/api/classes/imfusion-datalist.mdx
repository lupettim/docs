## Overview

Container for any number of [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instances such as image or meshes.

The [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) is merely a wrapper around a [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt;Data\*&gt; that provides various convenience functions to query certain types of [Data](/cpp-sdk/4.5/api/classes/imfusion-data). In particular, [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) will **not** take any ownership of its contents. Use [OwningDataList](/cpp-sdk/4.5/api/classes/imfusion-owningdatalist) to model ownership of the underlying data.

It is safe to access and modify the underlying container directly (e.g. if you want to apply STL algorithms on it).

**See also:** [OwningDataList](/cpp-sdk/4.5/api/classes/imfusion-owningdatalist)

**Examples:** [ExampleAlgorithm.cpp](/cpp-sdk/4.5/api/examples/example-algorithm-8cpp), [ExampleAlgorithm.h](/cpp-sdk/4.5/api/examples/example-algorithm-8h), [SqrtAlgorithm.cpp](/cpp-sdk/4.5/api/examples/sqrt-algorithm-8cpp), and [SqrtAlgorithm.h](/cpp-sdk/4.5/api/examples/sqrt-algorithm-8h).

## Constructor & Destructor Documentation

### DataList

```cpp
template<typename T> ImFusion::DataList::DataList (const std::vector < T * > & data)
```

Constructor from a vector of data.

**Note:** This constructor is deliberately implicit to allow for easy assignment of [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) queries (i.e. get-accessors) to DataLists. For instance, you can write dataList2 = dataList1.getAll&lt;[SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset)&gt;(); to filter all SharedImageSets from dataList1 and assign them to dataList2.

**Template Parameters:** | T | [Data](/cpp-sdk/4.5/api/classes/imfusion-data) type stored in the vector, must inherit from [Data](/cpp-sdk/4.5/api/classes/imfusion-data). |
