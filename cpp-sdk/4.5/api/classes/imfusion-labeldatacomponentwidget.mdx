## Member Enumeration Documentation

### VisibilityMode

```cpp
enum class ImFusion::LabelDataComponentWidget::VisibilityMode
```

*strong*

Enumeration to define which visibility flags of [LabelDataComponent](/cpp-sdk/4.5/api/classes/imfusion-labeldatacomponent) are to be modified by this widget.

**Enumerator:**

- `Visibility2d` — Modify only 2D label visibility.
- `Visibility3d` — Modify only 3D label visibility.
- `VisibilityAll` — Modify both 2D and 3D label visibility.
