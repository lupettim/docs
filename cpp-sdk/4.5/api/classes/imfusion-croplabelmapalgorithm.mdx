## Overview

Class for cropping a label map around the targets with certain label values.

If cropping the original volume of the label map at the same time is desired, the optional auxiliary image input allows cropping the volume the very same way. This algorithm allows for setting the desired label values, the margin (in pixels) around the object and whether to crop the image in place or create a new image when single label value is selected. When the input label map has one single frame, an additional functionality for splitting the label map and storing all the individual label images in one [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) is available.
