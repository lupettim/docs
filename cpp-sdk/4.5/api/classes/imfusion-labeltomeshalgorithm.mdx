## Member Data Documentation

### p_aboveIsoValue

```cpp
Parameter <bool> ImFusion::LabelToMeshAlgorithm::p_aboveIsoValue = {"Threshold Above Iso-Value", true, *this}
```

If true, the interior of the resulting mesh is greater than the isovalue.

Be aware that meshes which are "inside-out" may not behave as expected in some scenarions, including rendering.
