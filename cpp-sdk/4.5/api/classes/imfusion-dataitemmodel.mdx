## Overview

Wrapper class to transform the [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel) into a QAbstractItemModel so that it can be used in Qt views.

The underlying parentModel() will be represented as a tree with a single column. If a [DataGroup](/cpp-sdk/4.5/api/classes/imfusion-datagroup) has a proxy child, the index of the group will represent the proxy child data instead of the group. Furthermore, if a dataset implements the [CompoundData](/cpp-sdk/4.5/api/classes/imfusion-compounddata) interface (without being a [DataGroup](/cpp-sdk/4.5/api/classes/imfusion-datagroup)) the corresponding [DataItemModel](/cpp-sdk/4.5/api/classes/imfusion-dataitemmodel) index will have the nested datasets ([CompoundData::children()](/cpp-sdk/4.5/api/classes/imfusion-compounddata)) as children. These children have the ElementType::Internal instead of ElementType::Regular to illustrate their nature that they are internal to the parent dataset and not owned by/accessible through the underlying [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel).

A set of custom [Roles](#roles) enables you to query additional metadata from the datasets.

## Member Enumeration Documentation

### Roles

```cpp
enum ImFusion::DataItemModel::Roles
```

Enumeration of special roles supported by the model.

**Enumerator:**

- `DataRole` — The pointer to the data (use with variant.value&lt;ImFusion::Data\*&gt;()).
- `DescriptionRole` — A short description of the data type (e.g. '2D [Image](/cpp-sdk/4.5/api/classes/imfusion-image)' or 'Ultrasound Sweep').
- `AttributesRole` — A QList of QIcons representing special attributes of the data (e.g. multichannel, stream, mask, deformation...).
- `AttributesToolTipRole` — A QList of QString representing a tool tip for the elements of AttributesRole.
- `LinksRole` — A QList of QIcons representing links of the data.
- `LinksToolTipRole` — A QList of QString representing a tool tip for the elements of LinksRole.
- `LinksIndexRole` — A QList of the indices of the links in the DataModels list.
- `ElementTypeRole` — The [ElementType](#elementtype) of the data.

### ElementType

```cpp
enum class ImFusion::DataItemModel::ElementType
```

*strong*

Value type of Roles::ElementTypeRole.

**Enumerator:**

- `Regular` — A regular [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance owned and maintained by the underlying [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel).
- `Internal` — A [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance that is implicitly bound to/owned by the parent dataset, not accessible through the underlying [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel).

## Member Function Documentation

### index

```cpp
QModelIndex ImFusion::DataItemModel::index (const Data * data) const
```

QModelIndices for this item model are constructed as follows:

- The index for the root node is the (invalid) default QModelIndex()
- The index's row is the index of the element within the parent group
- The index's column is always 0
- The index's internalPointer points to the corresponding [DataItemModel::Item](/cpp-sdk/4.5/api/structs/imfusion-dataitemmodel-item) instance
