`#include <ImFusion/GUI/AnimationBackendQt.h>`

## Overview

Concrete implementation of [AnimationBackendBase](/cpp-sdk/4.5/api/classes/imfusion-animations-animationbackendbase) using the Qt event loop for dispatching the animation updates.

All animations will be executed on the thread that instantiated the [AnimationBackendQt](/cpp-sdk/4.5/api/classes/imfusion-animations-animationbackendqt). This thread must have a running event loop on construction.
