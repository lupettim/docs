
Checks if the given viewport coordinates are close to where one of the 8 corners of the box is rendered.

If yes, returns the index of the corresponding vertex of the underlying [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh).

### dataAnnotation

```cpp
bool ImFusion::GlBox::dataAnnotation () const
```

*inline*

Specifies whether this annotation should be treated as a data annotation for the underlying mesh.

If true, the algorithms available for meshes will be shown in the context menu when right-clicking on the annotation in the annotation widget, Otherwise they will not be listed
