## Overview

[Mahony](/cpp-sdk/4.5/api/classes/imfusion-mahony) sensor fusion.

Based on [https://github.com/PaulStoffregen/MahonyAHRS](https://github.com/PaulStoffregen/MahonyAHRS). Converges slightly faster than [Madgwick](/cpp-sdk/4.5/api/classes/imfusion-madgwick), but supposed to be less accurate. See [https://en.wikipedia.org/wiki/PID_controller](https://en.wikipedia.org/wiki/PID_controller) for the configuration constants
