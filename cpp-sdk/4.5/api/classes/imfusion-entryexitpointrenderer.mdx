## Overview

**See also:** [GlVolumeRenderer](/cpp-sdk/4.5/api/classes/imfusion-glvolumerenderer)

## Member Function Documentation

### render

```cpp
void ImFusion::EntryExitPointRenderer::render (const GL::ViewState & viewState, const SharedImageSet & sis, int which, const DisplayOptions3d & dop, bool clear, bool renderNearClippingPlane)
```

Renders the entry/exit points for the given image.

**Parameters:**

- `viewState` — [Viewport](/cpp-sdk/4.5/api/structs/imfusion-gl-viewport) used for output texture size, view/projection matrices used for rendering.
- `sis` — [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) to render EEP for.
- `which` — Index of the [SharedImage](/cpp-sdk/4.5/api/classes/imfusion-sharedimage) in *sis* to render the EEP for
- `dop` — DisplayOptions used during rendering (needed for [Octree](/cpp-sdk/4.5/api/classes/imfusion-octree) culling)
- `clear` — Flag whether to clear the existing EEP textures
- `renderNearClippingPlane` — Flag whether to render a near clipping plane
