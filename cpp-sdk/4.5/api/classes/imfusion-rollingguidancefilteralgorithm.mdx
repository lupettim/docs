## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for Rolling Guidance Filtering.

Scale-aware filter based on multiple iterations of a bilateral or guided filter see ECCV 2014 paper at [http://www.cse.cuhk.edu.hk/~leojia/projects/rollguidance/](http://www.cse.cuhk.edu.hk/~leojia/projects/rollguidance/)
