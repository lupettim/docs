## Overview

Compute cardiac or respiratory gating parameters by detecting periodic structural change in the images To use, feed a consecutive block of frames into initialize and get their phase values.

Then feed individual frames into update and get the corresponding phase value of that new frame. Timestamps need to be consistent and framerates match between the initialization and update frames. Initialization block needs to show the target periodic motion as clearly as possible.
