## Overview

template&lt;typename T&gt;

class ImFusion::TypedImage&lt; T &gt;
Concrete implementation of a [MemImage](/cpp-sdk/4.5/api/classes/imfusion-memimage) for a given pixel type.

A [TypedImage](/cpp-sdk/4.5/api/classes/imfusion-typedimage) is movable but not copyable. Use clone() in order to perform a deep copy.

**See also:** [Using Image Data](/cpp-sdk/4.5/api/guides/image-data-tutorial), [ImageProcessing](/cpp-sdk/4.5/api/namespaces/imfusion-imageprocessing)
