`#include <ImFusion/Core/GL/IndexBuffer.h>`

## Overview

Specialization of [Buffer](/cpp-sdk/4.5/api/classes/imfusion-gl-buffer) to store vertex index data (GL_ELEMENT_ARRAY_BUFFER).

Can be passed to a [GL::VertexBuffer::draw()](/cpp-sdk/4.5/api/classes/imfusion-gl-vertexbuffer) call for indexed rendering where the same vertex data can be used by multiple vertices to reduce memory usage.

Example for rendering a quad using indexed rendering:

```cpp
// vertex data describing 4 vertices
std::vector<vec2f> vertices = {
    vec2f(-1.0, -1.0),
    vec2f( 1.0, -1.0),
    vec2f(-1.0,  1.0),
    vec2f( 1.0,  1.0)
};
VertexBuffer vbo;
vbo.bind();
vbo.setData(vertices);

// index data to render two triangles of 3 vertices each
std::vector<uint16_t> indices = {
    0, 1, 2, 1, 2, 3
};
IndexBuffer ibo;
ibo.bind();
ibo.setData(indices);

// Note that this is an artificial example to showcase the usage of index buffers.
// You could achieve the same rendering result with the vertex data only and using
// the Primitive::TriangleStrip primitive instead.
vbo.draw(Primitive::Triangles, ibo);
```
 fragment

**Warning:** In WebGL, the PRIMITIVE_RESTART_FIXED_INDEX option is always enabled. If an index is the maximum for the data type (i.e. 0xFF, 0xFFFF or 0xFFFFFFFF), it is not treated as a regular index, but will restart the primitive. To maintain compatibility, make sure to only use indices smaller than the maximum and only use the maximum in setPrimitiveRestartIndex. See the [WebGL Spec](https://www.khronos.org/registry/webgl/specs/latest/2.0/#NO_PRIMITIVE_RESTART_FIXED_INDEX) for details.

- [https://www.khronos.org/opengl/wiki/Vertex_Specification#Index_buffers](https://www.khronos.org/opengl/wiki/Vertex_Specification#Index_buffers)

### primitiveRestartIndex

```cpp
uint32_t ImFusion::GL::IndexBuffer::primitiveRestartIndex () const
```

Returns the OpenGL primitive restart index to use during indexed rendering.

The primitive restart feature is disabled if the index is 0 (default behavior).

### setPrimitiveRestartIndex

```cpp
void ImFusion::GL::IndexBuffer::setPrimitiveRestartIndex (uint32_t value)
```

Sets the OpenGL primitive restart index to use during indexed rendering.

The primitive restart feature is disabled if the index is 0 (default behavior). On WebGL, the features is always enabled. Check the class documentation for details.
