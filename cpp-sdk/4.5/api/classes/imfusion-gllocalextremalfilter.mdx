## Member Function Documentation

### setKernelSize

```cpp
bool ImFusion::GlLocalExtremalFilter::setKernelSize (int size, double k)
```

*override, virtual*

**Parameters:**

- `size` — Size of quadratic kernel. Currently supported sizes are 3 and 5.
- `k` — Threshold value. Note that this is the slider value multiplied by 0.01

**Warning:** Filtering a 3D image using the 5x5x5 kernel is costly. Computation time is about 1min for a 512^3 image.
