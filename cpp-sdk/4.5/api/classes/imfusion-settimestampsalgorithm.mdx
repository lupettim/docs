## Overview

Sets the timestamps of an image data set based on a first timestamp and frame rate.

Will overwrite existing image timestamps.

- **SetTimestampsAlgorithm** ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) \*img) — Creates the algorithm instance with an image.
