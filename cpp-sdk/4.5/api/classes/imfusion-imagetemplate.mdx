## Member Enumeration Documentation

### Metric

```cpp
enum class ImFusion::ImageTemplate::Metric
```

*strong*

Define the metric used to measure pixel distance. [Image](/cpp-sdk/4.5/api/classes/imfusion-image) values (x and y) are scaled in range [0, 1] and s is the scale.

**Enumerator:**

- `Welsch` — d(x, y) = |x - y|
- `Cauchy` — d(x, y) = 1 - exp(-1/2 \* ((x-y)/s)^2)
d(x, y) = log(1/2 \* ((x-y)/s)^2 + 1)

## Member Function Documentation

### match

```cpp
double ImFusion::ImageTemplate::match (const MemImage & targetImage) const
```

Matches the image template onto the target image.

Returns the similarity between each region in the template and the same region in the target. Both images must have the same size (otherwise, 0 is returned).

### matchRegion

```cpp
double ImFusion::ImageTemplate::matchRegion (const Region & region, const MemImage & targetImage) const
```

Matches the image template onto the target image.

Returns the similarity between the two only in the region passed as argument. Both images must have the same size (otherwise, 0 is returned).
