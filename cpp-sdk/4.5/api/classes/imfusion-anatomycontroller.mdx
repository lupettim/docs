## Overview

[Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) for [AnatomyAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-anatomyalgorithm).

Shows the list of anatomies of an image ([AnatomyDataComponent](/cpp-sdk/4.5/api/classes/imfusion-anatomydatacomponent)). Also shows a list of available anatomies to browse and add or remove from the image.
