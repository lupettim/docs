## Overview

Implementation of the Broyden–Fletcher–Goldfarb–Shanno (BFGS) optimizer.

This optimizer uses derivatives and keeps an approximation of the (inverse) Hessian matrix of the objective function. Internally it uses line search with Armijio and weak Wolfe conditions.
