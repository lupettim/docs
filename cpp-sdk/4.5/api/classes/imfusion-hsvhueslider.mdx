## Overview

Specialized version of a [CompositeSlider](/cpp-sdk/4.5/api/classes/imfusion-compositeslider) that renders the HSV hue color map as background.

Default value range is 0..359 representing the hue angle in the HSV color space.

**See also:** [HsvColorPicker2d](/cpp-sdk/4.5/api/classes/imfusion-hsvcolorpicker2d), [ColorPickerWidget](/cpp-sdk/4.5/api/classes/imfusion-colorpickerwidget), [ColorButton](/cpp-sdk/4.5/api/classes/imfusion-colorbutton)
