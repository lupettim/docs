## Overview

Overlay for displaying one or multiple named scalar values represented as colored bars.

Displays the given title and scalar value in the form of

```cpp
<Title Bar 1> [bar describing value by width and color] <Value Bar 1>
<Title Bar 2> [bar describing value by width and color] <Value Bar 2>
```

## Member Enumeration Documentation

### RenderMode

```cpp
enum class ImFusion::GlScalarBarOverlay::RenderMode
```

*strong*

**Enumerator:**

- `All` — show all quality bars
- `MainOnly` — show only main tracking quality (or first)
- `Auto` — show all until 3 tracking sequence, collapse to 1 otherwise
- `EllipsisPoints` — hide bars and show ellipsis points
- `None` — show no tracking quality bar and no interaction
