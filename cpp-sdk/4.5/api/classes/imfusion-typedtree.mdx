## Member Function Documentation

### addNode

```cpp
template<typename T> Node * ImFusion::TypedTree < T >::addNode (std::unique_ptr < Node > node, Node * parent)
```

*inline, override, virtual*

Insert typed node below parent.

Also checks node type. Returns pointer to the node if node is added successfully, nullptr otherwise (probably due to invalid parent). nullptr parent means that the node should become a root of the tree. If tree already has a root, the node will not be added.
