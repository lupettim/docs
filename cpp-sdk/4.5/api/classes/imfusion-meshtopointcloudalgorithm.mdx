## Member Enumeration Documentation

### ColorSource

```cpp
enum ImFusion::MeshToPointCloudAlgorithm::ColorSource
```

**Enumerator:**

- `TexturePreferred` — Take texture if present, color otherwise.
- `ColorPreferred` — Take color if present, texture otherwise.
- `Texture` — Only take texture.
- `Color` — Only take color.

### Mode

```cpp
enum ImFusion::MeshToPointCloudAlgorithm::Mode
```

**Enumerator:**

- `Vertex` — Points are taken from the mesh surface.
- `Interior` — Points are sampled from within the mesh volume. To avoid the output being a subset of a regular grid, we add a pseudo-random offset to each point. For non-convex meshes, this can cause points to be slightly outside of the mesh.
