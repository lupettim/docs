## Constructor & Destructor Documentation

### LinkSelection

```cpp
ImFusion::LinkSelection::LinkSelection (const std::vector < Data * > & data)
```

*explicit*

Create a link containing a list of data.

If any of the data are not *[Selectable](/cpp-sdk/4.5/api/classes/imfusion-selectable)*, or have different values returned from [Selectable::size()](/cpp-sdk/4.5/api/classes/imfusion-selectable), the link will be empty.
