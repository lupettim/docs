## Overview

[Context](/cpp-sdk/4.5/api/classes/imfusion-gl-context) menu of the [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget).

Shows actions depending on the selected data, e.g. available algorithms, link options and grouping of data. Actions in the menu will call methods on the [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget) but otherwise the menu is independent: It does not use the current selection of [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget) nor does it interact with any UI element of the [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget). It's main use case is to provide a right click menu for widgets providing an alternative view (e.g. with different filtering) on the [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel).

**See also:** [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget)
