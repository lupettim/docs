`#include <ImFusion/GL/GlSlice.h>`

## Overview

Represents an MPR slice for rendering [Data](/cpp-sdk/4.5/api/classes/imfusion-data) into 2D views.

[GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) will render MPRs of one or multiple [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instances. While the actual rendering of individual data is delegated to the [GlSliceRenderer](/cpp-sdk/4.5/api/classes/imfusion-glslicerenderer) interface, [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) will take care of setting up the OpenGL context and blending the rendered images.

Furthermore, [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) can act as [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) to render the extent (and optionally image content) of the MPR slice into other third-party views.

**See also:** [ImageView2D](/cpp-sdk/4.5/api/classes/imfusion-imageview2d), [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview), [GlSliceRenderer](/cpp-sdk/4.5/api/classes/imfusion-glslicerenderer)

## Member Enumeration Documentation

### BlendingMode

```cpp
enum class ImFusion::GlSlice::BlendingMode
```

*strong*

Enumeration of the blending modes available in [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice).

**Enumerator:**

- `None` — No blending.
- `Default` — Default alpha blending.
- `Color` — Automatic color blending depending on modality (either SingleColor or DualColor blending mode).
- `Checkerboard` — Checkerboard blending.
- `DistanceToBorder` — Per-pixel blending based on distance to image border.
- `SingleColor` — Use no tint color for the first image, use the primary blendingColor for all other images.
- `DualColor` — Use primary blendingColor for the first image, use secondary blendingColor for all other images.

### ObjectRenderMode

```cpp
enum class ImFusion::GlSlice::ObjectRenderMode
```

*strong*

Enumeration for selecting what to render as [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject).

**Enumerator:**

- `Nothing` — Don't render MPR cross-sections at all.
- `Border` — Render the surrounding border of the slice.
- `ImageContent` — Render the actual image content.
- `Centerlines` — Render the centerlines of the slice.
- `Plane` — Render the slice as semi-transparent plane.
- `Crosssection` — Render lines forming a cross where the three MPR planes intersect (only if exactly 3 slice views exist).

### CrosssectionRenderMode

```cpp
enum class ImFusion::GlSlice::CrosssectionRenderMode
```

*strong*

Enumeration for selecting how to render the cross-section between this MPR and other MPR slice views.

**Enumerator:**

- `Nothing` — Don't render MPR cross-sections at all.
- `Line` — Render the actual MPR cross-section lines.
- `Thickness` — Render slice thickness indicators in MPR cross-sections.
- `OrientationMarkers` — Render orientation markers in MPR cross-sections.
- `OrientationMarkersQuadrupedDefault` — Default to quadruped orientation markers in MPR cross-sections.
