`#include <ImFusion/GL/GlSliceRendererImage.h>`

## Overview

Enables [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) to render normal images/volumes ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset)).

In [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice), this renderer is also referred to as the default renderer. This renderer accepts either 2D or 3D images exclusively depending on the flag given during construction.

**See also:** [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice), [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset)
