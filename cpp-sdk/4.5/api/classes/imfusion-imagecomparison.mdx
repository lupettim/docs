## Overview

Compares two image sets based on their original pixel values.

The comparison applies each image's shift/scale to convert storage values to original values:

```cpp
original = storage / scale - shift
```
 fragment
Images with different storage values can compare equal if their original values match.

If needed, the second image is resampled to the first image's geometry before comparison.
