## Overview

Runs a non-linear optimization on a [ParametricDeformation](/cpp-sdk/4.5/api/classes/imfusion-parametricdeformation) in order to match its deformation to a given reference displacement field.

The optimization is performed by evaluating the SSD between the given reference displacement field and the displacement field of the target deformation object.
