## Overview

Implementation of the Depth Darkening technique for a simple screen-space ambient occlusion effect.

Original paper: [http://graphics.uni-konstanz.de/publikationen/2006/unsharp_masking/Luft%20et%20al.%20&ndash;%20Image%20Enhancement%20by%20Unsharp%20Masking%20the%20Depth%20Buffer.pdf](http://graphics.uni-konstanz.de/publikationen/2006/unsharp_masking/Luft%20et%20al.%20&ndash;%20Image%20Enhancement%20by%20Unsharp%20Masking%20the%20Depth%20Buffer.pdf)

## Member Function Documentation

### apply

```cpp
void ImFusion::GlDepthDarkening::apply (const GlImage * inputColorTex, const GlImage * inputDepthTex)
```

Applies depth darkening to the given input textures and renders the result directly to the screen.

If you have an active FBO with a texture attached, it will render to that.

**Parameters:**

- `inputColorTex` — Input color image, must not be 0.
- `inputDepthTex` — Input depth image, must not be 0.
