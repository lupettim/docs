## Member Enumeration Documentation

### VolumeViewPickingMode

```cpp
enum class ImFusion::PointBasedAnnotationManipulator::VolumeViewPickingMode
```

*strong*

Enumeration to describe when [PointBasedAnnotationManipulator](/cpp-sdk/4.5/api/classes/imfusion-pointbasedannotationmanipulator) should perform 3D position picking in GlVolumeViews.

**Enumerator:**

- `On` — Always on.
- `Off` — Always off.
- `Automatic` — Automatic based on annotation type, and presence of meshes in the [AnnotationModel](/cpp-sdk/4.5/api/classes/imfusion-annotationmodel).
