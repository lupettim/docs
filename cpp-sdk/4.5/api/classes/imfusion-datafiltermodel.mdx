## Overview

The [DataFilterModel](/cpp-sdk/4.5/api/classes/imfusion-datafiltermodel) class is a proxy model to filter a [DataItemModel](/cpp-sdk/4.5/api/classes/imfusion-dataitemmodel).

By default, nothing is filtered. A [DataFilterModel](/cpp-sdk/4.5/api/classes/imfusion-datafiltermodel) will only work correctly if the source model is of type [DataItemModel](/cpp-sdk/4.5/api/classes/imfusion-dataitemmodel).

**Warning:** Do not change the filter model from within callbacks of DataModel/DataWidget-changed signals, since this can potentially corrupt the QModelIndex mapping inside.
