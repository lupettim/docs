## Member Data Documentation

### p_numSplinePoints

```cpp
Parameter <int> ImFusion::SplineLabelPainterAlgorithm::p_numSplinePoints = {"numSplinePts", 1000, *this}
```

The number of points used to interpolate the spline.

A value of 0 or less, disables interpolation and treats the spline as a polyline.

### p_adaptiveness

```cpp
Parameter <double> ImFusion::SplineLabelPainterAlgorithm::p_adaptiveness = {"adaptiveness", 0.8, *this}
```

How strong the labels adopt to the pixel intensities inside the radius.

A value of 0 disables this.
