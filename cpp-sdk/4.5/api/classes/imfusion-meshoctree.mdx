`#include <ImFusion/Mesh/MeshOctree.h>`

## Overview

Class representing a mesh octree. Faces of the mesh are split between the nodes of the tree, each node represents a partition of the parent volume (minimum 1/8th).

## Public Member Functions

- **MeshOctree** (const [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) \*mesh)
- bool [buildTree](#buildtree) (int depth) — Builds a tree from mesh faces.
- double [distanceToPoint](#distancetopoint) (const vec3 &point, bool signedDistance, int \*closestFaceId) const — Compute distance from point to mesh.
- bool [usesTriangleCache](#usestrianglecache) () const — Returns the flag whether to cache mesh triangles.
- void [setUseTriangleCache](#setusetrianglecache) (bool value) — Sets the flag whether to cache mesh triangles.
