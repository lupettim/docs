`#include <ImFusion/Mesh/Rendering/GlMeshRendererNormals.h>`

## Overview

Renders the normals of a [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) using line primitives.

This low-level renderer renders lines starting from each vertex of the mesh pointing in the direction of the local normal. The GlMeshBuffers instance passed to [render()](#render) defines whether to use vertex or face normals.

This class does not implement the [GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) interface by design as it is meant to serve as a reusable component to be integrated into high-level classes.
