## Member Enumeration Documentation

### Mode

```cpp
enum ImFusion::GlNoiseFilter::Mode
```

Noise model to use for computation.

**Enumerator:**

- `PHYSICAL_POISSON` — Assumes that the image values represent integer counts for physically plausible simulation of poisson noise.
Float images need to be prepared (i.e. normalized, and scale!=1, shift!=0). The parameters are intrinsically defined.
