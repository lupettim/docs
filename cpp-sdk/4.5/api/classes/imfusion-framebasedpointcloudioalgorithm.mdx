## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for reading and writing frame-dependent point clouds.

**Note:** Note that this algorithm does not store the dependent TrackedSharedImageSets. The user of this algorithm needs to make sure that they are saved/read independently, and their number and order matches for saving and reading. Due to the way the ImFusion Suite context menu is automatically populated, *reading* a frame-based point cloud works by selecting the dependent TrackedSharedImageSets and selecting this algorithm from the *export* sub-menu. This algorithm is not part of the SDK on purpose.
