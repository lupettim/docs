### channelStatisticsFromLabelMap

```cpp

```

Calculate the per-channel statistics on the currently selected [GlImage](/cpp-sdk/4.5/api/classes/imfusion-glimage) image of the [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) used to create this object, for each of the labels in the given label map.

The label map must be single channel and have unsigned char component type.
