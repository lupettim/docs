## Overview

Compute an auto-correlation (band) matrix of an image set.

The similarity of all frames in an image set with either all others, or a chosen number of previous frames is computed. In the latter case, the selection on the input image is set to one of the best sequences in terms of non-repetitive content. If an optional kernel image is provided, the similarity field is convolved with it for detecting peaks corresponding to e.g. a respiration or cardiac phase.
