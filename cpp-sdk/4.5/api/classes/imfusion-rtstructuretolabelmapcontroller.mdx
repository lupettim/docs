## Overview

[Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) that allows to select a reference volume for the [RTStructureToLabelMap](/cpp-sdk/4.5/api/classes/imfusion-rtstructuretolabelmap) algorithm.

[Selectable](/cpp-sdk/4.5/api/classes/imfusion-selectable) reference volumes are all volumes in the [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel). The default selected volume will be either the parent dataset of the [GlPointCloud](/cpp-sdk/4.5/api/classes/imfusion-glpointcloud) representing RTStructureToLabelMap::rtStructureSet or the first volume if there is no parent.
