## Overview

[Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) splitting based on vertex or face labels.

The class extracts subsets of a mesh according to label-based criteria. If an auxiliary label image is provided, triangles are kept either when all vertices sample the specified label or when any vertex samples it. If no auxiliary label image is provided, triangles are selected based on mesh labels: either by keeping only triangles with a specified face label, or by keeping only triangles where all vertices or any vertex has a specified vertex label.

## Member Enumeration Documentation

### Mode

```cpp
enum class ImFusion::SplitMeshAlgorithm::Mode
```

*strong*

**Enumerator:**

- `VertexLabelAny` — Split mesh based on vertex label, output mesh(es) will contain all faces that have at least one vertex with the given label. If a label map is passed, this is used to provide vertex labels.
- `VertexLabelAll` — Split mesh based on vertex label, output mesh(es) will contain only faces where all vertices have the given label. If a label map is passed, this is used to provide vertex labels.
- `FaceLabelAny` — Split mesh based on face labels.
- `ImageLabelAll` — Split mesh based on vertex label, output mesh(es) will contain all faces that have at least one vertex with the given label. Label map is used to provide vertex labels.
- `ImageLabelAny` — Split mesh based on vertex label, output mesh(es) will contain only faces where all vertices have the given label. Label map is used to provide vertex labels.
