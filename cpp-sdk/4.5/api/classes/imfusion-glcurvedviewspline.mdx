## Constructor & Destructor Documentation

### GlCurvedViewSpline

```cpp
ImFusion::GlCurvedViewSpline::GlCurvedViewSpline (bool inPlaneSplineNormal = true)
```

Creates the spline.

**Parameters:**

- `inPlaneSplineNormal` — flag indicating the initial sampling direction (curved spline normal). When set to true, the sampling direction is set as a y axis of the plane where the first point of spline was set. When set to false, the sampling direction is set as a z axis of the plane where the first point of spline was set.
