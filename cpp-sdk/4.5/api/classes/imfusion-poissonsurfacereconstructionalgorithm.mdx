## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for computing poisson reconstruction from a point cloud with normals.

This is an [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) frontend for the low-level algorithm in [PoissonReconstruction](/cpp-sdk/4.5/api/namespaces/imfusion-poissonreconstruction), which can be used directly if the application does not rely on the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) interface being used or if one needs fine control over all algorithm parameters.

In this algorithm only the most relevant parameters of [PoissonReconstruction](/cpp-sdk/4.5/api/namespaces/imfusion-poissonreconstruction) are forwarded to the respective getters and setters, see [PoissonReconstruction::PoissonParams](/cpp-sdk/4.5/api/structs/imfusion-poissonreconstruction-poissonparams) for their documentation and for additional parameters that were omitted here.

A common use case for this algorithm is creating a [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) out of a point cloud that was obtained from a 3D scanner (should there be multiple scans we need to register and merge them into a single point cloud first):

```cpp
std::unique_ptr<Mesh> createMesh(const PointCloud& pc)
{
    assert(pc.hasNormals());
    PoissonSurfaceReconstructionAlgorithm algo(&pc);
    algo.setSpacing(pc.bounds().extent().norm()*0.002); // triangle edge length approx 1/500th of object diameter
    algo.setLevel(-1);                                  //else this overrides setSpacing
 //alternative: algo.setLevel(9); //divide the point cloud bounding box 9 times, leading to minimum voxel side lengths of approx 0.002*cube side length
    algo.compute();
 if (algo.status() == static_cast<int>(Algorithm::Status::Success))
 return algo.takeOutput().extractFirst<Mesh>(); //this algorithm only has one output
 else
 return nullptr;
}
```
 fragment
In addition to the usually watertight [Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) produced by [PoissonReconstruction](/cpp-sdk/4.5/api/namespaces/imfusion-poissonreconstruction) this algorithm offers the option to filter the resulting mesh w.r.t distance to the original point cloud, allowing to keep gaps in the measurements open:

```cpp
// [...] see above
algo.setSurfaceDistanceR1(pc.bounds().extent().norm()*0.002); // erase triangles that do not have a point as close to them as their minimum side length
algo.compute();
// [...] see above
```
 fragment
[setSurfaceDistanceR2()](#setsurfacedistancer2) works together with setMaxAngleDegrees() to filter out faces by distance *if also* the angle between point and face normal falls below the given threshold.

Colors are taken from the point cloud by default if the point cloud has colors. If this is not desired setColorSource() can be used. The other parameters have reasonable default values that should usually not need to be changed.

### surfaceDistanceThresholds

```cpp
std::array < double, 2 > ImFusion::PoissonSurfaceReconstructionAlgorithm::surfaceDistanceThresholds () const
```

*inline*

Thresholds for filtering generated triangles w.r.t their distance to the original point cloud.

The two thresholds form a pair of &lt;lower bound, upper bound&gt;, which work as follows: The lower bound determines a distance *below* which triangles are *always kept*, the upper bound determines a distance *above* which triangles are *always deleted*. Between the two bounds the decision is based on [maxAngleDegrees()](#maxangledegrees): triangles in this range are deleted only if their normal deviates from the closest point's normal by *more than*[maxAngleDegrees()](#maxangledegrees) degrees.

Set both values to 0.0 to disable the filtering.

Default values are &lt;2.0, 4.0&gt;.

### surfaceDistanceR2

```cpp
double ImFusion::PoissonSurfaceReconstructionAlgorithm::surfaceDistanceR2 () const
```

*inline*

Maximum distance of generated triangles to original point cloud given similar normals.

Works in connection with [maxAngleDegrees()](#maxangledegrees) to filter out triangles that exceed the distance threshold *and* have a sufficiently similar normal to the closest point.

Set this and [surfaceDistanceR1()](#surfacedistancer1) to '0' to obtain watertight mesh which interpolates all holes.

Default value is 4.

****Deprecated**:** "PoissonSurfaceReconstructionAlgorithm::surfaceDistanceR2 is deprecated, instead use 'surfaceDistanceThresholds'"

```cpp
double ImFusion::PoissonSurfaceReconstructionAlgorithm::maxAngleDegrees () const
```
