## Overview

Point cloud picking manipulator The [PointCloudPickingManipulator](/cpp-sdk/4.5/api/classes/imfusion-pointcloudpickingmanipulator) can return the list of selected vertices The supported manipulation is alt + mouse key press with optional dragging.

alt + mouse wheel changes the size of selection area For proper use the [PointCloudPickingManipulator](/cpp-sdk/4.5/api/classes/imfusion-pointcloudpickingmanipulator) object must be added to a view as point cloud manipulator and the outer object should subscribe to its signals e.g.

```cpp
GlPointCloud* glPointCloud = ...;
PointCloudPickingManipulator* manip = new PointCloudPickingManipulator(glPointCloud, *m_disp->view3D()->view());
manip->setSelectionMode(PointCloudPickingManipulator::SelectionMode::Vertex);
manip->signalSelectedVertexIndices.connect(this, &ExampleController::onSelectedVertexIndices);
auto pointCloudPickingManipulator = new Interactive<GlPointCloud, PointCloudPickingManipulator>(glPointCloud, manip, true);
m_disp->view3D()->addObject(pointCloudPickingManipulator.get());
```
 fragment
The manipulator must be removed from the view before deleting!

### setHighlightClosestVertex

```cpp
void ImFusion::PointCloudPickingManipulator::setHighlightClosestVertex (bool v)
```

*inline*

Sets whether the closest vertex should be highlighted.

When enabled (true):

- The closest vertex to the cursor will be visually highlighted.
- signalClosestVertexIndexChanged will be emitted whenever the highlighted vertex changes.
- The selection area will remain fixed in size when using the mouse wheel, and signalChangedSelectionSize will **not** be emitted.

When disabled (false):

- No closest-vertex highlighting is performed.
- Selection-size adjustments and related signals behave normally.

**Parameters:**

- `v` — true to enable closest-vertex highlighting, false to disable it.
