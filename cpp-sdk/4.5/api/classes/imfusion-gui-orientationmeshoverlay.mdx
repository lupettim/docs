## Overview

The [OrientationMeshOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-orientationmeshoverlay) class wraps a [GlOrientationMeshOverlay](/cpp-sdk/4.5/api/classes/imfusion-glorientationmeshoverlay) and does not provide any interactions.

## Constructor & Destructor Documentation

### OrientationMeshOverlay

```cpp
ImFusion::GUI::OrientationMeshOverlay::OrientationMeshOverlay (const Mesh & mesh, View * view)
```

Constructs a new [OrientationMeshOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-orientationmeshoverlay) with the given mesh.

The modelView matrix of view is used to determine the orientation of the mesh. This is not necessarily the anchor view, e.g. the overlay could be part of one view but display the orientation of another. The mesh is not stored and can be deleted after this call.
