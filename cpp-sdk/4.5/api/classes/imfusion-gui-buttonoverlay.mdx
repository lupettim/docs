## Overview

[ViewOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-viewoverlay) representing a clickable button.

This overlay renders a graphical icon that will be highlighted when the mouse is hovering over it. When the user clicks on the button it will perform the following two actions:

1. The signalClicked will be emitted allowing you to perform any custom action as result.
2. The corresponding [EventResult's context menu](/cpp-sdk/4.5/api/structs/imfusion-gui-eventresult) will be populated with the result of the optional [menu generator](#setmenugenerator) you can set. This provides an convenient way to show a basic [Menu](/cpp-sdk/4.5/api/classes/imfusion-gui-menu) as result.
