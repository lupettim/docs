## Member Enumeration Documentation

### TrackingSequenceProcessingMode

```cpp
enum ImFusion::TrackingSequenceProcessing::TrackingSequenceProcessingMode
```

Mode for stream processing.

**Enumerator:**

- `ClearBadQuality` — Removal of bad quality samples (NaN or &lt;0.1), can be combined with ClearOutliers.
- `ClearOutliers` — Removal of outliers, can be combined with ClearBadQuality.
- `ResampleToReference` — Resample first input to match timing of second one.
- `TemporalCalibration` — Perform temporal calibration with respect to second input.
- `ConvertToPointCloud` — Perform conversion to point cloud.
- `Invert` — Invert matrices.
