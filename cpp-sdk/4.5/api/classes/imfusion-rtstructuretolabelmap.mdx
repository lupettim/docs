## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to convert a [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) with a [Dicom::RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent) datacomponent to a labelmap.

This is currently only supported for CLOSED_PLANAR contours in [Dicom::RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent). The algorithm requires a reference volume that determines the size of the labelmap. Each contour is expected to be planar on a slice in the reference volume. This algorithm works best when using the volume that is referenced by the original DICOM RTStructureDataSet (see [Dicom::RTStructureDataComponent::referencedFrameOfReferenceUID](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent)).
