## Overview

Specialization of [Buffer](/cpp-sdk/4.5/api/classes/imfusion-gl-buffer) for OpenGL Atomic Counter Objects.

**Note:** Requires at least OpenGL 4.2 or the ARB_shader_atomic_counters extension.

**See also:** [https://www.khronos.org/opengl/wiki/Atomic_Counter](https://www.khronos.org/opengl/wiki/Atomic_Counter)

## Constructor & Destructor Documentation

### AtomicCounterBuffer

```cpp
ImFusion::GL::AtomicCounterBuffer::AtomicCounterBuffer (int numCounters, Usage usageHint = Usage::DynamicDraw)
```

*explicit*

Create an OpenGL buffer for holding the specified number of atomic counters.

All counters will be initialized with 0.

**Parameters:**

- `numCounters` — Number of counter variables to allocate buffer space for
- `usageHint` — Expected usage pattern of the buffer

**Exceptions:**

- `std::runtime_error` — if the corresponding OpenGL buffer object could not be created.
