`#include <ImFusion/GL/Gl3DObject.h>`

## Overview

Abstract base class for 3D objects for which a center and an extent (i.e. a bounding box) can be specified.

## Public Member Functions

- virtual vec3 [center](#center) () const =0 — Return the center of the object in local coordinates without considering any matrix.
- virtual vec3 [extent](#extent) () const =0 — Return the extent of the object in local coordinates without considering any matrix.
