`#include <ImFusion/GUI/SliceView.h>`

## Overview

2D renderer for SharedImageSets that can be hosted inside a [Display](/cpp-sdk/4.5/api/classes/imfusion-gui-display).

This class wraps the [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview) renderer that implements 2D/MPR rendering of one or multiple images with compositing and blending of any number of regular geometry-based [GlObjects](/cpp-sdk/4.5/api/classes/imfusion-globject). It is mostly a wrapper object to combine the plain OpenGL renderer with input event handling functionality and put it into the a [Display](/cpp-sdk/4.5/api/classes/imfusion-gui-display). Note that the [GlSlice::is3D()](/cpp-sdk/4.5/api/classes/imfusion-glslice) flag of the underlying renderer defines whether this view accepts 2D or 3D images.

It comes with a [SliceViewDefaultEventHandler](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceviewdefaulteventhandler) pre-installed so that it is ready to use directly after instantiation. Furthermore, it provides a couple of additional high-level member functions to configure the most important properties of the renderer.

**See also:** [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview), [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice), [GlSliceRenderer](/cpp-sdk/4.5/api/classes/imfusion-glslicerenderer)

## Member Enumeration Documentation

### ZoomMode

```cpp
enum class ImFusion::GUI::SliceView::ZoomMode
```

*strong*

Enumeration to describe the zooming mode used for a 2D view.

**Enumerator:**

- `FitToView` — A zoom factor of 1 will fit the shown image to the viewport.
- `Fixed` — A zoom factor of 1 will show 1mm per viewport pixel.
- `OriginalPixelSize` — A zoom factor of 1 will show 1 image pixel (of the first visible image) per viewport pixel.

### setSliceDepth

```cpp
void ImFusion::GUI::SliceView::setSliceDepth (const vec3 & center, bool suppressAnimation = false)
```

Changes the slice matrix so that it matches *center* in Z direction but remains unchanged in X and Y direction (view space).

Essentially, it projects *center* onto the current slice plane and then translates the Z coordinate of the view matrix. This function is useful if you want to center a set of MPR views to show a particular point without changing the in-plane translation of each view.

## Member Data Documentation

### signalSettingsChanged

```cpp
Signal ImFusion::GUI::SliceView::signalSettingsChanged
```

Signal emitted whenever one of the view settings changed which can be linked by ViewGroup3D.

**Warning:** This signal is part of the high-level [SliceView](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview) API, however it relates to settings of the underlying low-level GlSlice(View). This signal will only be emitted if changes are originate from the [SliceView](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview) API. Changes applied directly to the underlying GlSlice(View) may not reliably trigger this signal.
