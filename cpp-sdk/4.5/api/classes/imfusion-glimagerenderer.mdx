## Overview

[GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) to render an image into a [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview).

In contrast to the default render path for images of a [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview), [GlImageRenderer](/cpp-sdk/4.5/api/classes/imfusion-glimagerenderer) will not take part in the blending of multiple images that [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice) does. This can be useful if you want to blend multiple images and additionally show an non-blended image on top.
