## Member Function Documentation

### load

```cpp
bool ImFusion::Dicom::PixelSpacingCalibration::load (ImageDescriptor & desc) const
```

Loads spacing into the given descriptor and returns true if anything was loaded.

Automatically registers an error when an existing spacing cannot be loaded.

### save

```cpp
void ImFusion::Dicom::PixelSpacingCalibration::save (const ImageDescriptor & desc, const std::string & sopClassUID = "")
```

Stores the spacing in the dataset if it's metric.

If it's not metric, spacing is not set. Depending on the sopClassUID, sliceThickness or spacingBetweenSlices is set as well.
