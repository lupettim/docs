## Overview

template&lt;int Order&gt;

class ImFusion::Tensor&lt; Order &gt;
[Data](/cpp-sdk/4.5/api/classes/imfusion-data) class for strongly typed dense tensors of fixed rank and data type.

Unlike most [Data](/cpp-sdk/4.5/api/classes/imfusion-data) classes, this one notices content changes and notifies listeners. Any method in this class that changes the allocation of the underlying buffer may throw an exception on failure.

## Classes

- class [ConstAccess](/cpp-sdk/4.5/api/classes/imfusion-tensor-constaccess) — Helper class for read-only access to the data content of a [Tensor](/cpp-sdk/4.5/api/classes/imfusion-tensor).
- class [Access](/cpp-sdk/4.5/api/classes/imfusion-tensor-access) — Helper class for read/write access to the data content of a [Tensor](/cpp-sdk/4.5/api/classes/imfusion-tensor).
