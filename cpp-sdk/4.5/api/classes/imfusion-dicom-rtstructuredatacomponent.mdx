`#include <ImFusion/Dicom/RTStructureDataComponent.h>`

## Overview

[DataComponent](/cpp-sdk/4.5/api/classes/imfusion-datacomponent) for PointClouds loaded from a DICOM RTStructureSet.

Provides information about the original structure/grouping of the points. See RTStructureIoAlgorithm for details about how RTStructureSets are loaded.

WARNING: Since this component uses fixed indices into the [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud)'s points structure, it can only be used if the [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) remains unchanged!

### id

```cpp
std::string ImFusion::Dicom::RTStructureDataComponent::id () const
```

*override, virtual*

Returns a unique string identifier for this type of data component.

**Note:** To avoid conflicts make sure to include potential plugin names into this ID. Due to the modular architecture of the ImFusion library, uniqueness of IDs cannot be checked at compile time. However, [DataComponentFactory](/cpp-sdk/4.5/api/classes/imfusion-datacomponentfactory) will check the uniqueness of the ID during registration.
