## Overview

[Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) for basic image processing tasks.

## Public Slots

- void **onApply** () — Apply the chosen processing.
- void **onSetProcessing** (int val) — Change of selected processing.
- void **onUpdateCropping** () — Called to update the visualization of the cropping.
- void **onCroppingSize** (bool flag) — Called when the way of specifying cropping changes.
- void **onGradationModeChanged** (int idx)
- void **onUpdateThreshold** () — Called to update the visualization of thresholding.
