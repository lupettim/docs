## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to compute a hash from image data.

It calculates a hash and stores it in the hash parameter of this algorithm, allowing to e.g. save a workspace with it, such that on re-creating the algorithm the data is automatically validated.
