## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to compute curvature measures (mean curvature, Gaussian curvature, curvedness and shape index) of a mesh based on its implicit surface representation as signed distance volume.

If only a mesh is provided as input, the distance volume is first generated using [DistanceVolumeAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-distancevolumealgorithm). If a distance volume is provided (either alone or together with a mesh), it is directly used to compute the curvatures. The output are curvature volumes of the same size as the distance volume. If a mesh was provided, the volumetric curvatures are additionally linearly interpolated to the mesh vertices and the resulting per-vertex curvatures can be obtained using getters.
