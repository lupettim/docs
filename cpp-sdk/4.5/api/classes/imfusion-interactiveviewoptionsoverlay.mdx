## Overview

Button overlay for [InteractiveView](/cpp-sdk/4.5/api/classes/imfusion-interactiveview) that allows for setting up the view or display options of the view.

The settings are shown in a custom styled popup menu when clicking the button. This class currently supports [ImageView2D](/cpp-sdk/4.5/api/classes/imfusion-imageview2d) and [ImageView3D](/cpp-sdk/4.5/api/classes/imfusion-imageview3d).
