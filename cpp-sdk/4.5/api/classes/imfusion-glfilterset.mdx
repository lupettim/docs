## Overview

Processing multiple images with a standard filter.

## Classes

- class [ParamInfo](/cpp-sdk/4.5/api/classes/imfusion-glfilterset-paraminfo) — Simple container to store information about filter parameters and names.

- enum FilterMode &#123; **SHARPEN** = 0 , **GAUSSIAN** = 1 , **GUIDED_FILTER** = 2 , **ANISOTROPIC_DIFFUSION** = 3 , **BILATERAL** = 4 , **XCLAHE** = 5 , **NOISE_UNIFORM** = 6 , **NOISE_GAUSSIAN** = 7 , **NOISE_POISSON** = 8 , **GRADIENT** = 9 , **EDGE** = 10 , **MEDIAN** = 11 , **ADSOC** = 12 , **LOCALEXTREMAL** = 13 , **CUSTOM** = 14 &#125; — Enum for the different filters.
- enum ParamType &#123; **FLOAT** &#125; — Categories for data types passed to filter.

### takeOutput

```cpp

```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
