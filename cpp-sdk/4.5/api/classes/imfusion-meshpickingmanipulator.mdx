## Overview

[Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) picking manipulator The [MeshPickingManipulator](/cpp-sdk/4.5/api/classes/imfusion-meshpickingmanipulator) can return the list of selected vertices/faces/edges It offers several manipulation modes like DoubleClick, hover, alt + mouse key press For proper use the [MeshPickingManipulator](/cpp-sdk/4.5/api/classes/imfusion-meshpickingmanipulator) object must be added to a view as mesh manipulator and the outer object should subscribe to its signals e.g.

```cpp
GlMesh* glMesh = ...;
MeshPickingManipulator* manip = new MeshPickingManipulator(glMesh, *m_disp->view3D()->view());
manip->setSelectionMode(MeshPickingManipulator::SelectionMode::Face);
manip->signalSelectedFaceIndices.connect(this, &ExampleController::onSelectedFaceIndices);
auto meshPickingManipulator = new Interactive<GlMesh, MeshPickingManipulator>(glMesh, manip, true);
m_disp->view3D()->addObject(meshPickingManipulator.get());
```
 fragment
The manipulator must be removed from the view before deleting!

## Member Enumeration Documentation

### SelectionMode

```cpp
enum class ImFusion::MeshPickingManipulator::SelectionMode
```

*strong*

**Enumerator:**

- `Face` — &lt; No signals will be emitted
signalSelectedFaceIndices will be emitted when mesh face/faces are selected
- `Vertex` — signalSelectedVertexIndices will be emitted when mesh vertex/vertices are selected
- `Edge` — signalSelectedEdgeIndices will be emitted when mesh edge/edges are selected

### ManipulatorMode

```cpp
enum class ImFusion::MeshPickingManipulator::ManipulatorMode
```

*strong*

**Enumerator:**

- `DoubleClick` — one index is picked up when the user double clicks on the element with the left mouse button
- `Hover` — one index is picked up when the user hovers with the mouse above the element
- `AltDrag` — several indices are picked up when the user presses ALT, clicks the left mouse button and drags above the elements (is not supported for edges),
ALT + mouse wheel changes the size of the selection area
