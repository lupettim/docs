## Member Enumeration Documentation

### Coordinates

```cpp
enum ImFusion::GUI::PixelInfoOverlay::Coordinates
```

Bitmask enum to select which coordinates to show in overlay.

**See also:** CoordinateSystems

**Enumerator:**

- `NoCoordinates` — Don't show coordinates under mouse cursor.
- `ShowPixelCoordinates` — Show pixel coordinates under mouse cursor.
- `ShowWorldCoordinates` — Show world coordinates under mouse cursor.

### ValueDomains

```cpp
enum ImFusion::GUI::PixelInfoOverlay::ValueDomains
```

Bitmask enum to select which value domain/unit to use when printing pixel intensities.

**See also:** PixelValueDomains

**Enumerator:**

- `ShowOriginalValuesOnly` — Show only values in original value domain, i.e. storage values with shift/scale applied.
- `ShowStorageValues` — Show only values as stored currently in memory.
- `ShowRealWorldUnits` — Additionally print units if a [RealWorldMappingDataComponent](/cpp-sdk/4.5/api/classes/imfusion-realworldmappingdatacomponent) is present.
Warning[PixelInfoOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-pixelinfooverlay) does not validate if the unit information is accurate.
- `ShowLabelNames` — Additionally print label names if a [LabelDataComponent](/cpp-sdk/4.5/api/classes/imfusion-labeldatacomponent) is present.
