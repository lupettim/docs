`#include <ImFusion/GL/GlImageHighlighter.h>`

## Overview

[GlObject](/cpp-sdk/4.5/api/classes/imfusion-globject) that highlights an image by drawing a colored layer on top of it.

The intensity of the color layer is multiplied by the intensity of the image in the corresponding shader. SharedImageSets can be added to/removed from this class via the addImage/removeImage methods.
