## Member Function Documentation

### add

```cpp
void ImFusion::OptimizationStudy::add (const std::vector < double > & pInit, const std::vector < double > & pOpt, int eval, double time, double costInit, double costOpt, const std::vector < double > & errorInit, const std::vector < double > & errorOpt)
```

Add results for one optimization run to study.

**Parameters:**

- `pInit` — start parameters
- `pOpt` — optimized parameters
- `eval` — number of optimizer evaluations
- `time` — execution time of the optimization
- `costInit` — initial cost function value
- `costOpt` — cost function value at optimum
- `errorInit` — initial vector of error measures
- `errorOpt` — vector of error measures at optimum

### plotData

std::vector < double > ImFusion::OptimizationStudy::plotData (int which) const

Return a single parameter for all study samples.

The numbering of parameters corresponds to the [OptimizationDialog](/cpp-sdk/4.5/api/classes/imfusion-optimizationdialog) study plot comboboxes.

### setCustomDisplacements

```cpp
void ImFusion::OptimizationStudy::setCustomDisplacements (const std::vector < std::vector < double > > & disp)
```

*inline*

Allows to set custom displacements to use by the [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) class (needs to be of selection dimension, not full parameter dimension).

The range parameter in the [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) class will be multiplied upon the values here.
