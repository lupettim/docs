## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to interpolate between two label maps.

Input are two binary masks, output is one (or more) interpolated frames. Inspired and adapted from 10.1109/TBME.2008.921158.
