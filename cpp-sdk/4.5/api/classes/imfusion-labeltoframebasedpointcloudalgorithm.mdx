## Overview

Given a label map and a tracked shared image set, create a frame based point cloud.

The outline of the label map is taken, resampled to a more sparse point cloud (strength of resampling configurable via the parameter p_takeEveryXPoint), and the [FrameBasedPointCloud](/cpp-sdk/4.5/api/classes/imfusion-framebasedpointcloud) is constructed with the provided [TrackedSharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-trackedsharedimageset). Note that the point cloud is valid only thought the lifetime of the [TrackedSharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-trackedsharedimageset).

Constrains:

- the label map must be encoded with an unsigned char
- The label map must have the same length as the [TrackedSharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-trackedsharedimageset).
- The labels will be taken as altogether. If you require different point cloud for each label, you need to preprocess them and feed them one by one.

**See also:** [FrameBasedPointCloud](/cpp-sdk/4.5/api/classes/imfusion-framebasedpointcloud), [TrackedSharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-trackedsharedimageset)
