## Overview

Stacks fragmented values of a single channel UByte/UShort-typed labelmap inplace into consecutive integer values [0, ..., n].

Some segmentation maps have fragmented values, e.g. [0, 1, 2, 7, 8, 20] which should rather be [0, 1, 2, 3, 4, 5]. This algorithm makes sure that there are no holes in the integer ranges so that all values are consecutive. This is especially important if at least one value is over 255 so that the image cannot be converted to uint8 and thus cannot be converted to the LABEL modality and thus not be properly displayed as a label map
