## Overview

Attaches a displacement field as a [GlDenseDeformation](/cpp-sdk/4.5/api/classes/imfusion-gldensedeformation).

Any existing deformation is replaced. An existing [GlDenseDeformation](/cpp-sdk/4.5/api/classes/imfusion-gldensedeformation) is not replaced by a new instance. A displacement field is a 3-channel float image. Only changes the focus frame of the set, unless both input images are volume sets of equal size.
