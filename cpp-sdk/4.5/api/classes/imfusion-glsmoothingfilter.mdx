### setKernelSize

```cpp
bool ImFusion::GlSmoothingFilter::setKernelSize (double size, vec3 spacing, double k = -1.0)
```

*override, virtual*

Roughly set kernel size in mm, which must be larger than each of the components in spacing.

Spacing is the pixel spacing of the image that should be filtered, in mm. The actual size of the resulting kernel in pixels will be the smallest odd integer that is at least size divided by spacing. k parameter is unused. Returns whether the kernel was set successfully.
