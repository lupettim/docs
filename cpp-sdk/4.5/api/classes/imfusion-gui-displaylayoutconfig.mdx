`#include <ImFusion/GUI/DisplayLayoutConfig.h>`

## Overview

The [DisplayLayoutConfig](/cpp-sdk/4.5/api/classes/imfusion-gui-displaylayoutconfig) serves as static "snapshot" of the view configuration of a [DisplayLayout](/cpp-sdk/4.5/api/classes/imfusion-gui-displaylayout).

Instead of using recursive binary subdivision, it uses a table/grid-like representation where a [View](/cpp-sdk/4.5/api/classes/imfusion-gui-view) is assigned to one cell. By default the grid is regular but each row/column can be adjusted in size and views can span multiple cells in either dimension. In a valid configuration each cell is occupied by exactly one view and rows/columns do not overlap each other.

When manually specifying a [DisplayLayout](/cpp-sdk/4.5/api/classes/imfusion-gui-displaylayout) it is often easier/more intuitive to do so in a grid-like fashion and create a temporary [DisplayLayoutConfig](/cpp-sdk/4.5/api/classes/imfusion-gui-displaylayoutconfig), and then assign it via [DisplayLayout::loadConfiguration()](/cpp-sdk/4.5/api/classes/imfusion-gui-displaylayout).

**See also:** [DisplayBase](/cpp-sdk/4.5/api/classes/imfusion-gui-displaybase), [DisplayLayout](/cpp-sdk/4.5/api/classes/imfusion-gui-displaylayout), [DisplayAutoLayouter](/cpp-sdk/4.5/api/classes/imfusion-gui-displayautolayouter)

## Member Function Documentation

### addView

```cpp
bool ImFusion::GUI::DisplayLayoutConfig::addView (const GUI::View * view, int x, int y, int spanX = 1, int spanY = 1)
```

Adds a new view at the given coordinates to the configuration.

**Parameters:**

- `view` — The view to add
- `x` — column number, must be between 0 and gridX()-1
- `y` — row number, must be between 0 and gridY()-1
- `spanX` — spans the view over spanX columns, default is 1. Must be larger than 0.
- `spanY` — spans the view over spanY rows, default is 1. Must be larger than 0.

**Returns:** false if the view overlaps with another view or the location/span is invalid, true otherwise

### addSubLayout

```cpp
bool ImFusion::GUI::DisplayLayoutConfig::addSubLayout (const DisplayLayoutConfig & sublayout, int x, int y, int spanX = 1, int spanY = 1)
```

Adds a new sublayout at the given coordinates to the configuration.

The sublayout must be valid.

**Parameters:**

- `sublayout` — The layout to add
- `x` — column number, must be between 0 and gridX()-1
- `y` — row number, must be between 0 and gridY()-1
- `spanX` — spans the view over spanX columns, default is 1. Must be larger than 0.
- `spanY` — spans the view over spanY rows, default is 1. Must be larger than 0.

**Returns:** false if the sublayout overlaps with another sublayout/view or the location/span is invalid, true otherwise

### resize

```cpp
void ImFusion::GUI::DisplayLayoutConfig::resize (int newX, int newY, int offsetX = 0, int offsetY = 0)
```

Resizes the grid of the configuration.

The grid will take size defined by newX and newY. All views are moved by the offset. WARNING: shrinking the view has not been tested yet

### removeRow

```cpp
void ImFusion::GUI::DisplayLayoutConfig::removeRow (int row)
```

### spanViewsHorizontal

```cpp
void ImFusion::GUI::DisplayLayoutConfig::spanViewsHorizontal (bool leftToRight)
```

Extends the column span of all views over empty, neighboring columns.

If leftToRight is true, views will only extend to the right. Otherwise, they will extend to the left.

### spanViewsVertical

```cpp

```

Extends the row span of all views over empty, neighboring rows.

If topToBottom is true, views will only extend to the bottom. Otherwise, they will extend to the top.
