`#include <ImFusion/GUI/InteractiveHistogramOverlay.h>`

## Overview

Wraps a GlHistogramOverlay object (that contains a [GlHistogram](/cpp-sdk/4.5/api/classes/imfusion-glhistogram)) in an [InteractiveOverlay](/cpp-sdk/4.5/api/classes/imfusion-interactiveoverlay).

## Public Slots

- void **onViewDataChanged** () — Update the connection to the View's DisplayOptions.
- void **setWindow** (double window) — Set the window.
- void **setLevel** (double level) — Set the level.
- void **update** () — Update the overlay (including the histogram).
- void **updateHistogram** () — Update the histogram.
