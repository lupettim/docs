## Overview

The [OverlayAnnotation](/cpp-sdk/4.5/api/classes/imfusion-dicom-overlayannotation) displays any DICOM overlay stored in a [OverlayDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-overlaydatacomponent).

The Annotation currently only supports single images and image sets. Only supports changing the color but no other properties of [GlAnnotation](/cpp-sdk/4.5/api/classes/imfusion-glannotation) (such as label or editable).

**See also:** [OverlayDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-overlaydatacomponent), OverlayAnnotationDataController ,
