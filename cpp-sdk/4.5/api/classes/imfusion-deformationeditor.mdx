## Overview

The [DeformationEditor](/cpp-sdk/4.5/api/classes/imfusion-deformationeditor) algorithm modifiers the deformation property of a [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset).

It can add a specific deformation model or remove all deformations. If the input [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) contains multiple images, the deformation is expected to have the same properties on all images (e.g. same model and same grid size but not necessarly the same deformation).

## Member Enumeration Documentation

### DeformationModel

```cpp
enum ImFusion::DeformationEditor::DeformationModel
```

Enumeration of supported deformation types.

**Enumerator:**

- `None` — No deformation model (also used to remove other models).
- `FFD` — Free-Form deformations.
- `TPS` — Thin Plate [Spline](/cpp-sdk/4.5/api/classes/imfusion-spline) deformations.
- `Dense` — Dense deformation field.
- `Parametric` — Other parametric deformation model not known by the controller.

```cpp
void ImFusion::DeformationEditor::setDeformationModel (DeformationModel model)
```

Sets the deformation model for the specified frame.

If the current deformation model differs, a new deformation object is created and the old object is replaced. To remove an exist deformation use DeformationModel::None.
