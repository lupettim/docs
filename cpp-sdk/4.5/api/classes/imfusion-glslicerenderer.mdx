`#include <ImFusion/GL/GlSliceRenderer.h>`

## Overview

Interface for a class that can render data into a GlSlice/GlSliceView.

Implement the [Configurable](/cpp-sdk/4.5/api/classes/imfusion-configurable) interface in order to provide an automatically generated [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) for configuration when using the [InteractiveViewOptionsOverlay](/cpp-sdk/4.5/api/classes/imfusion-interactiveviewoptionsoverlay).

**See also:** [ImageView2D](/cpp-sdk/4.5/api/classes/imfusion-imageview2d), [GlSliceView](/cpp-sdk/4.5/api/classes/imfusion-glsliceview), [GlSlice](/cpp-sdk/4.5/api/classes/imfusion-glslice), [InteractiveViewOptionsOverlay](/cpp-sdk/4.5/api/classes/imfusion-interactiveviewoptionsoverlay)

### render

```cpp
virtual void ImFusion::GlSliceRenderer::render (const GL::ViewState & viewState, const Slice & slice, const Data & data, const DisplayOptions2d & displayOptions, bool encodeDistance)
```

*pure virtual*

Renders the data in the given configuration.

**Parameters:**

- `viewState` — GlState object defining view and projection matrices.
- `slice` — [Slice](/cpp-sdk/4.5/api/classes/imfusion-slice) on which to render the data.
- `data` — The data to render
- `displayOptions` — The display options to use for rendering.
- `encodeDistance` — Flag whether to encode the distance to the border in the secondary color buffer.

**Note:** Call this method only with data that is accepted by this renderer.
