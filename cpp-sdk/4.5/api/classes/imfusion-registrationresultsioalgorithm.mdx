## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) for loading ImFusion Registration Results (irr) files.

Only meant to be used in the ImFusion Suite, in the SDK use [RegistrationResults::load](/cpp-sdk/4.5/api/classes/imfusion-registrationresults). This algorithm will add [RegistrationResultsAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-registrationresultsalgorithm) to the main window and load the results inside it.
