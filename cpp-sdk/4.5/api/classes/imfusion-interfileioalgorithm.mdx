## Overview

[IoAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-ioalgorithm) for loading and saving images stored in Interfile format.

This algorithm only supports a subset of the complete Interfile specification. Currently supported:

- Only 'Tomographic' type of data is supported
- loading pixel data (except bit and ASCII datatypes)
- loading of single 2D and 3D images
- loading patient and study info
- loading of spacing and rescaling

### takeOutput

```cpp
OwningDataList ImFusion::InterfileIoAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
