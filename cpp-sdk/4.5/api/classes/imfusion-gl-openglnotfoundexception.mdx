`#include <ImFusion/Core/GL/Context.h>`

## Overview

Specialization of [std::runtime_error](http://en.cppreference.com/w/cpp/error/runtime_error.html) indicating that OpenGL was not able to initialize correctly.

This is usually the case if the current graphics driver does not support OpenGL at all or no display is attached (in case of Windows and GLX).

## Public Member Functions

- **OpenGLNotFoundException** (const [std::string](http://en.cppreference.com/w/cpp/string/basic_string.html) &message) — Constructs a new [OpenGLNotFoundException](/cpp-sdk/4.5/api/classes/imfusion-gl-openglnotfoundexception) with the given message and OpenGL error code.
