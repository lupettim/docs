## Overview

### load

```cpp
std::vector < std::unique_ptr < Data > > ImFusion::Dicom::GeneralEquipmentExtension::load (const FrameDescriptor & frame, SharedImageSet & sis) const
```

*override, virtual*

Loads the relevant tags into a [GeneralEquipmentModuleDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-generalequipmentmoduledatacomponent).

Tags are only taken from the very first frame (frame.sharedImageIndex and .sliceIndex are 0). Empty string or 0 is used for any field in the [DataComponent](/cpp-sdk/4.5/api/classes/imfusion-datacomponent) when the corresponding tag is missing in the dataset.

### save

```cpp
void ImFusion::Dicom::GeneralEquipmentExtension::save (const FrameDescriptor & frame, const SharedImageSet & sis) const
```

*override, virtual*

Save data from the given [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) to DICOM.

This method will be called once for each frame of the [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset). Use [FrameDescriptor::iod](/cpp-sdk/4.5/api/structs/imfusion-dicom-framedescriptor) to access the DICOM tags and [FrameDescriptor::sharedImageIndex](/cpp-sdk/4.5/api/structs/imfusion-dicom-framedescriptor)/sliceIndex to figure out which frame data should be saved. The matrix parameters of frame should be ignored. Use the matrix of the [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) instead.
