## Overview

[Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) with a dummy algorithm.

Adds an [GUI::LiftChartOverlay](/cpp-sdk/4.5/api/classes/imfusion-gui-liftchartoverlay) ([GlLiftChartOverlay](/cpp-sdk/4.5/api/classes/imfusion-glliftchartoverlay)) to the views. Shows a list of annotations and controls which of these to show in the overlay, and with what color.
