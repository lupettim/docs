## Member Enumeration Documentation

### Mode

```cpp
enum class ImFusion::DistanceTransformAlgorithm::Mode
```

*strong*

**Enumerator:**

- `Outer` — Compute only the distance outside the object.
- `Inner` — Compute only the distance inside the object.
- `Both` — Compute the distance both inside and outside.
- `Signed` — Compute the distance both inside and outside, and encodes the inner distances as negative values.

### takeOutput

```cpp
OwningDataList ImFusion::DistanceTransformAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
