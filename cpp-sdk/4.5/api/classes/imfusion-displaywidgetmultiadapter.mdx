`#include <ImFusion/GUI/DisplayWidgetMultiAdapter.h>`

## Overview

Helper class providing the API of [DisplayWidgetMulti](/cpp-sdk/4.5/api/classes/imfusion-displaywidgetmulti) while wrapping around either a [DisplayWidgetMulti](/cpp-sdk/4.5/api/classes/imfusion-displaywidgetmulti) or a [GUI::RadiologyDisplay](/cpp-sdk/4.5/api/classes/imfusion-gui-radiologydisplay).

It is mainly intended as a temporary drop-in replacement for [DisplayWidgetMulti](/cpp-sdk/4.5/api/classes/imfusion-displaywidgetmulti) objects in Controllers that shall work with both display layers during the transition period while migrating from the old Qt-based display framework to the new ImFusion::GUI framework.

**Warning:** To mimic existing use cases this adapter class has nullable/pointer semantics and can be 'null'. If it is null conversion to bool will yield false and comparison to nullptr will be true, and you must not call any of its other member functions.
