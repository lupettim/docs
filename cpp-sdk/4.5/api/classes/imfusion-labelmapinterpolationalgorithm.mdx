## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to run a 3D interpolatation of a label map with sparsely labelled slices EITHER Input is a 3D label map with a couple of non-zero z-slices, output is the interpolated volume OR Input is a set of 2D label maps some of them being not empty, output is the interpolated 2D sequence.
