## Overview

[Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) for the [DeformationEditor](/cpp-sdk/4.5/api/classes/imfusion-deformationeditor) algorithm.

The [DeformationEditorController](/cpp-sdk/4.5/api/classes/imfusion-deformationeditorcontroller) provides means to configure and visualize different deformation models of a [DeformationEditor](/cpp-sdk/4.5/api/classes/imfusion-deformationeditor) algorithm.

**See also:** ImFusion::GlControlPoints

## Public Slots

- void **addDeformation** () — Sets the deformation model to the one currently selected in the combobox.
- void **linkDeformation** (const [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) &other) — Links the deformation to the deformation of another image.
- void **removeDeformation** () — Removes the deformation object by setting the deformation model to None.
- void **saveCurrentDeformation** () — Saves the current deformation.
- void **removeSavedDeformation** (int which) — Removes the given index from the list of saved deformations.
- void **restoreDeformation** (int which) — Restores the deformation of the given index.
- void [showDisplacementGrid](#showdisplacementgrid) (bool show) — Shows the deformation on a grid (.
