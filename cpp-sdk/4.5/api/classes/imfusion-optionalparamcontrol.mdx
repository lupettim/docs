`#include <ImFusion/GUI/PropertiesWidgetParamControls.h>`

## Overview

Composite control that wraps an existing [ParamControl](/cpp-sdk/4.5/api/classes/imfusion-paramcontrol) and adds a checkbox to the left of it.

The checkbox controls the enabled-state of the sub control and represents the [std::optional::has_value()](http://en.cppreference.com/w/cpp/utility/optional/operator_bool.html) state
