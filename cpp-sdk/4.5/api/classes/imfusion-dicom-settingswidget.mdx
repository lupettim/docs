## Member Enumeration Documentation

### Contents

```cpp
enum class ImFusion::Dicom::SettingsWidget::Contents
```

*strong*

Bitflag enumeration to define which settings to show.

**Enumerator:**

- `Cache` — Show settings regarding the DicomDatasetCache.
- `DcmtkSettings` — Show settings regarding the DCMTK backend.
- `Anonymization` — Show settings regarding the anonymization of datasets.
- `PACS` — Show PACS settings.
- `Everything` — Show all settings.

## Member Function Documentation

### updateWidget

```cpp
bool ImFusion::Dicom::SettingsWidget::updateWidget ()
```

*override, virtual*

Update the widget in the [Settings](/cpp-sdk/4.5/api/classes/imfusion-settings) Dialog.

By default the settings fields of the custom settings will not change after they were registered with [SettingsDialog](/cpp-sdk/4.5/api/classes/imfusion-settingsdialog). However if you need to update the fields of your settings widget by adding or removing sub-properties and parameters, have this function return true to indicate that.
