## Member Function Documentation

### clone

```cpp
std::unique_ptr < Context > ImFusion::GL::ContextMacOS::clone ()
```

*override, virtual*

Instantiates a new OpenGL context that is shared with this one.

The freshly created context will be made current. This method will throw if the context could not be initialized.

**Note:** If you plan to use this context in a different thread, you will need to call [moveToCurrentThread()](/cpp-sdk/4.5/api/classes/imfusion-gl-context) on the returned context before.
