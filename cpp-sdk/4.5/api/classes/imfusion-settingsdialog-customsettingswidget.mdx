### updateWidget

```cpp
virtual bool ImFusion::SettingsDialog::CustomSettingsWidget::updateWidget ()
```

*inline, virtual*

Update the widget in the [Settings](/cpp-sdk/4.5/api/classes/imfusion-settings) Dialog.

By default the settings fields of the custom settings will not change after they were registered with [SettingsDialog](/cpp-sdk/4.5/api/classes/imfusion-settingsdialog). However if you need to update the fields of your settings widget by adding or removing sub-properties and parameters, have this function return true to indicate that.
