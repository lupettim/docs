`#include <ImFusion/GUI/DefaultControllerDecorator.h>`

## Overview

Wrapper class to decorate [Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) instances and place them in the [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) of [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase) using Qt Widgets.

This class is designed to work with [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase) and supports the available default locations of Controllers. Usually, you don't need to create instances of this class manually since [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase) takes care of it.

**See also:** [MainWindowBase](/cpp-sdk/4.5/api/classes/imfusion-mainwindowbase), [Controller](/cpp-sdk/4.5/api/classes/imfusion-controller)
