`#include <ImFusion/Mesh/AssignMeshTextureAlgorithm.h>`

## Overview

Copies a RGB 2D image to use as texture for a mesh.

In order to accuess this in the Suite UI, the mesh annotation needs to be assigned to the image and the annotation context menu has to be used.
