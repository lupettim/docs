- [std::array](http://en.cppreference.com/w/cpp/container/array.html)&lt; int, 10 &gt; **m_iPars** — Integer meta information holding: 0 - number of [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence) instances 1 - number of currently used [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence) instance 2 - number of time stamps stored as part of the block (either size of image set or 0) 3 - 1 if time stamps are used, otherwise 0 4-9 - for free usage in derived classes.

### saveTrackingSequencesAndMeta

```cpp
bool ImFusion::TrackedImageSetImFusionFilePlugin::saveTrackingSequencesAndMeta (TrackedSharedImageSet * tsis, std::ostream & file)
```

*protected*

Save tracking sequences and further information of *tsis* to file.

Uses the last item of m_iPars (the first four are overridden) and all items of m_dPars, so make sure to set them to meaningful value for the desired data type, e.g. ultrasound geometry. Returns true if loading was successful.
