`#include <ImFusion/GL/ApplyWindowLevelAlgorithm.h>`

## Overview

Clamps (in-place) the image values to the range [level-window/2; level+window/2].

The algorithm automatically retrieves the level/window value from the 2D display options of its input, but those values can be overriden.
