`#include <ImFusion/GUI/SliceIndicator.h>`

## Overview

[Interactive](/cpp-sdk/4.5/api/classes/imfusion-interactive) renderer to show and manipulate the position/extent of [SliceView](/cpp-sdk/4.5/api/classes/imfusion-gui-sliceview) instances in context of other Views.

This class wraps a [GlSliceIndicator](/cpp-sdk/4.5/api/classes/imfusion-glsliceindicator) to visualize existing SliceViews in other SliceViews and VolumeViews and combines it with input event handling code so that you can change the position and orientation of a slice by dragging its intersection line.
