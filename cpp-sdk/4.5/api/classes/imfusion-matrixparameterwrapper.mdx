## Overview

template&lt;typename T&gt;

class ImFusion::MatrixParameterWrapper&lt; T &gt;
Wrap a matrix into a [ParameterWrapperBase](/cpp-sdk/4.5/api/classes/imfusion-parameterwrapperbase) instance.

This class is templated to allow for mat3 (rotation matrices), mat4 (6-dof rigid transformations), [std::vector&lt;\*&gt;](http://en.cppreference.com/w/cpp/container/vector.html) of such, and the corresponding [Parameter&lt;\*&gt;](/cpp-sdk/4.5/api/classes/imfusion-parameter) type.

**Note:** : Scaling of angles is in degrees.

**Warning:** : For [Parameter&lt;\*&gt;](/cpp-sdk/4.5/api/classes/imfusion-parameter) types, each deserialize may trigger a signalChanged
