## Overview

A [ContextMenuEvent](/cpp-sdk/4.5/api/classes/imfusion-gui-contextmenuevent) signals that a context-sensitive popup menu was requested by the user, for instance by a click with the right mouse button.

Receivers of this event should check if they should respond to it based on the position() and if so populate the contextMenu field of the [EventResult](/cpp-sdk/4.5/api/structs/imfusion-gui-eventresult) they return.

Constructor for a [ContextMenuEvent](/cpp-sdk/4.5/api/classes/imfusion-gui-contextmenuevent).

**Parameters:**

- `position` — Position of the mouse pointer wrt. the main GUI element enabling receiver to infer the context.
- `modifiers` — List of pressed keyboard modifier keys.
- `mappingFunction` — Optional mapping function used by [mapToGlobalCoord()](/cpp-sdk/4.5/api/classes/imfusion-gui-inputevent) to convert event coordinates to global coordinates if the underlying windowing system supports it; [mapToGlobalCoord()](/cpp-sdk/4.5/api/classes/imfusion-gui-inputevent) will use the identity function if empty.
