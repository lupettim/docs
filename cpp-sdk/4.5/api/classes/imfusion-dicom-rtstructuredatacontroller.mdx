`#include <ImFusion/Dicom/GUI/RTStructureDataController.h>`

## Overview

A [DataController](/cpp-sdk/4.5/api/classes/imfusion-datacontroller) for PointClouds that contain a [RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent).

It generates rendering data for the RTStructureAnnotations (which are usually managed by [RTStructureDataDisplayHandler](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatadisplayhandler)). The meshes are generated with [RTStructureToLabelMap](/cpp-sdk/4.5/api/classes/imfusion-rtstructuretolabelmap) and [LabelToMeshAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-labeltomeshalgorithm) in a separate thread. It also shows a list of all RT Structure Sets together with a checkbox to make them visible in the display (providing similar features as the [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget)).
