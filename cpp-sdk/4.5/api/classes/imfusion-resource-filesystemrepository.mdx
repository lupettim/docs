`#include <ImFusion/Core/Resource/FilesystemRepository.h>`

## Overview

[Repository](/cpp-sdk/4.5/api/classes/imfusion-resource-repository) for loading resources from the local filesystem.

Maintains a set of search paths, in which it looks for matching files.

**Note:** Be aware that the search paths may be different for every system. Thus, pay particular attention if you plan deploying your software to other computers than your local build system. Writing a custom [Repository](/cpp-sdk/4.5/api/classes/imfusion-resource-repository) may be the more robust alternative in such cases.
