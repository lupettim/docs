`#include <ImFusion/Base/PropertiesInspector.h>`

## Overview

Small algorithm allowing the user to inspect a [Data](/cpp-sdk/4.5/api/classes/imfusion-data)'s properties and DataComponents.

The actual [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) does almost nothing. All the logic is done in the corresponding [Controller](/cpp-sdk/4.5/api/classes/imfusion-controller).

**See also:** PropertiesInspectorController
