`#include <ImFusion/Base/ParameterOptimization.h>`

## Overview

Class for combining a [ParameterWrapperBase](/cpp-sdk/4.5/api/classes/imfusion-parameterwrapperbase) instance with a cost function over C++ objects to produce a [CostFunction](/cpp-sdk/4.5/api/classes/imfusion-costfunction) that can be optimized over with an [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer) instance.

**See also:** [ParameterWrapperBase](/cpp-sdk/4.5/api/classes/imfusion-parameterwrapperbase)
