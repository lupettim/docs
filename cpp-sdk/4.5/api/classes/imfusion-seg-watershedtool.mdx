## Overview

Wrapper class around the [WatershedComputer](/cpp-sdk/4.5/api/classes/imfusion-watershedcomputer) to simplify common use cases.

## Member Enumeration Documentation

### StopReason

```cpp
enum ImFusion::Seg::WatershedTool::StopReason
```

**Enumerator:**

- `MissingBackground` — the computation was successful and the result can be retrieved
- `MissingLabels` — there were no background seeds in the label map
- `Canceled` — there were no label seeds in the label map
- `Unknown` — the computation was cancelled by the user
