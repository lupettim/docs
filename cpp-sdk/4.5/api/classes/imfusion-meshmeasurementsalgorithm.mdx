## Overview

A class that provides functions for measuring the lengths of certain curves on a mesh.

See the function documentation for the different kinds of supported curves.

### computePlaneAndContour

```cpp
std::vector < vec3 > ImFusion::MeshMeasurementsAlgorithm::computePlaneAndContour (const std::pair < unsigned int, vec3 > & face1, const std::pair < unsigned int, vec3 > & face2, mat4 & planeMat, double & planeWidth, double & planeHeight)
```

Compute a measurement plane and a countour resulting from its intersection with the mesh.

**Parameters:**

- `face1` — face index and a point from this face. The normal of this face will be used as an x axis for the measurement plane
- `face2` — face index and a point from this face. The vector connecting this point and a point from face1 will be used as a y axis of the measurement plane
- `planeMat` — output matrix of the measurement plane
- `planeWidth` — output width of the plane (computed as a dimension of the bounding box of intersections of the measurement plane with the connected faces starting from face1)
- `planeHeight` — output height of the plane (computed as a dimension of the bounding box of intersections of the measurement plane with the connected faces starting from face2)

**Returns:** contour resulting from intersecting a measurement plane with the mesh (only one connected contour that passes through face1 is returned)

Compute a coutour representing a geodesic distance between points on the mesh faces.

**Parameters:**

- `face1` — face index and a point from this face
- `face2` — face index and a point from this face

**Returns:** contour representing a geodesic distance between points from face1 and face2

### computeContours

```cpp

```

Compute countours resulting from intersection of plane with the mesh.

**Parameters:**

- `planeMat` — matrix of the measurement plane
- `planeWidth` — width of the plane
- `planeHeight` — height of the plane

**Returns:** contours resulting from intersecting a measurement plane with the mesh (all connected contours are returned)
