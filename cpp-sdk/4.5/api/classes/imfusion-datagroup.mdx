## Overview

[Data](/cpp-sdk/4.5/api/classes/imfusion-data) element that groups multiple [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instances together in a hierarchical fashion.

A [DataGroup](/cpp-sdk/4.5/api/classes/imfusion-datagroup) consists of a mutable set of other [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instances forming its children that can be added, removed or moved around at any time after construction. Furthermore, you can define an optional proxy child, which is one item of the list of children that may serve as representative dataset for the group. Clients may use or refer to the proxy child instead of the entire [DataGroup](/cpp-sdk/4.5/api/classes/imfusion-datagroup) in some situations. For instance, if a proxy child is defined the [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget) may show the dataset in the place of the [DataGroup](/cpp-sdk/4.5/api/classes/imfusion-datagroup) instead of listing it as a child.

**Note:** This class serves mainly as hierarchical container type to be used by [DataModel](/cpp-sdk/4.5/api/classes/imfusion-datamodel). It is discouraged to derive from [DataGroup](/cpp-sdk/4.5/api/classes/imfusion-datagroup) in order to create your own hierarchical [Data](/cpp-sdk/4.5/api/classes/imfusion-data) type. Use the [CompoundData](/cpp-sdk/4.5/api/classes/imfusion-compounddata) interface instead which gives you more flexibility, for instance wrt. to ownership semantics.

- Signal [signalProxyChildChanged](#signalproxychildchanged) — &lt; Signal emitted when a [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance has changed its position in [children()](#children)
- Signal&lt; Data \* &gt; **signalChildAboutToBeRemoved** — Signal emitted when a [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance is about to be removed or taken from [children()](#children).
- Signal&lt; Data \* &gt; **signalChildAdded** — &lt; Signal emitted when a [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance is about to be removed or taken from [children()](#children)
- Signal&lt; Data \* &gt; **signalChildMoved** — &lt; Signal emitted when a new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance has been added to [children()](#children)

- bool **move** (Data \*data, int position) — Moves *data* within the list of children to the given position.
- [std::unique_ptr](http://en.cppreference.com/w/cpp/memory/unique_ptr.html)&lt; Data &gt; [remove](#remove) (Data \*data) — Removes *data* from the list of children and returns its ownership to the caller.
- void **clear** () — Removes all children from this group.
- [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) [children](#children) () const override — Returns the list of direct children of this group.
- [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) **childrenRecursive** () const — Returns the flat list of children of this group and of all recursive subgroups in a depth-first order.
- Data \* **proxyChild** () const — Returns an optional item from [children()](#children) that shall act as representative [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance for this group.
- bool [setProxyChild](#setproxychild) (Data \*data) — Sets an optional item from [children()](#children) that shall act as representative [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance for this group.

## Member Function Documentation

### remove

```cpp
std::unique_ptr < Data > ImFusion::DataGroup::remove (Data * data)
```

Removes *data* from the list of children and returns its ownership to the caller.

If *data* is the same as proxyChild() the proxy child will be set to nullptr. Returns a nullptr in case of failure.
