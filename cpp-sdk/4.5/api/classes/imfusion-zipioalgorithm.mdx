`#include <ImFusion/IO/ZipIoAlgorithm.h>`

## Overview

Loads data from a ZIP archive.

Will extract the archive to a temporary location and then launch [IO::FileLoader](/cpp-sdk/4.5/api/classes/imfusion-io-fileloader) or [IO::FolderLoader](/cpp-sdk/4.5/api/classes/imfusion-io-folderloader) to complete the loading process.

### takeOutput

```cpp
OwningDataList ImFusion::ZipIoAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).
