### setInitVariances

```cpp
void ImFusion::OptimizerXNES::setInitVariances (std::vector < double > initVar)
```

*inline*

Set initial standard deviations for the parameters by supplying a vector&lt;double&gt; of values.

As a rule of thumb, set it to the distance you expect the optimum to difer from the initial value. Alternatively, you can use setInitVariance(double var) to set a single value for all dimensions. If the vector is shorter than the number of dimensions, the default variance is used for the dimensions that are not covered by the vector.

### setSparseMode

```cpp
void ImFusion::OptimizerXNES::setSparseMode (bool sparse = true)
```

Set the mode of the optimizer to SPARSE.

Supply false as argument to turn SPARSE off. In sparse mode, by default the covariance matrix only updates the diagonal. Additional correlations have to be add explicitly using addSparseCorrelation(int k, int l) The SPARSE mode can be used to optimize large dimensional problems

### addSparseConnection

```cpp
void ImFusion::OptimizerXNES::addSparseConnection (int k, int l)
```

Add a sparse correlation when using the algorithm in SPARSE mode.

In SPARSE mode, the correlation matrix is only updated for the diagonal and the connections explicitly added using this function.
