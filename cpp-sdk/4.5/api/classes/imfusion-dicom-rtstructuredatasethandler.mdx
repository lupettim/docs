## Overview

A [DatasetHandler](/cpp-sdk/4.5/api/classes/imfusion-dicom-datasethandler) for loading DICOM RTStructureSet.

Each ROI is loaded as a [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud). All points from all 'Contour [Data](/cpp-sdk/4.5/api/classes/imfusion-data)' inside a 'ROI Contour [Sequence](/cpp-sdk/4.5/api/classes/imfusion-dicom-sequence)' are combined in into one [PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud). The original structure of contours is stored in a [RTStructureDataComponent](/cpp-sdk/4.5/api/classes/imfusion-dicom-rtstructuredatacomponent).

Instead of using this class directly, [DicomIoAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-dicomioalgorithm) should be used.
