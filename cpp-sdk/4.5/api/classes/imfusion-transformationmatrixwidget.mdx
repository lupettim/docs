`#include <ImFusion/GUI/TransformationMatrixWidget.h>`

## Overview

Widget to transform a matrix, with global/local and toWorld/fromWorld modifiers.

Flexible widget that can be inserted in a dialog, allows the user to modify a matrix. Supports copy/paste, using a different matrix convention, and reset/invert actions

### Type


Bitset enumeration to define how to show the menu.

**Enumerator:**

- `Translation` — Translation.
- `Rotation` — Rotation.
- `ScaleShear` — Scale, Shear.
- `Rigid` — Translation, Rotation.
- `Affine` — Translation, Rotation, Scale, Shear.

### Options

```cpp
enum ImFusion::TransformationMatrixWidget::Options
```

**Enumerator:**

- `Tools` — Set Identity, Reset, Invert.
- `CopyPaste` — Copy, Paste.
- `GlobalLocal` — Global or Local.
- `ToFromWorld` — To world, from world.
- `All` — Set Identity, Reset, Invert, Copy, Paste, To world, from world.
