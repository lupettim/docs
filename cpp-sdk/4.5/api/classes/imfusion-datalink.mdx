## Overview

Interface for linking properties between multiple [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instances.

A [DataLink](/cpp-sdk/4.5/api/classes/imfusion-datalink) provides functionality to keep certain aspects of [Data](/cpp-sdk/4.5/api/classes/imfusion-data) (e.g. windowing information) in sync across multiple instances. Usually, syncing will be done in a bidirectional fashion (i.e. changes to any participating [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance will be synced to all other instances). If concrete subclasses of [DataLink](/cpp-sdk/4.5/api/classes/imfusion-datalink) behave differently (e.g. unidirectional linking single source -&gt; target(s)) they shall explicitly mention this in their class documentation.

**Note:** When mentioning *properties* in the context of [DataLink](/cpp-sdk/4.5/api/classes/imfusion-datalink), this can refer to *any* aspect of a [Data](/cpp-sdk/4.5/api/classes/imfusion-data) instance and does *not* necessarily mean the [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties) interface.
