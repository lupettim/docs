### takeOutput

```cpp
OwningDataList ImFusion::PointCloudIoAlgorithm::takeOutput ()
```

*override, virtual*

Return any new [Data](/cpp-sdk/4.5/api/classes/imfusion-data) that was created by the [Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) during the last call to [compute()](#compute).

The default implementation will return an empty list.

**Note:** Since ownership of the data is transferred, you can call this method only once between calls to [compute()](#compute).

### setIgnoreNanPoints

```cpp
void ImFusion::PointCloudIoAlgorithm::setIgnoreNanPoints (bool value)
```

*inline*

When reading the file, ignore points which have at least one coordinate equal to NAN (doesn't apply to dense point clouds).

Change this flag to true only if it is absolutely necessary to read in NAN points. Might be useful when the point cloud is not dense, but it is important to preserve the original indexing of points
