## Member Function Documentation

### addMapping

```cpp
template<typename T> bool ImFusion::GUI::InputEventMapper < T >::addMapping (T action, MouseEvent::Button button, Flags < InputEvent::KeyboardModifier > modifiers)
```

*inline*

Adds a new mouse interaction metaphor and map it to the given *action*.

Returns false if the given action was not added due to a conflict (same button+modifiers but different action and [allowsDuplicateMappings()](/cpp-sdk/4.5/api/classes/imfusion-gui-inputeventmapperbase) is off). You can use [matchAny()](#matchany-1-3) to query in advance if a particular mapping will conflict, e.g. matchAny(button, modifiers) != action.
