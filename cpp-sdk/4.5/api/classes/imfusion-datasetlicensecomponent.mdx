## Overview

[Data](/cpp-sdk/4.5/api/classes/imfusion-data) component for keeping track of the data's license (when it comes from a public dataset).

This components holds a vector of such information so that each element of a batch (for instance each frame of an image set) can have its own info. This vector is expected to have either one entry (in which case it is assumed to be valid for the whole image set) or one entry per image of the set.

**Note:** To avoid conflicts make sure to include potential plugin names into this ID. Due to the modular architecture of the ImFusion library, uniqueness of IDs cannot be checked at compile time.
