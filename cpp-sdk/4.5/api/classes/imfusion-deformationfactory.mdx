## Member Function Documentation

### typename

```cpp
template<class T, typename = std::enable_if_t<std::is_base_of_v<Deformation, T>>> bool ImFusion::DeformationFactory::registerDeformation (const std::string & id)
```

*inline*

Registers a [Deformation](/cpp-sdk/4.5/api/classes/imfusion-deformation) with the [DeformationFactory](/cpp-sdk/4.5/api/classes/imfusion-deformationfactory).

**Note:** The string IDs of Deformations have to be unique. If you register an ID a second time, the previously registered factory method will be overwritten.

**Parameters:**

- `id` — Unique string ID for the [Deformation](/cpp-sdk/4.5/api/classes/imfusion-deformation) to register.

**Returns:** true if the deformation has been successfully registered

Registers a [Deformation](/cpp-sdk/4.5/api/classes/imfusion-deformation) with the [DeformationFactory](/cpp-sdk/4.5/api/classes/imfusion-deformationfactory).

**Note:** The string IDs of Deformations have to be unique. If you register an ID a second time, the previously registered factory method will be overwritten.

**Parameters:**

- `id` — Unique string ID for the [Deformation](/cpp-sdk/4.5/api/classes/imfusion-deformation) to register.
- `factoryMethod` — Method to use for instantiating the [Deformation](/cpp-sdk/4.5/api/classes/imfusion-deformation) to register.

**Returns:** true if the deformation has been successfully registered
