## Member Enumeration Documentation

### StatusExt

```cpp
enum class ImFusion::MeshDistanceAlgorithm::StatusExt
```

*strong*

Enumeration listing possible errors during the computation.

**Enumerator:**

- `InvalidDistanceType` — Attempt to compute point-to-surface distance for the point clouds occurred.
This operation is not supported
- `DistanceComputationFailed` — Something went wrong during computation.
- `DiceCoefficientComputationFailed` — Something went wrong during the dice coefficient computation (most probably at least one of the meshes is not manifold).

### computeDiceCoefficient

```cpp
double ImFusion::MeshDistanceAlgorithm::computeDiceCoefficient ()
```

Compute the dice coefficient for two closed mesh volumes (returns -1 if calculation could not be performed (i.e.

at least one of the meshes is not closed)) dice = 2 \* volume(intersection(mesh1, mesh2))/(volume(mesh1) + volume(mesh2))
