`#include <ImFusion/Base/PointsOnData.h>`

## Overview

Interface class for points linked to [Data](/cpp-sdk/4.5/api/classes/imfusion-data).

Implementations of this class handle the addition of points and update of such points when the data changes. This class supports selection of a subset of points. Be default all the points are selected, selection can be changed using the setSelected method

### rank

```cpp
size_t ImFusion::PointsOnData::rank (const std::vector < vec3 > & points, double eps = 2.0)
```

*static*

Computes the rank (i.e.

intrinsic dimensionality) of the given set of 3D points, considering the threshold eps

The calculated rank can be interpreted as follows: 0: All points are located around a single point 1: All points are located on a single line 2: All points are located on a single plane 3: The points are spread out in 3D space

Returns 0 if the points vector is empty

**Parameters:**

- `points` — A vector of 3D points to compute the rank of.
- `eps` — An absolute threshold to determine significant singular values for rank calculation.

**Returns:** The rank of the given set of 3D points.
