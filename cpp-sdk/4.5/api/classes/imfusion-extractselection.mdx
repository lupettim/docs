## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to extract the current selection into a new object instance ([SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) or [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence)).

It will clone the input data with all aspects (e.g. DataComponents, [Properties](/cpp-sdk/4.5/api/classes/imfusion-properties)) but only the selected frames or samples. Example:

```cpp
SharedImageSet inputWith10Images = ...;    // some SharedImageSet with 10 images
Selection sel;
sel.setFirst(2);
sel.setLast(4);
inputWith10Images.setSelection(sel);       // select images 2, 3, and 4

ExtractSelection esAlgorithm({&inputWith10Images});
esAlgorithm.compute();
std::unique_ptr<SharedImageSet> result = esAlgorithm.takeOutput().extractFirstImage();
// `result` will be a copy of `inputWith10Images` but only containing images 2, 3, and 4
```
