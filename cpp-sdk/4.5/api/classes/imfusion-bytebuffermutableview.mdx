## Overview

Specialization of a [ByteBufferView](/cpp-sdk/4.5/api/classes/imfusion-bytebufferview) that additionally provides non-const access to the underlying data.

This class is usually used as argument type for processing functions that receive an output buffer to write into.

**See also:** [ByteBufferStream](/cpp-sdk/4.5/api/classes/imfusion-bytebufferstream)
