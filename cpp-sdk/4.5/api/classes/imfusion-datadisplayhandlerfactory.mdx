## Member Function Documentation

### createAllHandlers

```cpp
std::vector < std::unique_ptr < DataDisplayHandler > > ImFusion::DataDisplayHandlerFactory::createAllHandlers ()
```

*static*

Creates one instance of all registered [DataDisplayHandler](/cpp-sdk/4.5/api/classes/imfusion-datadisplayhandler) types.

The order of handlers is reversed wrt. their registration order so that handlers that were registered at a later time have precedence over handlers that were registered earlier.
