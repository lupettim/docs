## Overview

Qt Widget for displaying an overview of a transfer function and providing basic interaction.

Usually not intended to be used directly, consider using the [TransferFunctionWidget](/cpp-sdk/4.5/api/classes/imfusion-transferfunctionwidget) instead.

**See also:** [TransferFunctionWidget](/cpp-sdk/4.5/api/classes/imfusion-transferfunctionwidget)
