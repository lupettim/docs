## Overview

Default algorithm controller for [LinkPoseDeformable](/cpp-sdk/4.5/api/classes/imfusion-linkposedeformable) algorithm, with a slight modification to have [GlMesh](/cpp-sdk/4.5/api/classes/imfusion-glmesh) access.

This controller will become obsolete once there is SharedMesh class to accommodate Mesh/GlMesh.

Two-step initialization function to be implemented by subclasses.
