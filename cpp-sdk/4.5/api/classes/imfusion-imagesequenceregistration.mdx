## Overview

Registers a [SharedImageSet](/cpp-sdk/4.5/api/classes/imfusion-sharedimageset) of multiple images.

The images in the set are registered to each other, either to a fixed frame or to the predecessor of each frame. Internally an ImFusion::ImageRegistration to register two frames. The registration is only performed on the current selection.

### registrationObject

```cpp
const ImageRegistration * ImFusion::ImageSequenceRegistration::registrationObject () const
```

Returns the ImFusion::ImageRegistration object used for the per-frame registration.

The object is persistent and is used for all frames. The ImFusion::ImageRegistration is created with two dummy SharedImageSets that are filled with the current registering frames.

### setBackwards

```cpp
void ImFusion::ImageSequenceRegistration::setBackwards (bool backwards)
```

Sets if the registration should start with the last frame or not.

By default the registration starts with the first frame of the selection.
