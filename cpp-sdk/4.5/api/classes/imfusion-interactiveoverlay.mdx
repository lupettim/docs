`#include <ImFusion/GUI/InteractiveOverlay.h>`

## Overview

Wraps a [GlOverlay](/cpp-sdk/4.5/api/classes/imfusion-gloverlay) in order to display it at specific positions in an [InteractiveView](/cpp-sdk/4.5/api/classes/imfusion-interactiveview).

**Note:** The size hint of an [InteractiveOverlay](/cpp-sdk/4.5/api/classes/imfusion-interactiveoverlay) is managed/defined by the underlying [GlOverlay](/cpp-sdk/4.5/api/classes/imfusion-gloverlay). Thus, the corresponding functions of this class are only a forwarding convenience API.

**See also:** [GlOverlay](/cpp-sdk/4.5/api/classes/imfusion-gloverlay), [OverlayInteraction](/cpp-sdk/4.5/api/classes/imfusion-overlayinteraction)

## Member Enumeration Documentation

### AnchorPoint

```cpp
enum ImFusion::InteractiveOverlay::AnchorPoint
```

[Anchor](/cpp-sdk/4.5/api/classes/imfusion-gui-viewoverlay) point of the overlay in the hosting [InteractiveView](/cpp-sdk/4.5/api/classes/imfusion-interactiveview).

**Enumerator:**

- `BottomLeft` — Bottom left corner.
- `BottomRight` — Bottom right corner.
- `TopLeft` — Top left corner.
- `TopRight` — Top right corner.
- `Top` — Top center.
- `Bottom` — Bottom center.
- `Left` — Left center.
- `Right` — Right center.
- `Fill` — Always fills the complete view.

### setSizeHint

```cpp
virtual void ImFusion::InteractiveOverlay::setSizeHint (int width, int height)
```

*virtual*

Sets the preferred size of the overlay for a DPI scaling of 1.0.

The size hint should be the optimal size of the overlay to display all its information. If the overlay does not have a preferred size, the size hint should be set to to the minimalSizeHint.

### setMinimalSizeHint

```cpp
virtual void ImFusion::InteractiveOverlay::setMinimalSizeHint (int width, int height)
```

*virtual*

Sets the minimal size of the overlay for a DPI scaling of 1.0.

The minimal size should reflect the size to display the content in an undistorted way. The actual size of the overlay can still be lower than this, in which case the overlay should scale its content.

### setAnchorPoint

```cpp
void ImFusion::InteractiveOverlay::setAnchorPoint (AnchorPoint anchorPoint)
```

Set the anchor point of overlay.

If several overlays use the same anchor point the will be rendered above/below (depending on Bottom or Top) the previous overlays.
