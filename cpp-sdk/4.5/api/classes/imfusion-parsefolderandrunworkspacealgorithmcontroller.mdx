## Overview

[Controller](/cpp-sdk/4.5/api/classes/imfusion-controller) for the algorithm parsing folders and running a workspace on them.

Since workspace execution and file-type agnostic loading requires the application controller, the algorithm is only parsing the folders for files to process, and this controller does everything else.

See [ParseFolderAndRunWorkspaceAlgorithm](/cpp-sdk/4.5/api/classes/imfusion-parsefolderandrunworkspacealgorithm) on how to use the algorithm.
