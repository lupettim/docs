`#include <ImFusion/Base/PointCloudAnalysis.h>`

## Overview

Geometric analysis of a point cloud.

## Public Member Functions

- **~PointCloudAnalysis** () override — Destructor.
- int **size** () const — Return number of samples (point clounds).
- double \* **statistics** (int which) — Return selected statistics over all samples.
- double **correlation** (int which) — Return correlation for selected statistic.
- double **spearmanRankCorrelation** (int which) — Return Spearman rank correlation for selected statistic.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; vec3 &gt; **pointCloud** (int which) — Return point cloud.
- void **preprocessPointClouds** () — Pre-processing of the point clouds.
- virtual bool **isPredictionAvailable** () — Return whether RF Prediction has been learnt.
- bool **isSurfaceErrorAvailable** () — Return whether the surface error is available.

## Protected Member Functions

- void **updateCorrelations** ()

## Protected Attributes

- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; vec3 &gt; &gt; **m_pc** — [Vector](/cpp-sdk/4.5/api/classes/imfusion-vector) of point clouds to analyse.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; double &gt; &gt; **m_stats** — Computed point cloud statistics.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; double &gt; **m_error** — Reference error values to compare.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; double &gt; **m_surfaceError** — Reference surface error values to compare.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; double &gt; **m_corr** — Correlation of the different statistics with the error values.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; double &gt; **m_spearmanCorr** — Spearman rank correlation of the different statistics with the error values.
- int **m_preprocessPointClouds** — Pre-processing mode (0 none, 1 translation, 2 translation+eigenvectors).
- int **m_skipFiles** — Skip files (e.g. distinguish Femur/Tibia).
