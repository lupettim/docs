## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to change properties of one or multiple tracked SharedImageSets, or a single [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence).

Note that this algorithm directly modifies/returns the [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence) properties in configure/configuration. This algorithm serves two purposes: 1) To allow configuring a [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence) from a workspace, and 2) as a dummy algorithm to launch the corresponding controller with the properties dialog. SDK users should directly modify the [TrackingSequence](/cpp-sdk/4.5/api/classes/imfusion-trackingsequence).

## Public Types

- enum class SampleTimeoutCriteria &#123; **NoCriteria** = 0 , **Average** , **Median** , **UserDefined** &#125; — Set sample timeout criteria (0=None, 1=Average, 2=Median, 3=UserDefined) The timeout criteria is used define the operation that should be performed to the TrackedImageSharedSet timestamps in order to calculate the corresponding timeout between samples.
