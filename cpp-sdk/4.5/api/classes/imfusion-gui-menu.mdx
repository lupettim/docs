`#include <ImFusion/GUI/Menu.h>`

## Overview

Represents a hierarchical menu of actions used to describe menu bars, context menus, or popup menus.

This class does not implement any rendering or other visual representation of the underlying menu but serves only as [GUI](/cpp-sdk/4.5/api/namespaces/imfusion-gui) library-agnostic interface to describe the appearance and behavior of it.

A menu is formed by a list of items, where each item is either a clickable action, a divider separating groups of related actions, or a recursive submenu of the same form.

Example:

```cpp
Menu m;
m.addAction("Reset", [this]() { reset(); }, "Reset to default settings");
m.addToggleAction("Show details", showsDetails(); [this]() { setShowDetails(!showsDetails()); });
m.addSeparator();
{
 Menu submenu("Extras", "Advanced options");
    submenu.addAction(...);
    m.addSubmenu(std::move(submenu));
}
```
 fragment

**Warning:** Due to the asynchronous nature of context menus make sure to follow the following guidelines on object lifetimes: [Action](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-action) callbacks should only capture references/pointers to objects that have at least the same lifetime as the object that is creating the menu. Receivers of a [Menu](/cpp-sdk/4.5/api/classes/imfusion-gui-menu) instance must ensure that the menu does *not* outlive the lifetime of all objects to which the original [ContextMenuEvent](/cpp-sdk/4.5/api/classes/imfusion-gui-contextmenuevent) was passed to.

## Classes

- struct [Action](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-action) — Represents an item inside a [Menu](/cpp-sdk/4.5/api/classes/imfusion-gui-menu) where a callback function is called when user clicks on it.
- struct [Separator](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-separator) — Represents a separator to be shown as divider between unrelated groups of actions.

### addRadioAction

```cpp
void ImFusion::GUI::Menu::addRadioAction (const std::string & title, bool checkState, Action::ActivationCallback activationCallback, const std::string & description = "", std::shared_ptr < TypedImage < uint8_t > > icon = nullptr)
```

Adds a new exclusively checkable [Action](/cpp-sdk/4.5/api/structs/imfusion-gui-menu-action) (representing one of multiple mutually exclusive options) to the end of the list of actions.

A group of exclusive options is formed by the set of adjacent actions of this type where there is no separator in-between.

**Parameters:**

- `title` — Text to be displayed on the menu entry.
- `checkState` — Flag if the underlying option is currently selected.
- `activationCallback` — Callback function to be called when this action was clicked by the user
- `description` — Optional text that explains the menu entry, to be shown as tooltip/and or in the status bar.
- `icon` — Optional icon to show next to the menu entry
