## Overview

Links the pose of two or more data instances if there is deformation present.

## Public Member Functions

- **LinkPoseDeformable** (const [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) &[data](/cpp-sdk/4.5/api/classes/imfusion-linkpose)) — Constructor with data list, pose linking starts right away.
- **~LinkPoseDeformable** () override — Destructor, stops the linking of the transformation.
- void **onDeformationChanged** (const [SharedImage](/cpp-sdk/4.5/api/classes/imfusion-sharedimage) \*img) — Called when the deformation instance of an image has changed.
- void **onDeformationEvent** (const [DeformationEvent](/cpp-sdk/4.5/api/classes/imfusion-deformationevent) &event) — Called when deformation parameters have changed.
- [DataList](/cpp-sdk/4.5/api/classes/imfusion-datalist) **getData** () — List of data instances to link.
- [std::vector](http://en.cppreference.com/w/cpp/container/vector.html)&lt; [Gl3DObject](/cpp-sdk/4.5/api/classes/imfusion-gl3dobject) \* &gt; & **getGlObjectsVector** () — [Vector](/cpp-sdk/4.5/api/classes/imfusion-vector) of GlMeshes and GlPointClouds to be filled by controller.
- void **setPauseDefUpdating** (bool b) — Turns updates based on the deformation field (expensive!) on or off.
