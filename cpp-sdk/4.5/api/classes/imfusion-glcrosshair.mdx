`#include <ImFusion/GL/GlCrosshair.h>`

## Overview

Renders a crosshair as two symmetrical and orthogonal lines, with a gap in the center with a given length.

Crosshair total size and gap is defined side to side. As follows for one of the crosshair lines:

```cpp
<-------  size  ------->
 -----           -----
|     | <-gap-> |     |
 -----           -----
```
 fragment
