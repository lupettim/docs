## Overview

[Base](/cpp-sdk/4.5/api/structs/imfusion-dicom-base) interface for a Qt item delegate that defines how the entries in a [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget) are supposed to be shown.

**See also:** [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget)

## Public Member Functions

- virtual void [resetHover](#resethover) ()=0 — Called if the parent [DataWidget](/cpp-sdk/4.5/api/classes/imfusion-datawidget) received a Qt::LeaveEvent to remove any potential hover effect renderings.
