`#include <ImFusion/GUI/PropertiesWidgetParamControls.h>`

## Overview

Composite control that wraps an existing [ParamControl](/cpp-sdk/4.5/api/classes/imfusion-paramcontrol) and adds a label to the left of it.

The returned widget contains a QLabel and the widget of the [ParamControl](/cpp-sdk/4.5/api/classes/imfusion-paramcontrol) in an horizontal layout.
