`#include <ImFusion/Reg/FitBoundingBoxAlgorithm.h>`

## Overview

[Algorithm](/cpp-sdk/4.5/api/classes/imfusion-algorithm) to fit an oriented bounding box.

## Public Member Functions

- **FitBoundingBoxAlgorithm** ([SharedImage](/cpp-sdk/4.5/api/classes/imfusion-sharedimage) \*volume) — Constructor with a volume, the border points will be computed here.
- **FitBoundingBoxAlgorithm** ([PointCloud](/cpp-sdk/4.5/api/classes/imfusion-pointcloud) \*pc) — Constructor using a point cloud.
- **FitBoundingBoxAlgorithm** ([Mesh](/cpp-sdk/4.5/api/classes/imfusion-mesh) \*mesh) — Constructor using a mesh.
- void **setOutsideMode** (bool flag) — If true, the smallest outer bounding box is computed (default), the largest inner one otherwise.
- void **setApply** (bool flag) — If true, the resulting bounding box matrix will be applied on the input data (enabled by default).
- void **setResampling** (double spacing) — If greater zero, a new resampled image will be created with the given isotropic spacing.
- void **boundingTransform** (mat4 &matOut, vec3 &extOut) const — Return the resulting bounding box transformation and extent.
- double [evaluate](#evaluate) (int n, const double \*x, double \*dx=0) override — Cost function evaluation for the [Optimizer](/cpp-sdk/4.5/api/classes/imfusion-optimizer).
- void **computeOuterBoundingBox** (mat4 &matInOut, vec3 &extOut) — Compute the outer bounding box for the given transformation and update it.
