## Member Function Documentation

### sceneEvent

```cpp
bool ImFusion::InteractiveAnnotation::sceneEvent (QEvent * event, InteractiveView * view)
```

*override, virtual*

Called by the hosting [InteractiveView](/cpp-sdk/4.5/api/classes/imfusion-interactiveview) to notify on user input events.

Default implementation will forward the event to the [Manipulator](/cpp-sdk/4.5/api/classes/imfusion-manipulator) if the object is visible.

**Returns:** true if the event has been handled and should no longer be passed to other objects.
