The ImFusion C\+\+ SDK 4.4 is a modular, plugin-based framework for building production medical imaging applications on Windows, Linux, and macOS.

## What you get

- A unified data model for 2D, 3D, and 4D medical images
- GPU-accelerated image processing and rendering
- A plugin system for DICOM, ML, registration, segmentation, ultrasound, CT, vision, and navigation modules
