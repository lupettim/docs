## Classes

### `Expression`

```python
class Expression
```

Expressions to be used for lazy evaluation.

This class serves as a wrapper for `~imfusion.SharedImage`, `~imfusion.SharedImageSet`, and scalar values to be used for lazy evaluation. Lazy evaluation approach delays the actual evaluation until the point that the result is needed. If you prefer the eager evaluation approach, you can directly invoke operations on `~imfusion.SharedImage` and `~imfusion.SharedImageSet` objects.

Here is an example how to use lazy evaluation approach:

&gt;&gt;&gt; from imfusion import imagemath

Create expressions from `~imfusion.SharedImage` instances:

&gt;&gt;&gt; expr1 = imagemath.lazy.Expression(si1)
&gt;&gt;&gt; expr2 = imagemath.lazy.Expression(si2)

Any operation with the expressions will return another expression. Expressions are stored in the expression tree and not evaluated yet without any evaluation. 

&gt;&gt;&gt; expr3 = expr1 + expr2

Expressions must be explicitly evaluated to get results. Use the `evaluate` method for this purpose:

&gt;&gt;&gt; res = expr3.evaluate()

Here, result is a `~imfusion.SharedImage` instance:

&gt;&gt;&gt; print(res)
imfusion.SharedImage(USHORT width: 512 height: 512 spacing: 0.661813x0.661813x1 mm)
