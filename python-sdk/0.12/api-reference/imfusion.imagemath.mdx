# `imfusion.imagemath`

imfusion.imagemath - Bindings for ImageMath Operations  
======================================================

This module provides element-wise arithmetic operations for `~imfusion.SharedImage` and `~imfusion.SharedImageSet`. You can apply these `~imfusion.imagemath` functionalities directly to objects of `~imfusion.SharedImage` and `~imfusion.SharedImageSet` with eager evaluation. Alternatively, the module offers lazy evaluation functionality through the submodule `~imfusion.imagemath.lazy`. You can create wrapper expressions using the `~imfusion.imagemath.lazy.Expression` provided by `~imfusion.imagemath.lazy`.

See `~imfusion.imagemath.lazy.Expression` for details.

*Example for eager evaluation:*

&gt;&gt;&gt; from imfusion import imagemath

Add ``si1`` and ``si2``, which are `~imfusion.SharedImage` instances:

&gt;&gt;&gt; si1 = imfusion.load(ct_image_png)[0][0]
&gt;&gt;&gt; si2 = si1.clone()
&gt;&gt;&gt; res = si1 + si2

``res`` is a `~imfusion.SharedImage` instance.

&gt;&gt;&gt; print(res)
imfusion.SharedImage(USHORT width: 512 height: 512 spacing: 0.661813x0.661813x1 mm)

*Example for lazy evaluation:*

&gt;&gt;&gt; from imfusion.imagemath import lazy

Create expressions from `~imfusion.SharedImage` instances:

&gt;&gt;&gt; expr1 = lazy.Expression(si1)
&gt;&gt;&gt; expr2 = lazy.Expression(si2)

Add ``expr1`` and ``expr2``:

&gt;&gt;&gt; expr3 = expr1 + expr2

Alternatively, you could add ``expr1`` and ``si2`` or ``si1`` and ``expr2``. Any expression containing an instance of `~imfusion.imagemath.lazy.Expression` will be converted to lazy evaluation expression. 

&gt;&gt;&gt; expr3 = expr1 + si2

Find the result with lazy evaluation: 

&gt;&gt;&gt; res = expr3.evaluate()

``res`` is a `~imfusion.SharedImage` instance similar to eager evaluation case.

&gt;&gt;&gt; print(res)
imfusion.SharedImage(USHORT width: 512 height: 512 spacing: 0.661813x0.661813x1 mm)
