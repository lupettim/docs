Core concepts in the ImFusion Python SDK 0.12:

- **SharedImageSet** — the core container for 2D/3D/4D image data.
- **DataModel** — groups related data and results for a session.
- **Properties** — key-value configuration passed to algorithms.
- **Algorithm** — unit of processing; run built-in or register your own.
