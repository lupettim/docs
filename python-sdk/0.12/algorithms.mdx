---
title: "Run and Register Algorithms in ImFusion 0.12"
sidebarTitle: "Algorithms"
description: "Execute built-in and custom algorithms, discover available algorithm IDs, inspect parameters, and register Python algorithms."
---

Run built-in algorithms with `imfusion.execute_algorithm`, discover IDs via `imfusion.available_algorithms()`, inspect parameters, and register custom Python algorithms with `imfusion.register_algorithm`.
