Load, manipulate, and save medical images with `imfusion.load`, NumPy interop, and `imfusion.save`.

```python
import imfusion
sis = imfusion.load('scan.dcm')
arr = sis[0].numpy()
imfusion.save(sis, 'out.nii.gz')
```
